# 01.09 - Frequency

## What is Frequency?

Frequency is the number of complete cycles of an alternating current (AC) waveform that occur in one second. It measures how fast the current alternates or how many times per second it changes direction and returns to its starting point.

The unit of frequency is the **Hertz** (symbol: Hz), named after German physicist **Heinrich Hertz** (1857-1894), who first demonstrated the existence of electromagnetic waves. One Hertz equals one cycle per second.

## Understanding Cycles

### One Complete Cycle

A cycle is one complete repetition of a waveform. For a sine wave AC signal, one cycle starts at zero, goes positive to a peak, returns through zero, goes negative to a trough, and returns to zero.

```
ONE COMPLETE CYCLE (Sine Wave)

    Voltage (+)
         │
    170V ┼          ● Peak (maximum positive)
         │         ╱ ╲
         │        ╱   ╲
         │       ╱     ╲
         │      ╱       ╲
         │     ╱         ╲
         │    ╱           ╲
       0 ┼───●─────────────●───●─────────────► Time
         │  ╱               ╲ ╱
         │ ╱                 ╲
         │╱                   ╲
   -170V ┼                    ● Trough (maximum negative)
         │
    
    CYCLE ANATOMY:
    ├── Point 0 to Point 1: Positive half-cycle (0° to 180°)
    ├── Point 1 to Point 2: Negative half-cycle (180° to 360°)
    ├── One full cycle = 360 degrees
    └── Returns to starting value and direction


CYCLE COUNTING:

    1 cycle = complete repetition
    2 cycles = two repetitions
    60 cycles = sixty repetitions (60 Hz)
    
    ┌─────────────────────────────────────────────────┐
    │                                                 │
    │   60 Hz means: 60 cycles in 1 second            │
    │               = 1 cycle every 16.67 ms          │
    │               = direction reverses 120 times/s  │
    │                                                 │
    └─────────────────────────────────────────────────┘
```

## Period: The Time Per Cycle

Period is the time duration of one complete cycle. Frequency and period are inversely related.

```
PERIOD FORMULA

    T = 1 / f
    f = 1 / T

WHERE:
T = Period in seconds (s)
f = Frequency in Hertz (Hz)


VISUAL RELATIONSHIP:

    Higher frequency = Shorter period (cycles packed tighter)
    Lower frequency = Longer period (cycles spread apart)


EXAMPLES:

    60 Hz AC:
    T = 1 / 60 = 0.01667 seconds = 16.67 milliseconds
    
    ┌─────────────────────────────────────────────────┐
    │  1 cycle takes 16.67 ms                         │
    │  ├── Positive half: 8.33 ms                     │
    │  └── Negative half: 8.33 ms                     │
    └─────────────────────────────────────────────────┘


    50 Hz AC:
    T = 1 / 50 = 0.02 seconds = 20 milliseconds
    
    ┌─────────────────────────────────────────────────┐
    │  1 cycle takes 20 ms                            │
    │  ├── Positive half: 10 ms                       │
    │  └── Negative half: 10 ms                       │
    └─────────────────────────────────────────────────┘


    1 kHz audio tone:
    T = 1 / 1000 = 0.001 seconds = 1 millisecond


    1 MHz clock signal:
    T = 1 / 1,000,000 = 0.000001 seconds = 1 microsecond


    1 GHz processor:
    T = 1 / 1,000,000,000 = 0.000000001 seconds = 1 nanosecond


PERIOD vs FREQUENCY TABLE:

    f (Hz)    T (seconds)    T (milliseconds)    Example
    ─────────────────────────────────────────────────────────
    0.5       2.0 s          2000 ms             Heartbeat
    1         1.0 s          1000 ms             1 Hz signal
    2         0.5 s          500 ms              Slow blink
    5         0.2 s          200 ms              Fast blink
    10        0.1 s          100 ms              Muscle twitch
    20        0.05 s         50 ms               Bass note
    50        0.02 s         20 ms               50 Hz AC
    60        0.01667 s      16.67 ms            60 Hz AC
    100       0.01 s         10 ms               Low hum
    440       0.00227 s      2.27 ms             Concert A note
    1000      0.001 s        1 ms                Beep tone
    10000     0.0001 s       0.1 ms              Ultrasonic
    1000000   0.000001 s     0.001 ms (1 µs)     1 MHz clock
```

## Frequency in Power Grids

### 50 Hz vs 60 Hz

The world uses two main frequencies for electrical power: 50 Hz and 60 Hz.

```
WORLD FREQUENCY MAP (Simplified)

    60 Hz Regions:                   50 Hz Regions:
    ┌─────────────────┐              ┌──────────────────┐
    │  United States  │              │  Europe (most)   │
    │  Canada         │              │  United Kingdom  │
    │  Mexico         │              │  China           │
    │  Japan (east)   │              │  India           │
    │  Taiwan         │              │  Australia       │
    │  South Korea    │              │  Africa (most)   │
    │  Philippines    │              │  Russia          │
    │  Brazil (most)  │              │  Middle East     │
    │  Saudi Arabia   │              │  Japan (west)    │
    └─────────────────┘              └──────────────────┘


WHY TWO DIFFERENT FREQUENCIES?

    Historical reasons:
    ├── 60 Hz: Developed by Nikola Tesla and Westinghouse (US)
    ├── 50 Hz: Adopted by AEG (German company) in Europe
    ├── No technical superiority of one over the other
    └── Each region simply chose and standardized


COMPARISON: 50 Hz vs 60 Hz

    Feature               50 Hz                    60 Hz
    ─────────────────────────────────────────────────────────
    Period                20 ms                    16.67 ms
    Cycles per second     50                       60
    Transformer size      Larger (more iron)       Smaller (less iron)
    Motor speed (sync)    3000 RPM (2-pole)        3600 RPM (2-pole)
    Light flicker         More noticeable          Less noticeable
    Transmission loss     Slightly higher          Slightly lower
    Clock timing          50 cycles = 1 second     60 cycles = 1 second
    Arc extinguishing     Slower                   Faster
    Countries using       ~150                     ~40


PRACTICAL EFFECTS:

    Traveling with appliances:
    ├── 120V 60Hz device in 230V 50Hz country → Needs voltage converter
    ├── Motor speed will change (50 Hz motor runs faster on 60 Hz)
    ├── Clocks using line frequency will run fast/slow
    └── Universal power supplies (most electronics) work on both


FREQUENCY TOLERANCE:

    Grid operators maintain frequency very precisely:
    ├── Normal: 60 Hz ± 0.05 Hz (very tight)
    ├── Under heavy load: Drops slightly (59.9-60.0 Hz)
    ├── Under light load: Rises slightly (60.0-60.1 Hz)
    ├── If frequency drifts too far → Blackout!
    └── Utilities use "load shedding" to maintain frequency
```

## Special Frequencies

### Resonant Frequency

Every electrical and mechanical system has a natural frequency where it vibrates most easily.

```
RESONANT FREQUENCY (LC Circuit)

    For an inductor (L) and capacitor (C) in parallel or series:

    f_resonant = 1 / (2π √(LC))

    WHERE:
    f = resonant frequency in Hz
    L = inductance in Henries (H)
    C = capacitance in Farads (F)


EXAMPLES:

    Example 1: AM radio tuning
    L = 250 µH (0.00025 H)
    C = 365 pF (0.000000000365 F)
    
    f = 1 / (2 × 3.14 × √(0.00025 × 3.65e-10))
    f = 1 / (6.28 × √(9.125e-14))
    f = 1 / (6.28 × 3.02e-7)
    f = 1 / 1.90e-6 = 526,000 Hz = 526 kHz (AM band)


    Example 2: FM radio tuning
    L = 0.1 µH (1e-7 H)
    C = 10 pF (1e-11 F)
    
    f = 1 / (6.28 × √(1e-7 × 1e-11))
    f = 1 / (6.28 × √(1e-18))
    f = 1 / (6.28 × 1e-9)
    f = 1 / 6.28e-9 = 159 MHz (FM band, approximate)


PRACTICAL APPLICATIONS:

    ├── Radio tuning (selecting stations)
    ├── Wireless charging (resonant coupling)
    ├── Metal detectors (change in resonant frequency)
    ├── Crystal oscillators (precise frequency reference)
    ├── Filters (pass certain frequencies, block others)
    └── Avoid structural resonance (bridges, buildings)
```

## Frequency and Wavelength

For electromagnetic waves (radio, light), frequency and wavelength are related by the speed of light.

```
WAVELENGTH FORMULA

    λ = c / f
    f = c / λ

WHERE:
λ = wavelength in meters (m)
c = speed of light = 299,792,458 m/s ≈ 3 × 10⁸ m/s
f = frequency in Hertz (Hz)


EXAMPLES:

    Power line (60 Hz):
    λ = 3e8 / 60 = 5,000,000 m = 5,000 km
    
    AM radio (1 MHz):
    λ = 3e8 / 1e6 = 300 m
    
    FM radio (100 MHz):
    λ = 3e8 / 1e8 = 3 m
    
    WiFi (2.4 GHz):
    λ = 3e8 / 2.4e9 = 0.125 m = 12.5 cm
    
    Visible light (500 THz = 5 × 10¹⁴ Hz):
    λ = 3e8 / 5e14 = 6 × 10⁻⁷ m = 600 nm (orange)


WAVELENGTH vs FREQUENCY TABLE:

    Frequency       Wavelength        Classification
    ─────────────────────────────────────────────────────────
    60 Hz           5,000 km          Power grid
    1 kHz           300 km            Audio (VLF radio)
    1 MHz           300 m             AM radio
    100 MHz         3 m               FM radio, VHF TV
    1 GHz           30 cm             UHF, cell phones
    2.4 GHz         12.5 cm           WiFi, Bluetooth
    10 GHz          3 cm              Satellite, radar
    100 GHz         3 mm              EHF, 5G
    1 THz           0.3 mm            Terahertz (between radio and IR)
    500 THz         600 nm            Visible light (orange)
    10¹⁵ Hz         300 nm            Ultraviolet
    10¹⁸ Hz         0.3 nm            X-ray
```

## Frequency Measurement

### Using an Oscilloscope

```
FREQUENCY MEASUREMENT ON OSCILLOSCOPE

STEP 1: Connect signal to channel input

STEP 2: Adjust timebase (seconds per division)
    ├── For 60 Hz: 5 ms/div (one cycle takes ~3.3 divisions)
    ├── For 1 kHz: 0.5 ms/div (one cycle takes 2 divisions)
    ├── For 1 MHz: 0.5 µs/div (one cycle takes 2 divisions)
    └── Adjust so 1-5 cycles visible on screen

STEP 3: Measure period (T)
    ├── Count horizontal divisions for one cycle
    ├── Multiply by timebase setting
    └── T = divisions × seconds/division

STEP 4: Calculate frequency
    f = 1 / T


EXAMPLE:

    Oscilloscope display:
    ├── Timebase: 5 ms/div
    ├── One cycle occupies 3.33 divisions horizontally
    ├── T = 3.33 × 5 ms = 16.67 ms
    └── f = 1 / 0.01667 = 60 Hz ✓


USING CURSORS (modern scopes):
    ├── Place cursor A at start of cycle
    ├── Place cursor B at end of same cycle
    ├── Scope displays ΔT (period) automatically
    └── Some scopes calculate frequency directly
```

### Using a Frequency Counter

```
FREQUENCY COUNTER MEASUREMENT

DIRECT COUNTING METHOD:
    ├── Count number of cycles in a fixed time (e.g., 1 second)
    ├── Display = cycle count = frequency
    └── Good for frequencies > 10 Hz


RECIPROCAL METHOD (better for low frequencies):
    ├── Measure period with high resolution
    ├── Calculate f = 1/T
    └── Better accuracy for < 10 Hz


TYPICAL FREQUENCY COUNTER ACCURACY:
    ├── Basic counters: ±1 count ± timebase error
    ├── Good counters: ±0.001% (±10 ppm)
    ├── Lab grade: ±0.000001% (±0.01 ppm)
    └── GPS-disciplined: ±1 × 10⁻¹² (incredibly accurate)


MEASUREMENT TIPS:
    ├── Use shortest gate time for fast updates (1 ms for MHz)
    ├── Use longest gate time for best resolution (10 seconds for Hz)
    ├── For low frequencies, measure period instead
    └── For high frequencies (>1 GHz), use prescaler (divide by 10/100)
```

### Using a Multimeter (Limited)

```
MULTIMETER FREQUENCY MEASUREMENT

Some multimeters have frequency measurement capability:

    Typical specifications:
    ├── Range: 10 Hz - 100 kHz (some to MHz)
    ├── Accuracy: ±(0.1% + 1 digit)
    ├── Input voltage: 0.5V - 10V RMS (typical)
    └── Requires sufficient signal amplitude


HOW TO MEASURE:
    ├── Set dial to Hz (or V with Hz button)
    ├── Connect leads as for voltage measurement
    ├── Read frequency directly
    └── Polarity doesn't matter (AC)


LIMITATIONS:
    ├── Won't measure very low frequencies (<10 Hz)
    ├── Won't measure very high frequencies (>100 kHz)
    ├── Needs clean signal (noise causes errors)
    └── Not as accurate as dedicated counter
```

## Frequency Effects on Components

### Capacitors and Frequency

```
CAPACITOR BEHAVIOR vs FREQUENCY

Capacitive reactance (opposition to AC) decreases with frequency:

    Xc = 1 / (2π × f × C)

    WHERE:
    Xc = Capacitive reactance (Ω)
    f = Frequency (Hz)
    C = Capacitance (F)


BEHAVIOR:

    Frequency        Xc (for 1 µF capacitor)
    ─────────────────────────────────────────
    1 Hz             159,000 Ω (very high)
    60 Hz            2,654 Ω
    1 kHz            159 Ω
    100 kHz          1.59 Ω (very low)
    1 MHz            0.159 Ω (almost short)


PRACTICAL EFFECTS:
    ├── At low frequencies: Capacitor looks like open circuit
    ├── At high frequencies: Capacitor looks like short circuit
    ├── Used in filters: High-pass (blocks low, passes high)
    └── Used for decoupling: Shorts high-frequency noise to ground
```

### Inductors and Frequency

```
INDUCTOR BEHAVIOR vs FREQUENCY

Inductive reactance (opposition to AC) increases with frequency:

    XL = 2π × f × L

    WHERE:
    XL = Inductive reactance (Ω)
    f = Frequency (Hz)
    L = Inductance (H)


BEHAVIOR:

    Frequency        XL (for 1 mH inductor)
    ─────────────────────────────────────────
    1 Hz             0.00628 Ω (almost short)
    60 Hz            0.377 Ω (very low)
    1 kHz            6.28 Ω
    100 kHz          628 Ω
    1 MHz            6,280 Ω (very high)


PRACTICAL EFFECTS:
    ├── At low frequencies: Inductor looks like short circuit
    ├── At high frequencies: Inductor looks like open circuit
    ├── Used in filters: Low-pass (passes low, blocks high)
    └── Used in chokes: Block high-frequency noise
```

### Resistors and Frequency

```
RESISTOR BEHAVIOR vs FREQUENCY

Ideal resistor: Same resistance at all frequencies

Real resistor:
    ├── At low frequencies (DC - MHz): Nearly ideal
    ├── At high frequencies (>100 MHz): Parasitic capacitance affects value
    ├── At very high frequencies (>1 GHz): Acts like inductor/capacitor
    └── Wirewound resistors: Significant inductance at high frequencies


PRACTICAL ADVICE:
    ├── For most circuits (<1 MHz): Resistors are purely resistive
    ├── For RF circuits (>10 MHz): Use carbon/metal film (not wirewound)
    ├── For very high frequencies: Use surface mount (less parasitic)
    └── For precision: Consider frequency coefficient in datasheet
```

## Summary

1. **Frequency** is the number of complete cycles per second, measured in Hertz (Hz)

2. **Period** (T) is the time for one cycle: T = 1/f

3. **World power grids** use 50 Hz (Europe, Asia, Africa) or 60 Hz (Americas)

4. **Human hearing** ranges from 20 Hz to 20,000 Hz

5. **Radio frequencies** span from 3 kHz to 300 GHz, divided into bands (VLF to EHF)

6. **Digital devices** use frequencies from kHz (real-time clocks) to GHz (processors)

7. **Crystal oscillators** provide very precise frequencies (32.768 kHz for timekeeping)

8. **Resonant frequency** occurs when inductive and capacitive reactances cancel

9. **Wavelength** relates to frequency: λ = c/f (for electromagnetic waves)

10. **Capacitors** pass high frequencies, block low frequencies (Xc ∝ 1/f)

11. **Inductors** block high frequencies, pass low frequencies (XL ∝ f)

12. **Frequency measurement** uses oscilloscopes (period measurement), frequency counters (direct counting), or multimeters (limited range)
