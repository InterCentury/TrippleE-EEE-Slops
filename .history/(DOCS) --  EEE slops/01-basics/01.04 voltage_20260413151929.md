# 01.03 - Electric Current

## What is Electric Current?

Electric current is the flow of electric charge. In most circuits, this charge is carried by electrons moving through a conductor. Current is measured as the rate at which charge passes a given point in a circuit.

The word "current" comes from the Latin *"currere"*, meaning "to run" or "to flow." Just as water current is the flow of water molecules, electric current is the flow of charged particles.

## The Two Types of Current Flow

### Conventional Current vs Electron Flow

There are two ways to describe current direction, and this causes confusion for many beginners.

```
CONVENTIONAL CURRENT (Historical Standard)
─────────────────────────────────────────────────

    +  ──────────────────────────────────►  -
    
    "Current flows from positive to negative"
    
    Benjamin Franklin's assumption (1750s):
    ├── Believed positive charges moved
    ├── Became the standard
    ├── Used in all circuit diagrams
    ├── Used in formulas (I = V/R)
    └── Still taught worldwide


ELECTRON FLOW (Physical Reality)
─────────────────────────────────────────────────

    -  ──────────────────────────────────►  +
    
    "Electrons flow from negative to positive"
    
    The truth (discovered after Franklin):
    ├── Electrons are negatively charged
    ├── Repelled by negative terminal
    ├── Attracted to positive terminal
    ├── Discovered by J.J. Thomson (1897)
    └── What actually happens in wires


COMPARISON SIDE-BY-SIDE
─────────────────────────────────────────────────

    CIRCUIT: Battery → Wire → Light Bulb → Wire → Battery
    
    CONVENTIONAL CURRENT:
    Battery(+) ──► Bulb ──► Battery(-)
         ↑                      ↓
         └──────────────────────┘
    
    ELECTRON FLOW:
    Battery(-) ──► Bulb ──► Battery(+)
         ↑                      ↓
         └──────────────────────┘
    
    BOTH ARE CORRECT (just different conventions)!
```

### Which One Should You Use?

| Situation | Use Conventional Current | Use Electron Flow |
|-----------|-------------------------|-------------------|
| Circuit diagrams | ✓ (standard) | ✗ (rare) |
| Formulas (Ohm's Law) | ✓ (I = V/R) | ✗ (same result) |
| Vacuum tubes | ✗ | ✓ (actual electron movement) |
| Semiconductor theory | ✗ | ✓ (N-type/P-type) |
| Battery chemistry | ✗ | ✓ (electrons move inside) |
| Most textbooks | ✓ | ✗ |
| Troubleshooting | ✓ (easier) | ✗ |
| Arrows on diodes | ✓ (points to negative) | ✗ |

**PRACTICAL ADVICE:** Learn both. Use conventional current for circuit analysis and electron flow for understanding physical components.

### The Ammeter (Measuring Current)

```
HOW TO CONNECT AN AMMETER

Ammeters must be connected IN SERIES with the circuit.

CORRECT CONNECTION (Series):
═══════════════════════════════════════════════════════

    Battery(+) ────┬──── Ammeter ────┬──── Light ──── Battery(-)
                   │     ┌─────┐     │       Bulb
                   │     │  A  │     │
                   │     └─────┘     │
                   │                 │
                   └─────────────────┘
    
    Current flows THROUGH the ammeter
    Ammeter becomes part of the path


INCORRECT CONNECTION (Parallel - DANGEROUS!):
═══════════════════════════════════════════════════════

    Battery(+) ────┬────────────────┬──── Light ──── Battery(-)
                   │                │       Bulb
                   │     ┌─────┐    │
                   │     │  A  │    │
                   │     └─────┘    │
                   │                │
                   └────────────────┘
    
    Current bypasses the ammeter? NO!
    Ammeter creates a SHORT CIRCUIT!
    Massive current flows → blows fuse or destroys meter


AMMETER RULES:
├── ALWAYS connect in series (in line with the path)
├── NEVER connect across a voltage source (parallel)
├── Start with highest range (if unsure)
├── Observe polarity (+ to +, - to -)
└── Never exceed the meter's rating
```

## Current in Series Circuits

In a series circuit, current is the same at every point.

```
SERIES CIRCUIT - CURRENT IS CONSTANT

    ┌─────────────────────────────────────────────────┐
    │                                                 │
    │    I = 0.5 A                                    │
    │    ──►                                          │
    │       ┌─────┐    ┌─────┐    ┌─────┐             │
    │   +   │     │    │     │    │     │             │
    │  ──┼──┤ R1  ├────┤ R2  ├────┤ R3  ├──┼──        │
    │       │ 10Ω │    │ 20Ω │    │ 30Ω │             │
    │       └─────┘    └─────┘    └─────┘             │
    │   -                                             │
    │    ◄──                                          │
    │    I = 0.5 A                                    │
    │                                                 │
    └─────────────────────────────────────────────────┘

    MEASUREMENTS:
    ├── Current before R1: 0.5 A
    ├── Current between R1 and R2: 0.5 A
    ├── Current between R2 and R3: 0.5 A
    ├── Current after R3: 0.5 A
    └── Current returning to battery: 0.5 A

    WHY? Electrons have only ONE path. They cannot
    "choose" a different route. All must go through
    every component.

    FORMULA: I_total = I_1 = I_2 = I_3 = ... = I_n
```

### Series Current Examples

```
EXAMPLE 1: Christmas lights (old style)

    ┌─────┐    ┌─────┐    ┌─────┐
    │Bulb1│    │Bulb2│    │Bulb3│
    └──┬──┘    └──┬──┘    └──┬──┘
       │          │          │
       └──────────┴──────────┘
              │
            Battery

    Current = 0.3 A through EVERY bulb
    
    If one bulb burns out (opens):
    ┌─────┐    ┌─────┐    ┌─────┐
    │Bulb1│    │Bulb2│    │Bulb3│
    └──X──┘    └──┬──┘    └──┬──┘
       │          │          │
       └──────────┴──────────┘
              │
            Battery
    
    ALL lights go out! (Open circuit)


EXAMPLE 2: Calculate current in series

    Given: 12V battery, resistors: 10Ω, 20Ω, 30Ω
    
    Step 1: Find total resistance
    R_total = R1 + R2 + R3 = 10 + 20 + 30 = 60Ω
    
    Step 2: Apply Ohm's Law
    I = V / R = 12V / 60Ω = 0.2 A
    
    Result: 0.2 A flows through every resistor
```

## Current in Parallel Circuits

In a parallel circuit, current divides among the branches.

```
PARALLEL CIRCUIT - CURRENT DIVIDES

                    ┌─────────────────────┐
                    │                     │
              I_total = 1.0 A             │
                    │                     │
              ┌─────┴─────┐               │
              │           │               │
            I₁=0.5 A    I₂=0.3 A    I₃=0.2 A
              │           │               │
           ┌──┴──┐     ┌──┴──┐         ┌──┴──┐
           │ R1  │     │ R2  │         │ R3  │
           │ 24Ω │     │ 40Ω │         │ 60Ω │
           └──┬──┘     └──┬──┘         └──┬──┘
              │           │               │
              └───────────┴───────────────┘
                          │
                      Battery(-)
                          
    MEASUREMENTS:
    ├── Total current from battery: 1.0 A
    ├── Current through R1 (24Ω): 0.5 A
    ├── Current through R2 (40Ω): 0.3 A
    ├── Current through R3 (60Ω): 0.2 A
    └── I_total = I₁ + I₂ + I₃ = 1.0 A

    WHY? Electrons have MULTIPLE paths. They divide
    based on resistance (more current through less resistance).

    FORMULA: I_total = I₁ + I₂ + I₃ + ... + I_n
```

### Current Divider Rule

```
CURRENT DIVIDER FORMULA

For two resistors in parallel:

           R₂
    I₁ = ─────── × I_total
         R₁ + R₂

           R₁
    I₂ = ─────── × I_total
         R₁ + R₂


EXAMPLE: Find current through each resistor

    Given: I_total = 2 A, R₁ = 10Ω, R₂ = 30Ω
    
    I₁ = (30 / (10+30)) × 2 = (30/40) × 2 = 0.75 × 2 = 1.5 A
    
    I₂ = (10 / (10+30)) × 2 = (10/40) × 2 = 0.25 × 2 = 0.5 A
    
    Check: 1.5 A + 0.5 A = 2.0 A ✓


OBSERVATIONS:
├── Smaller resistance → MORE current
├── Larger resistance → LESS current
├── Equal resistances → EQUAL current
└── Current divides inversely to resistance ratio
```

### Parallel Current Examples

```
EXAMPLE 1: Household wiring

    All outlets and lights are in parallel.
    
    ┌─────────┐
    │ Breaker │
    │ 15 A    │
    └───┬─────┘
        │
    ┌───┼───┬───────────┬───────────┐
    │   │   │           │           │
    │  ┌┴─┐ │          ┌┴─┐        ┌┴─┐
    │  │L1│ │          │L2│        │L3│
    │  └┬─┘ │          └┬─┘        └┬─┘
    │   │   │           │           │
    └───┼───┴───────────┴───────────┘
        │
    ┌───┴───┐
    │Neutral│
    └───────┘
    
    Each light draws 0.5 A:
    ├── 1 light on: 0.5 A total
    ├── 2 lights on: 1.0 A total
    ├── 3 lights on: 1.5 A total
    └── 30 lights on: 15 A (breaker trips)


EXAMPLE 2: Calculate total current

    Given: 12V battery, three parallel resistors
    
    R₁ = 6Ω, R₂ = 12Ω, R₃ = 4Ω
    
    Step 1: Calculate each branch current
    I₁ = V / R₁ = 12 / 6 = 2 A
    I₂ = V / R₂ = 12 / 12 = 1 A
    I₃ = V / R₃ = 12 / 4 = 3 A
    
    Step 2: Add branch currents
    I_total = 2 + 1 + 3 = 6 A
    
    Step 3: Verify with equivalent resistance
    1/R_eq = 1/6 + 1/12 + 1/4 = 2/12 + 1/12 + 3/12 = 6/12
    R_eq = 2Ω
    I_total = V / R_eq = 12 / 2 = 6 A ✓
```

## AC Current vs DC Current

### Direct Current (DC)

```
DIRECT CURRENT CHARACTERISTICS

VOLTAGE OVER TIME:
     +
     │ ──────────────────────────
     │
     │
   0 ┼─────────────────────────► Time
     │
     │
     │
     -
    
    Constant polarity (always + and -)
    Constant magnitude (ideally)
    Flows one direction only


SOURCES OF DC:
├── Batteries (all types)
├── Solar cells
├── DC power supplies
├── USB chargers
├── Car electrical systems (12V DC)
└── Rectified AC (converted from wall power)


DC APPLICATIONS:
├── Electronics (phones, computers, TVs)
├── LED lighting
├── Electric vehicles
├── Battery-powered devices
└── Industrial control systems


DC CURRENT GRAPH:
    Current (A)
        │
    2A  ┼─────────────────────────
        │
    1A  ┼
        │
    0A  ┼─────────────────────────► Time
        │
   -1A  ┼
        │
   -2A  ┼
    
    Flat line at positive value = DC
```

### Alternating Current (AC)

```
ALTERNATING CURRENT CHARACTERISTICS

VOLTAGE OVER TIME (Sine Wave):
    +
    │      ╱╲
    │     ╱  ╲
    │    ╱    ╲
   0 ┼──╱──────╲───────────────► Time
    │  ╱        ╲
    │ ╱          ╲
    │╱            ╲
    -
    
    Polarity alternates (changes direction)
    Magnitude changes continuously
    Flows back and forth (50 or 60 times per second)


SOURCES OF AC:
├── Wall outlets (grid power)
├── Generators
├── Alternators (cars)
├── Transformers
└── Inverters (DC to AC)


AC APPLICATIONS:
├── Household power
├── Industrial machinery
├── Motors (fans, pumps, compressors)
├── Lighting (fluorescent, some LED)
├── Power distribution (long distances)
└── Heating (ovens, dryers, heaters)


AC CURRENT GRAPH (60 Hz):
    Current (A)
        │      ╱╲
    2A  │     ╱  ╲
        │    ╱    ╲
    1A  │   ╱      ╲
        │  ╱        ╲
    0A  ┼─╱──────────╲──────────► Time
        │ ╱            ╲
   -1A  │╱              ╲
        │                ╲
   -2A  ╱                  ╲
        │
    
    Wave crosses zero 120 times per second (60 Hz)
    Positive half-cycle: current one direction
    Negative half-cycle: current opposite direction
```

### DC vs AC Comparison Table

| Characteristic | DC | AC |
|----------------|-----|-----|
| Direction | Constant one way | Alternating (reverses) |
| Polarity | Fixed (+ and -) | Swaps 50/60 times/sec |
| Voltage magnitude | Constant | Varies (0 to peak) |

## Current Density

Current density is current per unit area. Important for wire sizing and circuit board design.

```
CURRENT DENSITY FORMULA

         I
    J = ───
         A

WHERE:
J = Current density (A/m² or A/mm²)
I = Current (A)
A = Cross-sectional area (m² or mm²)


EXAMPLES:

Wire gauge (AWG) comparison:

    AWG 18 (typical lamp cord):
    ├── Area = 0.823 mm²
    ├── Max current = 7-10 A
    └── J = 10 A / 0.823 mm² = 12.1 A/mm²
    
    AWG 12 (household wiring):
    ├── Area = 3.31 mm²
    ├── Max current = 20 A
    └── J = 20 / 3.31 = 6.0 A/mm²
    
    AWG 2 (service entrance):
    ├── Area = 33.6 mm²
    ├── Max current = 95 A
    └── J = 95 / 33.6 = 2.8 A/mm²


WHY CURRENT DENSITY MATTERS:

    Too high J = Too much heat
    ┌─────────────────────────────────┐
    │                                 │
    │  Thin wire ──► High J ──► Heat  │
    │      │                          │
    │      ▼                          │
    │  Melting insulation             │
    │      │                          │
    │      ▼                          │
    │  Fire hazard!                   │
    │                                 │
    └─────────────────────────────────┘

SAFE CURRENT DENSITIES:
├── PCB traces: 10-20 A/mm² (short term)
├── PCB traces: 5-10 A/mm² (continuous)
├── Household wiring: 4-6 A/mm²
├── Transformer windings: 2-4 A/mm²
└── Very high reliability: 1-2 A/mm²
```

## Current and Heat (Joule Heating)

When current flows through resistance, heat is generated.

```
JOULE HEATING (I²R Losses)

    Heat generated = I² × R × t
    
    WHERE:
    I = Current (A)
    R = Resistance (Ω)
    t = Time (seconds)


WHY THIS MATTERS:

    Current doubles → Heat QUADRUPLES!
    
    Example: 1A through 10Ω = 10W heat
             2A through 10Ω = 40W heat (4×)
             3A through 10Ω = 90W heat (9×)
             4A through 10Ω = 160W heat (16×)


PRACTICAL IMPLICATIONS:

    Wire sizing:
    ├── 10A wire must be thicker than 5A wire
    ├── 20A circuit needs larger wire than 15A
    └── Breakers prevent wire overheating
    
    Component ratings:
    ├── Resistors have power ratings (¼W, ½W, 1W, etc.)
    ├── Exceeding rating = smoke and fire
    └── Heat sinks needed for high current devices
    
    Efficiency:
    ├── Lower current = less heat = more efficient
    ├── High voltage transmission reduces current
    └── P = V × I, same power at higher V = lower I
```

## Measuring Current with a Multimeter

```
DIGITAL MULTIMETER (DMM) CURRENT MEASUREMENT

STEP-BY-STEP:

Step 1: Identify current jacks on multimeter
    ┌─────────────────────────┐
    │    ┌─────┐ ┌─────┐     │
    │    │ COM │ │ VΩ  │     │
    │    │  -  │ │  +  │     │
    │    └──┬──┘ └──┬──┘     │
    │       │       │         │
    │       │       │         │
    │    ┌──┴──┐ ┌──┴──┐     │
    │    │ 10A │ │ mA  │     │
    │    │     │ │ µA  │     │
    │    └─────┘ └─────┘     │
    └─────────────────────────┘
    
    COM = Common (black lead)
    mA = Up to 400 mA (red lead for small currents)
    10A = Up to 10 A (red lead for large currents)

Step 2: Select current range
    ├── Turn dial to A (Amps) or mA/µA
    ├── Choose AC or DC (DC for batteries, AC for wall)
    └── Start with highest range if unsure

Step 3: Break the circuit
    ┌─────┐        ┌─────┐
    │     │        │     │
    │     │        │     │
    │     │        │     │
    └──┬──┘        └──┬──┘
       │              │
       │   OPEN       │
       │   HERE       │
       │              │
       └──────────────┘
    
    You must open the circuit and insert the meter!

Step 4: Connect meter in series
    ┌─────┐        ┌─────────────┐        ┌─────┐
    │     │        │    METER    │        │     │
    │     ├────────┤    COM      ├────────┤     │
    │     │        │    A        │        │     │
    │     │        └─────────────┘        │     │
    └─────┘                               └─────┘
    
    Current flows FROM circuit ──► COM
                 THROUGH meter ──► A terminal
                 BACK to circuit

Step 5: Read the display
    ├── If reading is "OL" (overload), move to higher range
    ├── If reading is very low, move to lower range
    └── Note the units (A, mA, or µA)


COMMON MISTAKES:

✗ Measuring current with leads in voltage jacks
  → Blows internal fuse!

✗ Connecting meter in parallel (across component)
  → Short circuit! Large current! Fuse blows!

✗ Using mA jack for >400 mA current
  → Fuse blows (or meter destroyed)

✗ Forgetting to move leads back to voltage jacks
  → Next voltage measurement shorts the source!

✓ Always return leads to V/Ω jacks after current measurement
```

## Kirchhoff's Current Law (KCL)

One of the most important laws for understanding current.

```
KIRCHHOFF'S CURRENT LAW

"The sum of currents entering a junction equals the sum of currents leaving."

    I₁ ──►      ┌─────┐
               │     │
    I₂ ──►     │ JCT │
               │     │
    I₃ ──►     └─────┘
                  │
                  ▼ I₄


FORMULA: I₁ + I₂ + I₃ = I₄

Or more generally: Σ I_in = Σ I_out


PRACTICAL EXAMPLES:

Example 1: Simple junction
    I₁ = 2A ──► ┌─────┐
                │     │
    I₂ = 3A ──► │     │
                │     │
                └─────┘
                   │
                   ▼ I₃ = 5A
    
    2A + 3A = 5A ✓


Example 2: One input, multiple outputs
              ┌─────┐
    I₁ = 10A ─┤     ├──► I₂ = 4A
              │     │
              │     ├──► I₃ = 3A
              │     │
              └─────┴──► I₄ = 3A
    
    10A = 4A + 3A + 3A ✓


Example 3: Complex node (with directions)
    
    I₁ = 5A ──►
                ┌─────┐
    I₂ = 3A ◄───┤     │ (I₂ is entering? No, leaving!)
                │     │
    I₃ = 2A ──► └─────┘
                   │
                   ▼ I₄ = ?
    
    Careful with directions!
    Assume entering = positive, leaving = negative
    
    I₁ + I₃ - I₂ = I₄
    5A + 2A - 3A = 4A
    
    I₄ = 4A leaving ✓
```

## Summary

1. **Current** is the flow of electric charge (electrons)

2. **Two conventions:**
   - Conventional current: Positive to negative (used in diagrams)
   - Electron flow: Negative to positive (physical reality)

3. **Unit of current:** Ampere (A) = 1 Coulomb per second = 6.24 × 10¹⁸ electrons/second

4. **Series circuits:** Current is the same everywhere

5. **Parallel circuits:** Current divides among branches (I_total = I₁ + I₂ + ...)

6. **DC (Direct Current):** Flows one direction, constant magnitude (batteries)

7. **AC (Alternating Current):** Reverses direction, varies magnitude (wall outlets)

8. **Current divider rule:** Smaller resistance = more current

9. **Kirchhoff's Current Law:** Current in = Current out at any junction

10. **Joule heating:** Heat = I² × R (current quadruples heat)

11. **Measure current** by connecting ammeter in SERIES (never parallel)

12. **Higher voltage** allows lower current for same power (reduces losses)
