# 03.05 - Diodes

## What is a Diode?

A diode is a two-terminal semiconductor device that conducts current primarily in one direction. It acts as an electrical one-way valve, allowing current to flow freely in the forward direction while blocking current in the reverse direction.

The diode was discovered in 1874 by Karl Ferdinand Braun, who observed that certain crystals (galena) conducted current in only one direction. This "cat's whisker" detector was used in early radio receivers. The modern semiconductor diode was developed in the 1940s, leading to the invention of the transistor and all modern electronics.

### I-V Characteristic Curve

The current-voltage relationship of a diode is nonlinear and exponential.

```
DIODE I-V CURVE

    Current (I)
        │
    100mA┼                                          /
        │                                         /
     50mA┼                                       /
        │                                      /
     10mA┼                                    /
        │                                   /
      5mA┼                                 /
        │                                /
      1mA┼──────────────────────────────●────────────────►
        │                             ╱
        │                           ╱
        │                         ╱
        │                       ╱
        │                     ╱
        │                   ╱
        │                 ╱
        │               ╱
        │             ╱
        │           ╱
        │         ╱
        │       ╱
        │     ╱
        │   ╱
        │ ╱
       0┼───────────────────────────────────────────────► Voltage (V)
       │   -100V  -50V    -1V   0   0.5V  1.0V  1.5V
       │
       │   Reverse bias        Forward bias
       │
       │   (tiny leakage current)


KEY REGIONS:

    Region              Voltage        Current        Behavior
    ──────────────────────────────────────────────────────────────
    Forward bias        V > 0.6V       Exponential    Conducting
    Forward threshold   0.5-0.7V       ~1mA           Start conducting
    Reverse bias        V < 0V         ~nA-µA         Blocking
    Reverse breakdown   V < -V_BR      Very high      Avalanche (conducts!)
```

### Diode Equation

The Shockley diode equation describes the I-V relationship.

```
SHOCKLEY DIODE EQUATION

    I = I_S × (e^(V_D / (n × V_T)) - 1)

    WHERE:
    I = diode current (A)
    I_S = reverse saturation current (very small, ~nA to µA)
    V_D = voltage across diode (V)
    n = ideality factor (1 for ideal, 1-2 for real)
    V_T = thermal voltage = kT/q ≈ 26mV at room temperature (25°C)


SIMPLIFIED RULES (Practical use):

    Silicon diode (most common):
    ├── Forward voltage drop ≈ 0.6V - 0.7V at ~1mA
    ├── Forward voltage drop ≈ 0.7V - 1.0V at ~100mA
    ├── Forward voltage drop ≈ 1.0V - 1.2V at ~1A
    └── Reverse leakage ≈ 1nA - 10µA (depends on type)

    Schottky diode:
    ├── Forward voltage drop ≈ 0.2V - 0.4V
    ├── Faster switching
    └── Higher reverse leakage

    Germanium diode (obsolete):
    ├── Forward voltage drop ≈ 0.2V - 0.3V
    └── High reverse leakage, low temperature stability
```

### Diode Symbols

```
CIRCUIT SYMBOLS

Standard diode (PN junction):
    Anode ──►││──► Cathode
    
    Or:     ────|>│────
    
    Arrow points in direction of conventional current

Schottky diode:
    ────|>│────  (similar to standard, sometimes with S symbol)
    Or:   ────|>│─S─

Zener diode:
    ────|>│─Z─   or   ────|>│─── with bent cathode
    
    ────|>┴─   (cathode looks like Z)

LED (Light Emitting Diode):
    ────|>│─◯─   or   ────|>│─▶─  (with arrows for light)

Photodiode:
    ────|>│─◯─   with arrows pointing IN (light hits diode)
    (opposite of LED)

Tunnel diode:
    ────|>│─  with extra line (looks like E or special symbol)

Varactor (Varicap) diode:
    ────|>│─  with capacitor symbol (two parallel lines)
```

### Schottky Diode

Fast switching, low forward voltage drop.

```

### Voltage Regulation (Zener)

Zener diodes maintain a constant voltage across a load.

```
ZENER VOLTAGE REGULATOR

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Vin (unregulated) ──┬────[R]────┬─── Vout (regulated)    │
    │                       │           │                        │
    │                       │          ┌┴┐                       │
    │                       │          │ │ Zener                  │
    │                       │          │ │ (Vz)                   │
    │                       │          └┬┘                       │
    │                       │           │                        │
    │   GND ────────────────┴───────────┴────────────────────────│
    │                                                             │
    └─────────────────────────────────────────────────────────────┘

    Operation:
    ├── If Vout tries to rise, Zener conducts more current
    ├── More current through R → more voltage drop across R
    ├── Vout stays constant (Vz)
    └── Works only if Vin > Vz + some margin


DESIGN EXAMPLE:

    Need: 5V regulated from 9V battery, load current 0-50mA
    
    Choose: 1N4733A (5.1V Zener)
    Choose Zener current: Iz_min = 5mA (knee current)
    
    R = (Vin_min - Vz) / (Iz_min + I_load_max)
    R = (9 - 5.1) / (0.005 + 0.05) = 3.9 / 0.055 = 71Ω
    
    Use standard value: 68Ω or 75Ω
    
    Power in R: P_R = I_total² × R = 0.055² × 68 = 0.206W (use 1/2W)
    Power in Zener: P_Z = Vz × Iz_max = 5.1 × 0.055 = 0.28W (use 1/2W)


LIMITATIONS:
├── Inefficient (wastes power in R and Zener)
├── Limited to low current (<100mA)
├── Requires Vin > Vz
├── Output voltage not adjustable
└── For higher current, use linear regulator (7805) or switching supply
```

```

    │                                                             │

## Diode Specifications

### Key Parameters

```
DIODE DATASHEET PARAMETERS

FORWARD VOLTAGE (V_F):
    Voltage drop when conducting forward current
    Silicon: 0.6V - 1.2V (depends on current)
    Schottky: 0.2V - 0.5V
    LED: 1.8V - 3.6V (depends on color)


REVERSE VOLTAGE (V_RRM or V_R):
    Maximum reverse voltage that can be applied without breakdown
    Exceeding may destroy diode (except Zener, which is designed for it)


FORWARD CURRENT (I_F(AV)):
    Maximum average forward current (DC)
    For 1N4007: 1A
    For 1N5408: 3A


PEAK FORWARD CURRENT (I_FSM):
    Maximum surge current (short pulse, e.g., 8.3ms half-cycle)
    1N4007: 30A
    1N5408: 200A


REVERSE LEAKAGE (I_R):
    Small current that flows in reverse bias (before breakdown)
    Silicon: 1nA - 10µA (increases with temperature)
    Schottky: 0.1mA - 1mA (higher than silicon)


REVERSE RECOVERY TIME (t_rr):
    Time to switch from conducting to blocking
    Standard silicon: 1-100µs
    Fast recovery: 50-500ns
    Schottky: <10ns
    For power line rectification (50/60Hz): any is fine
    For switching supplies (50kHz-1MHz): need fast recovery or Schottky


JUNCTION CAPACITANCE (C_J):
    Capacitance of reverse-biased junction
    Small signal: 1-10pF
    Power: 10-100pF
    Affects high-frequency performance


POWER DISSIPATION (P_D):
    Maximum power the diode can dissipate (heat)
    P_D = V_F × I_F
    For 1N4007: P_D = 1.0V × 1A = 1W (theoretical, derated by temperature)
```

## Measuring Diodes

### Using a Multimeter (Diode Mode)

```
DIODE TESTING

STEP 1: Set multimeter to diode mode (symbol: ─►││─)

STEP 2: Connect leads:
    ├── Red lead to anode (+)
    ├── Black lead to cathode (-)
    └── (Or use component tester socket)

STEP 3: Read forward voltage


INTERPRETING RESULTS:

    Good silicon diode:
    ├── Forward reading: 0.5V - 0.8V (depends on current)
    ├── Reverse reading: "OL" (open) or very high
    └── Example: 0.62V forward, OL reverse

    Good Schottky diode:
    ├── Forward reading: 0.15V - 0.4V
    ├── Reverse reading: "OL" or high (may show some leakage)
    └── Example: 0.25V forward, 1.0MΩ reverse

    Good LED:
    ├── Forward reading: 1.8V - 3.6V (may light dimly)
    ├── Reverse reading: "OL"
    └── Example: 2.1V forward (green LED)

    Shorted diode:
    ├── Forward reading: 0.00V or very low (0.001V)
    ├── Reverse reading: 0.00V or very low
    └── Diode failed short (replace)

    Open diode:
    ├── Forward reading: "OL"
    ├── Reverse reading: "OL"
    └── Diode failed open (replace)

    Leaky diode:
    ├── Forward reading: normal (0.6V)
    ├── Reverse reading: low resistance (10kΩ - 100kΩ)
    └── Replace (may cause circuit problems)


TESTING WITHOUT DIODE MODE (Resistance mode):

    Set to 20kΩ range (or auto)
    Forward: Should read low resistance (few hundred ohms)
    Reverse: Should read high resistance (MΩ or OL)
    
    Not as accurate (depends on meter voltage)
```

## Common Diode Problems

### Problem 1: Shorted Diode

```
SYMPTOMS:
├── Diode conducts both directions (measures 0V in diode mode)
├── Circuit draws excessive current
├── Rectifier output has AC ripple (missing half-cycles)
├── Power supply voltage low or zero
├── Fuse blows or transformer overheats

CAUSES:
├── Overvoltage (exceeded V_RRM)
├── Overcurrent (exceeded I_FSM)
├── Excessive temperature (thermal runaway)
├── Reverse voltage spike (no snubber on inductive load)
├── Manufacturing defect

DIAGNOSIS:
├── Diode mode: reads 0.00V both directions
├── Resistance mode: reads 0Ω both directions
├── In-circuit: voltage drop = 0V when forward biased
├── Bridge rectifier: shorted diode causes high current

SOLUTIONS:
├── Replace diode
├── Check for cause (overvoltage, overcurrent)
├── Add snubber or flyback diode for inductive loads
├── Use higher voltage rating (e.g., 1N4007 instead of 1N4004)
├── Use higher current rating
├── Add heatsink if overheating
├── For bridge: replace entire bridge module
```

### Problem 2: Open Diode

```
SYMPTOMS:
├── Diode reads "OL" both directions
├── No current flow in that branch
├── Rectifier output half-wave (missing half of cycles)
├── Circuit doesn't work (no power)

CAUSES:
├── Overcurrent (burned open like a fuse)
├── Mechanical stress (broken lead)
├── Solder stress (cracked)
├── Old age (rare for silicon)

DIAGNOSIS:
├── Diode mode: "OL" both directions
├── Resistance mode: infinite both directions
├── In-circuit: voltage across diode = supply voltage (no drop)

SOLUTIONS:
├── Replace diode
├── Check for overcurrent condition
├── Use higher current rating
├── Add mechanical support (strain relief)
├── Avoid bending leads close to body
```

### Problem 3: Leaky Diode (High Reverse Current)

```
SYMPTOMS:
├── Reverse resistance lower than spec (e.g., 10kΩ instead of MΩ)
├── Circuit draws excess current (but not short)
├── Rectifier runs hot
├── Output voltage lower than expected
├── Noisy operation (for switching supplies)

CAUSES:
├── Overheating (damaged junction)
├── Overvoltage (partial breakdown)
├── Old age (especially Schottky)
├── Manufacturing defect
├── ESD damage

DIAGNOSIS:
├── Diode mode: normal forward, but reverse reads low voltage
├── Resistance mode: forward normal, reverse low (kΩ range)
├── Leakage increases with temperature (warm diode gets leakier)

SOLUTIONS:
├── Replace diode
├── Use higher voltage rating
├── Improve cooling (heatsink, ventilation)
├── For Schottky: derate voltage more (use 2× safety margin)
├── Add reverse protection (TVS across diode)
```

## Summary

1. **Diode** conducts current in one direction only (one-way valve)

2. **Forward bias** (anode > cathode): conducts, V_F ≈ 0.7V (silicon)

3. **Reverse bias** (cathode > anode): blocks, tiny leakage only

4. **Shockley equation:** I = I_S × (e^(V/nV_T) - 1) (exponential I-V)

5. **Silicon rectifier** (1N4007): 1A, 1000V, general purpose

6. **Schottky diode:** Low V_F (0.2-0.4V), fast switching, higher leakage

7. **Zener diode:** Operates in reverse breakdown; voltage regulator

8. **LED:** Emits light when forward biased; V_F depends on color (1.8V-3.6V)

9. **Bridge rectifier:** 4 diodes in one package; full-wave rectification

10. **Flyback diode** protects switches from inductive kickback (V = L × dI/dt)

11. **TVS diode** protects circuits from ESD and voltage spikes

12. **Rectification:** AC → DC (half-wave: 1 diode, full-wave bridge: 4 diodes)

13. **Voltage regulation:** Zener + resistor maintains constant voltage (inefficient)

14. **Reverse polarity protection:** Series diode (drops voltage) or shunt + fuse (no drop)

15. **Testing:** Diode mode (0.6V forward, OL reverse) = good

16. **Failures:** Short (0V both ways), open (OL both ways), leaky (low reverse resistance)
