# 01.01 - What is Electricity?

## What is Electricity?

Electricity is the flow of electric charge. It is a fundamental form of energy resulting from the existence of charged particles (electrons and protons). In simple terms, electricity is the movement of electrons from one atom to another through a conductive path.

The name comes from the Greek word *"ēlektron"*, meaning amber, because ancient Greeks discovered that rubbing amber against fur created an attractive force (static electricity).

## Basic Concepts

### Atomic Structure
To understand electricity, you must first understand the atom:

```text
                    NUCLEUS
                   (+ charge)
                  /         \
                 /   PROTONS  \
                /    (+ charge)\
               |     NEUTRONS   |
               |    (no charge) |
                \              /
                 \            /
                  \          /
                   \________/
                        |
                        |
                   ELECTRON CLOUD
                    (- charge)
                   (orbits nucleus)
```

| Particle | Location | Charge | Role in Electricity |
|----------|----------|--------|---------------------|
| Proton | Nucleus | Positive (+) | Fixed in place |
| Neutron | Nucleus | Neutral (0) | Structural only |
| Electron | Shells | Negative (-) | **Moves to create current** |

### The Principle of Charges

```
LIKE CHARGES REPEL        OPPOSITE CHARGES ATTRACT

    (+)      (+)              (+)      (-)
     \       /                  \     /
      \     /                    \   /
       \   /                      \ /
        \ /                        \
         X                         (attract)
        / \                        /
       /   \                      /
      /     \                    /
     /       \                  /
    (+)      (+)              (+)      (-)
   
   REPULSION                 ATTRACTION
```

**Fundamental Laws:**
- Like charges repel (++ or --)
- Opposite charges attract (+-)
- Charges want to balance (neutralize)

## How Electricity Works

### The Electron Flow Model

```text
CONDUCTOR (e.g., Copper Wire)

Before connection:
Atom 1    Atom 2    Atom 3    Atom 4
  e-        e-        e-        e-
  |         |         |         |
 [Cu]      [Cu]      [Cu]      [Cu]

During current flow:
Atom 1    Atom 2    Atom 3    Atom 4
  |         |         |         |
 [Cu] --> [Cu] --> [Cu] --> [Cu]
  |         |         |         |
  e-  -->   e-  -->   e-  -->   e-
(flow)

Electrons move from negative to positive terminal
```

| Term | Symbol | Unit | Definition |
|------|--------|------|-------------|
| **Voltage** | V | Volt (V) | Electrical pressure / potential difference |
| **Current** | I | Ampere (A) | Flow rate of electrons |

## Types of Electricity

### 1. Static Electricity
Electric charge at rest. Built up on a surface through friction.

```text
STATIC ELECTRICITY EXAMPLE

Step 1: Rub balloon on hair
    Balloon              Hair
    (neutral)           (neutral)
        \                 /
         \               /
          \             /
           \           /
            \         /
             \       /
              \     /
               \   /
                \ /
    [RUBBING CREATES FRICTION]

Step 2: Electrons transfer
    Balloon              Hair
    (NEGATIVE) ------> (POSITIVE)
    (extra e-)          (missing e-)

Step 3: Attraction
    (-) BALLOON
         \
          \
           \
            \
             (+) HAIR
    
    Balloon sticks to hair (opposite charges attract)
```

**Examples:** Lightning, static shock from doorknob, clothes sticking together from dryer

### 2. Current Electricity
Flow of electric charge through a conductor.

```text
CURRENT ELECTRICITY

DIRECT CURRENT (DC)          ALTERNATING CURRENT (AC)
────────────────────          ────────────────────────

    +5V ┤                    +5V ┤╱╲╱╲╱╲
        │                         ╱  ╲  ╱
     0 ─┼──────►              0 ─┼──────►
        │                         ╲  ╱  ╲
    -5V ┤                         ╲╱╲╱╲╱

Constant direction           Direction reverses
(Battery, solar cell)        (Wall outlet, generator)

FLOW: ───────►               FLOW: ◄───►◄───►
(one way)                    (back and forth)
```

| Characteristic | DC (Direct Current) | AC (Alternating Current) |
|----------------|---------------------|--------------------------|
| Direction | Constant | Alternating |
| Voltage | Fixed (e.g., 5V, 12V) | Sinusoidal (e.g., 120V RMS) |
| Frequency | 0 Hz | 50/60 Hz |
| Sources | Batteries, solar cells | Power grid, generators |
| Applications | Electronics, cars | Homes, industrial |

## Ohm's Law (The Most Important Formula)

Ohm's Law describes the relationship between voltage, current, and resistance.

```text
OHM'S LAW TRIANGLE

        ┌─────────┐
        │    V    │
        │  ─────  │
        │  I × R  │
        └─────────┘

    V = I × R          Voltage = Current × Resistance
    I = V / R          Current = Voltage ÷ Resistance
    R = V / I          Resistance = Voltage ÷ Current

WHERE:
V = Voltage (Volts)
I = Current (Amperes)
R = Resistance (Ohms, Ω)
```

### Practical Examples

```text
EXAMPLE 1: Find Current
────────────────────────
Given: 9V battery, 300Ω resistor

I = V / R
I = 9V / 300Ω
I = 0.03A = 30mA

   9V ──┬────[300Ω]────┐
        │              │
        └──────────────┘
        
Current = 30 milliamps


EXAMPLE 2: Find Resistance
──────────────────────────
Given: 12V battery, need 0.5A current

R = V / I
R = 12V / 0.5A
R = 24Ω

   12V ──┬────[24Ω]────┐
        │              │
        └──────────────┘

Use 24Ω resistor for 0.5A


EXAMPLE 3: Find Voltage
───────────────────────
Given: 2A current through 10Ω resistor

V = I × R
V = 2A × 10Ω
V = 20V

   20V ──┬────[10Ω]────┐
        │              │
        └──────────────┘
        (2A flows)
```

## Electrical Power

Power is the rate at which electrical energy is converted.

```text
POWER FORMULAS

    P = V × I          Power = Voltage × Current
    P = I² × R         Power = Current² × Resistance
    P = V² / R         Power = Voltage² ÷ Resistance

WHERE:
P = Power (Watts)
V = Voltage (Volts)
I = Current (Amperes)
R = Resistance (Ohms)

EXAMPLES:

1. 12V, 2A → P = 12 × 2 = 24 watts

2. 5V, 0.5A → P = 5 × 0.5 = 2.5 watts

3. 10Ω, 2A → P = 2² × 10 = 40 watts
```

### Resistance and Resistivity

```text
FACTORS AFFECTING RESISTANCE

1. MATERIAL (ρ = resistivity)
   Copper: low resistance
   Nichrome: high resistance

2. LENGTH (L)
   Longer wire = More resistance
   Shorter wire = Less resistance

3. THICKNESS (A = cross-sectional area)
   Thicker wire = Less resistance
   Thinner wire = More resistance

4. TEMPERATURE (T)
   Hotter = More resistance (metals)
   Hotter = Less resistance (semiconductors)

FORMULA:
         L
    R = ρ ─
         A

WHERE:
R = Resistance (Ω)
ρ = Resistivity (Ω·m)
L = Length (m)
A = Cross-sectional area (m²)
```

## Safety and Real-World Context

### Voltage Levels and Danger

```text
VOLTAGE RANGES

   1.5V ──► Safe (AA battery)
     5V ──► Safe (USB, Arduino)
    12V ──► Safe (Car battery)
    24V ──► Safe (Industrial control)
    48V ──► Safe limit (Telecom)
    120V ──► DANGEROUS (US wall outlet)
    230V ──► DANGEROUS (EU wall outlet)
   1kV+ ──► EXTREMELY DANGEROUS (Power lines)

SAFETY RULES:
├── Never work on live circuits (when possible)
├── Use one hand when troubleshooting
├── Remove metal jewelry
├── Use insulated tools
├── Check for absence of voltage before touching
└── Understand that current kills, not voltage
```

### Effects of Current on Human Body

| Current (mA) | Effect |
|--------------|--------|
| < 1 mA | Usually not perceptible |
| 1-5 mA | Mild tingling sensation |
| 5-15 mA | Painful shock, muscle contraction |
| 15-20 mA | "Let-go" threshold (can't release grip) |
| 20-50 mA | Severe muscle contraction, breathing difficulty |
| 50-100 mA | Ventricular fibrillation (often fatal) |
| > 100 mA | Severe burns, cardiac arrest |

## Summary

1. **Electricity** is the flow of electrons from atom to atom

2. **Three key quantities:**
   - Voltage (V) = Electrical pressure
   - Current (I) = Electron flow rate
   - Resistance (R) = Opposition to flow

3. **Ohm's Law:** V = I × R (the foundation of electronics)

4. **Two types:**
   - DC: Constant direction (batteries)
   - AC: Alternating direction (wall outlets)

5. **Circuits must be complete loops** for current to flow

6. **Materials:**
   - Conductors: Allow flow (copper, aluminum)
   - Insulators: Block flow (rubber, plastic)
   - Semiconductors: Controlled flow (silicon)

7. **Safety:** Current kills, respect electricity, use proper precautions

## Basic Template (Simple Circuit Calculator)

```python
#!/usr/bin/env python3

class ElectricityBasics:
    """Simple electrical calculator"""
    
    @staticmethod
    def ohms_law(v=None, i=None, r=None):
        """Calculate missing value using Ohm's Law"""
        if v is None:
            return i * r  # V = I × R
        elif i is None:
            return v / r  # I = V / R
        elif r is None:
            return v / i  # R = V / I
        else:
            raise ValueError("Provide exactly two parameters")
    
    @staticmethod
    def power(v=None, i=None, r=None):
        """Calculate electrical power"""
        if v and i:
            return v * i  # P = V × I
        elif i and r:
            return i**2 * r  # P = I² × R
        elif v and r:
            return v**2 / r  # P = V² / R
        else:
            raise ValueError("Provide V&I, I&R, or V&R")
    
    @staticmethod
    def resistor_color_code(colors):
        """Decode 4-band resistor"""
        codes = {
            'black': 0, 'brown': 1, 'red': 2, 'orange': 3,
            'yellow': 4, 'green': 5, 'blue': 6, 'violet': 7,
            'gray': 8, 'white': 9
        }
        multipliers = {
            'black': 1, 'brown': 10, 'red': 100, 'orange': 1000,
            'yellow': 10000, 'green': 100000, 'blue': 1000000
        }
        
        value = (codes[colors[0]] * 10 + codes[colors[1]]) * multipliers[colors[2]]
        return value
    
    @staticmethod
    def led_resistor(led_voltage, led_current, supply_voltage):
        """Calculate resistor for LED"""
        # R = (Vsupply - Vled) / Iled
        return (supply_voltage - led_voltage) / led_current

# Example usage
if __name__ == "__main__":
    calc = ElectricityBasics()
    
    # Ohm's Law examples
    print("=== OHM'S LAW ===")
    print(f"V = 9V, R = 300Ω → I = {calc.ohms_law(v=9, r=300):.3f}A")
    print(f"I = 0.5A, R = 24Ω → V = {calc.ohms_law(i=0.5, r=24)}V")
    print(f"V = 12V, I = 2A → R = {calc.ohms_law(v=12, i=2)}Ω")
    
    # Power examples
    print("\n=== POWER ===")
    print(f"V = 12V, I = 2A → P = {calc.power(v=12, i=2)}W")
    print(f"I = 2A, R = 10Ω → P = {calc.power(i=2, r=10)}W")
    
    # LED resistor example
    print("\n=== LED CALCULATOR ===")
    r_led = calc.led_resistor(2.0, 0.02, 9.0)  # Red LED: 2V, 20mA, 9V supply
    print(f"Resistor for LED: {r_led:.0f}Ω (use 330Ω or 390Ω)")
```
