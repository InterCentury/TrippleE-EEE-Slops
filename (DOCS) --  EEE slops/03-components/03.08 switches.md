# 03.08 - Switches

## What is a Switch?

A switch is an electrical component that can make (close) or break (open) an electrical circuit, interrupting or diverting the flow of electric current. Switches are the primary means of human interaction with electrical and electronic systems, allowing users to control power, select modes, and input data.

The first electrical switch was developed alongside early telegraph systems in the 1830s and 1840s. Since then, switches have evolved into thousands of varieties, from the simple light switch on a wall to the complex key switches on a computer keyboard and the tiny push buttons on a smartphone.

## Basic Switch Operation

### How a Switch Works

A switch works by physically moving conductive contacts together or apart.

```
SWITCH BASIC STRUCTURE

    OPEN SWITCH (OFF):                    CLOSED SWITCH (ON):
    
    ┌─────────────────────┐              ┌─────────────────────┐
    │                     │              │                     │
    │   Terminal 1        │              │   Terminal 1        │
    │        │            │              │        │            │
    │        │            │              │        │            │
    │        │            │              │        │            │
    │        └─┐          │              │        └──────┐     │
    │          │ (gap)    │              │               │     │
    │        ┌─┘          │              │        ┌──────┘     │
    │        │            │              │        │            │
    │        │            │              │        │            │
    │   Terminal 2        │              │   Terminal 2        │
    │                     │              │                     │
    └─────────────────────┘              └─────────────────────┘
    
    Current CANNOT flow               Current CAN flow
    Circuit is OFF                    Circuit is ON


SWITCH TERMINOLOGY

    Pole:   Number of independent circuits the switch can control
            (Single pole = 1 circuit, Double pole = 2 circuits)
    
    Throw:  Number of positions/connections per pole
            (Single throw = ON/OFF, Double throw = ON/ON or ON/OFF/ON)
    
    Position: Number of stable states (e.g., 2-position, 3-position)
    
    Momentary: Returns to normal position when released
    Maintained: Stays in position until changed
```

## Switch Electrical Characteristics

### Contact Resistance

The resistance across closed contacts; should be as low as possible.

```
CONTACT RESISTANCE

    Good switch: <0.1Ω (100mΩ)
    Acceptable: <0.5Ω
    Marginal: 0.5-2Ω (cleaning may help)
    Failed: >2Ω or intermittent


WHAT CAUSES HIGH CONTACT RESISTANCE:

    ├── Oxidation on contact surfaces
    ├── Contamination (dirt, oil)
    ├── Worn contacts (arcing damage)
    ├── Insufficient contact pressure (worn spring)
    └── Low current (self-cleaning not activated)


MEASURING CONTACT RESISTANCE:

    Use 4-wire Kelvin measurement for accuracy
    Or subtract lead resistance from multimeter reading
    
    Example: Multimeter shows 1.2Ω with leads shorted (0.3Ω)
    Contact resistance = 1.2Ω - 0.3Ω = 0.9Ω
```

```

    ──────────────────────────────────────────────────────────────
    Resistive (heater, lamp)      1.0 (100% of rating)
    Inductive (motor, solenoid)   0.3-0.5 (30-50%)
    Lamp (incandescent)           0.2-0.3 (20-30%) (high inrush)

    Example: 10A AC-rated switch with DC motor load

```

### Insulation Resistance

Resistance between open contacts and between contacts and ground.

```
INSULATION RESISTANCE

    Good: >100 MΩ (megohms)
    Acceptable: >10 MΩ
    Marginal: 1-10 MΩ (clean if possible)
    Failed: <1 MΩ (replace)


    WHEN INSULATION RESISTANCE MATTERS:

    ├── High impedance circuits (leakage affects readings)
    ├── High voltage circuits (leakage can cause shock)
    ├── Battery-powered devices (drains battery)
    └── Medical equipment (patient safety)


    CAUSES OF LOW INSULATION RESISTANCE:

    ├── Contamination (flux residue, moisture)
    ├── Carbon tracking (arcing residue)
    ├── Damaged insulation
    └── Dust and humidity
```

### Dielectric Strength

Voltage the switch can withstand without breaking down.

```
DIELECTRIC STRENGTH (HIPOT)

    Rating indicates maximum voltage between:
    ├── Open contacts (switch off)
    ├── Contacts to ground (case, frame)
    ├── Between poles (DPDT, etc.)


    TYPICAL VALUES:

    Switch Type               Dielectric Strength
    ──────────────────────────────────────────────────────────────
    Signal (low voltage)      500V-1000V
    General purpose           1500V (UL standard)
    Power (250V rated)        1500V-2500V
    Industrial (480V rated)   2500V-4000V
    Medical (high isolation)  4000V-6000V


    IMPORTANCE:

    ├── Safety: Prevents shock through switch body
    ├── Arc prevention: Avoids internal flashover
    └── Reliability: Withstands voltage spikes
```

## Measuring and Testing Switches

### Continuity Test

```
CONTINUITY TEST (Multimeter)

    Set to resistance (Ω) or continuity (beep) mode

    CLOSED (ON) position:
    ├── Meter should read <1Ω (0.1Ω-0.5Ω typical)
    ├── Continuity beeper should sound
    └── If >2Ω or no beep → high resistance (clean or replace)

    OPEN (OFF) position:
    ├── Meter should read "OL" (open, infinite)
    ├── Continuity beeper should NOT sound
    └── If beeps or low resistance → welded contacts (replace)


    TESTING ALL POSITIONS (SPDT, DPDT, Rotary):

    For each switch position:
    ├── Check COM to NO (should be closed in one position)
    ├── Check COM to NC (should be closed in other position)
    ├── Verify no continuity between NO and NC
    └── For DPDT: Test both poles independently


    CONTACT RESISTANCE VARIATION:

    Press and release switch multiple times
    Reading should be consistent (no intermittent opens)
    Variation indicates dirty or worn contacts
```

### Insulation Test

```
INSULATION RESISTANCE TEST (Megohmmeter)

    Test voltage: 500V for most switches (or 250V for low voltage)

    Test points:
    ├── Between open contacts (switch OFF)
    ├── Between contacts and frame/case
    ├── Between poles (DPDT, etc.)
    └── Between contacts and terminals (different circuits)


    EXPECTED READINGS:

    New, clean:      >100 MΩ
    Acceptable:      >10 MΩ
    Marginal:        1-10 MΩ (clean)
    Failed:          <1 MΩ (replace)


    WARNING: Do not perform insulation test on sensitive electronics!
    Remove switch from circuit before testing.
```

## Common Switch Problems

### Problem 1: Intermittent Operation

```
SYMPTOMS:
├── Device works sometimes, fails others
├── Wiggling switch changes operation
├── Lights flicker when switch is "on"
├── Intermittent continuity when testing

CAUSES:
├── Dirty or oxidized contacts
├── Worn contacts (arcing damage)
├── Loose terminal connection
├── Broken internal spring (low contact pressure)
├── Contamination (dust, oil, flux)

DIAGNOSIS:
├── Operate switch 10-20 times (wiping action cleans contacts)
├── Measure contact resistance while moving actuator
├── Check terminal screws (tighten if loose)
├── For low current circuits (<10mA): gold contacts required

SOLUTIONS:
├── Operate switch several times (wipes contacts clean)
├── For large switches: contact cleaner spray
├── Tighten terminal screws
├── Replace switch if intermittent persists
├── For low-level signals: use gold-plated contacts
├── Use sealed switch in dirty environments

PREVENTION:
├── Use switch rated for actual load (not oversized)
├── For DC loads: derate rating 30-50%
├── Use sealed switch in humid/dusty areas
├── Apply dielectric grease to terminals
```

### Problem 2: Switch Sticks or Jams

```
SYMPTOMS:
├── Actuator (lever, button) hard to move
├── Switch does not return to normal position
├── Grinding or rough feel when operating
├── Actuator stuck in one position

CAUSES:
├── Mechanical obstruction (dirt, debris)
├── Melted or deformed plastic (overheating)
├── Broken internal spring
├── Bent actuator (physical damage)
├── Worn or broken internal parts

DIAGNOSIS:
├── Visual inspection (debris, damage)
├── Check for overheating (melted plastic)
├── Compare feel to known good switch
├── Check mounting (overtightened screws distort body)

SOLUTIONS:
├── Clean with compressed air or contact cleaner
├── For overheating: replace with higher current switch
├── Replace switch if damaged (cannot repair internal)
├── Loosen mounting screws (should not distort case)
├── For sealed switches: cannot clean internally (replace)

PREVENTION:
├── Don't exceed current rating (prevents overheating)
├── Use switch within environmental rating
├── Don't overtighten mounting hardware
├── Use sealed switch in dirty environments
```

### Problem 3: High Contact Resistance

```
SYMPTOMS:
├── Device operates but poorly (dim light, slow motor)
├── Voltage drop across switch when ON
├── Switch gets warm or hot
├── Intermittent operation

CAUSES:
├── Oxidized contacts (silver contacts tarnish)
├── Low current (self-cleaning not activated)
├── Worn contacts (arcing damage)
├── Loose terminal connection
├── Insufficient contact pressure

DIAGNOSIS:
├── Measure voltage across switch (should be <0.5V at rated current)
├── Measure contact resistance (should be <1Ω)
├── Compare to known good switch
├── Check terminal screws (tighten)

SOLUTIONS:
├── Operate switch 20-30 times (wipes clean)
├── For AC loads: rated current helps self-clean
├── For low current: use gold-plated contacts
├── Tighten terminals
├── Replace switch if resistance >2Ω
├── For silver contacts: allow higher current for cleaning

CONTACT CLEANING (Large switches only):

    Use contact cleaner spray (e.g., DeoxIT, CRC QD)
    Do NOT use sandpaper (damages plating)
    Do NOT use WD-40 (attracts dirt, conductive)
```

### Problem 4: Switch Sparking (Arcing)

```
SYMPTOMS:
├── Visible spark when switch opens/closes
├── Audible crackling or sizzling
├── Contacts blackened (visible damage)
├── Switch eventually fails (welded contacts)

CAUSES:
├── Inductive load (motor, solenoid, relay)
├── High inrush current (lamp, capacitor)
├── DC switching (arc extinguishes slowly)
├── Switch under-rated for load
├── Slow switching action (no snap mechanism)

DIAGNOSIS:
├── Identify load type (resistive vs inductive)
├── Compare switch rating to actual load
├── For DC: measure load voltage and current
├── Check for missing snubber (RC circuit)

SOLUTIONS:
├── Use switch with higher current rating
├── Add snubber circuit (RC across contacts)
├── For DC loads: add diode across inductive load
├── For AC: use zero-crossing SSR instead
├── Use relay to switch heavy loads (switch controls relay)
├── Use switch with snap action (micro switch)

SNUBBER CALCULATION (AC loads):

    C = I_load² × 10⁻⁵ (approx)
    R = V_load / (10 × I_load) (approx)
    
    Example: 120V AC, 2A motor
    C = 4 × 1e-5 = 40nF (use 47nF, class X2)
    R = 120 / (10 × 2) = 6Ω (use 10Ω, 1W)

    Place snubber ACROSS switch contacts (not across load)
```

## Switch Ratings Quick Reference

```
RATING GUIDELINES

    Load Type               Derating       Example (10A switch)
    ──────────────────────────────────────────────────────────────
    Resistive (heater)      100%           10A
    Incandescent lamp       20-30%         2-3A (high inrush)
    Fluorescent lamp        30-50%         3-5A (ballast)
    LED lamp                80-100%        8-10A (low inrush)
    Motor (inductive)       30-50%         3-5A
    Solenoid                30-40%         3-4A
    Capacitive load         30-50%         3-5A (power supply)
    DC load (general)       50-80%         5-8A (no zero crossing)
    DC motor                20-30%         2-3A
    Low level (<10mA)       100% (special) Requires gold contacts


MINIMUM CURRENT (Self-cleaning):

    Silver contacts: Need >10mA to stay clean
    Gold contacts: OK with <1mA (no oxidation)
    If current <10mA, use gold-plated contacts

    Otherwise: Contacts may oxidize and fail (intermittent)
```

## Summary

1. **Switch** makes or breaks an electrical circuit (ON/OFF)

2. **Pole:** Number of independent circuits (Single, Double, Triple, Four)

3. **Throw:** Number of positions per pole (Single, Double)

4. **Momentary** returns when released; **Maintained** stays in position

5. **SPST** is simplest ON/OFF (2 terminals)

6. **SPDT** switches between two outputs (3 terminals)

7. **DPDT** switches two circuits simultaneously (6 terminals)

8. **Toggle switch:** Maintained, high current, visible indication

9. **Rocker switch:** Modern standard for appliances, snap-in mounting

10. **Push button:** Momentary or alternate action; keyboards, reset

11. **Slide switch:** Small, PCB mount; configuration, power

12. **Rotary switch:** Multi-position selection (2-12 positions)

13. **DIP switch:** Multiple SPST in one package; configuration settings

14. **Limit switch:** Detects mechanical position; end-of-travel

15. **Micro switch:** Snap action; fast switching, long life

16. **Contact rating** must exceed load (derate for inductive, DC, lamps)

17. **Contact resistance** should be <0.5Ω (high resistance causes heating)

18. **Debouncing** needed for digital inputs (RC filter or software)

19. **Gold contacts** required for low current (<10mA) applications

20. **Sealed switches** (IP65-IP68) for outdoor, wet, dusty environments
