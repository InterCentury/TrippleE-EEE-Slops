# 01.04 - Voltage

## What is Voltage?

Voltage, also called electric potential difference, is the electrical "pressure" that pushes electric current through a circuit. It is the force that causes electrons to move from one point to another.

The unit of voltage is the **Volt** (symbol: V), named after Italian physicist **Alessandro Volta** (1745-1827), who invented the first electric battery (the Voltaic Pile) in 1800.

Voltage is always measured between two points. A single point cannot have voltage—only a difference in potential between two points.

## Understanding Voltage

### The "Push" Concept

Voltage is what pushes electrons through a conductor. Without voltage, no current flows.

```
VOLTAGE AS A "PUSH"

    NO VOLTAGE (0V):
    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Battery disconnected:                                     │
    │                                                             │
    │   ┌─────┐     ┌─────────────────────────┐                   │
    │   │  +  │     │                         │                   │
    │   │     │     │   Copper wire           │                   │
    │   │  -  │     │   e⁻  e⁻  e⁻  e⁻  e⁻    │                   │
    │   └─────┘     │   (random motion)       │                   │
    │               └─────────────────────────┘                   │
    │                                                             │
    │   Electrons move randomly.                                  │
    │   NO NET CURRENT.                                           │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘


    WITH VOLTAGE (12V):
    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Battery connected:                                        │
    │                                                             │
    │   ┌─────┐     ┌─────────────────────────┐                   │
    │   │  +  │────►│                         │                   │
    │   │     │     │   Copper wire           │                   │
    │   │  -  │◄────│   e⁻→e⁻→e⁻→e⁻→e⁻→e⁻     │                   │

    │                                                             │

```

## The Unit of Voltage: The Volt

### Definition

```
THE VOLT (V)

DEFINITION:
1 Volt = 1 Joule of energy per 1 Coulomb of charge

1 V = 1 J / 1 C

WHERE:
1 Joule = unit of energy
1 Coulomb = 6.24 × 10¹⁸ electrons


ALTERNATIVE DEFINITION (using Ohm's Law):
1 Volt = 1 Ampere × 1 Ohm

1 V = 1 A × 1 Ω


VISUALIZING 1 VOLT:

    A 1V battery pushes 1A through a 1Ω resistor
    
    ┌─────┐        ┌─────┐
    │ 1V  │────────┤ 1Ω  │────────┐
    │     │        └─────┘        │
    │     │                       │
    └─────┘───────────────────────┘
    
    Current = 1A
```

## Voltage in Circuits

### Voltage in Series Circuits

In a series circuit, voltage divides among components.

```
SERIES CIRCUIT - VOLTAGE DIVIDES

    ┌─────[R1]─────[R2]─────[R3]──────┐
    │      10Ω       20Ω       30Ω    │
    │                                 │
    └─────────────[12V]───────────────┘

    Total voltage = 12V
    Voltage across each resistor:
    V₁ = 2V (across 10Ω)
    V₂ = 4V (across 20Ω)
    V₃ = 6V (across 30Ω)
    
    V_total = V₁ + V₂ + V₃ = 12V ✓


VOLTAGE DIVIDER RULE:

    V_n = V_total × (R_n / R_total)
    
    Example (R₂ = 20Ω, R_total = 60Ω):
    V₂ = 12V × (20/60) = 12V × 0.333 = 4V


KEY PRINCIPLES:
├── Voltage divides in proportion to resistance
├── Higher resistance = higher voltage drop
├── Sum of all voltage drops = source voltage
└── Current is the same everywhere (Ohm's Law)
```

### Voltage in Parallel Circuits

In a parallel circuit, voltage is the same across all branches.

```
PARALLEL CIRCUIT - VOLTAGE IS CONSTANT

              ┌─────[R1]─────┐
              │     10Ω      │
              │              │
              ├─────[R2]─────┤
              │     20Ω      │
              │              │
              ├─────[R3]─────┤
              │     30Ω      │
              │              │
             (+)            (-)
              │              │
              └─────[12V]────┘

    Voltage across each resistor:
    V₁ = 12V
    V₂ = 12V
    V₃ = 12V
    
    Same voltage across all branches!


KEY PRINCIPLES:
├── Voltage is identical across all parallel branches
├── Current divides based on resistance
└── Adding more parallel branches does NOT change voltage
```

### Voltage Drop

When current flows through a resistance, voltage drops across that resistance.

```
VOLTAGE DROP EXPLANATION

    Without current (open circuit):
    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   12V ──┬────[100Ω]────┬────                                │
    │         │              │                                    │
    │         │              │   (open)                           │
    │         │              │                                    │
    │         └──────────────┘                                    │
    │                                                             │
    │   No current → No voltage drop                              │
    │   Voltage at both ends of resistor = 12V                    │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘


    With current (closed circuit):
    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   12V ──┬────[100Ω]────┬────[Load]────┐                     │
    │         │              │              │                     │
    │         │              │              │                     │
    │         └──────────────┴──────────────┘                     │
    │                                                             │
    │   Current flows → Voltage drops across resistor             │
    │   Voltage at resistor output = less than 12V                │
    │                                                             │
    │   Formula: V_drop = I × R                                   │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘


PRACTICAL EXAMPLE (Wire voltage drop):

    12V battery, 10m of thin wire (0.1Ω/m), 2A load
    
    Wire resistance (round trip) = 20m × 0.1Ω = 2Ω
    Voltage drop in wires = I × R = 2A × 2Ω = 4V
    Voltage at load = 12V - 4V = 8V (too low!)
    
    SOLUTION: Use thicker wire (lower resistance)
```

## Measuring Voltage

### Using a Voltmeter / Multimeter

```
HOW TO MEASURE VOLTAGE

STEP 1: Set multimeter to DC volts (⎓) or AC volts (~)

STEP 2: Connect leads:
    ┌─────────┐
    │   COM   │─── black lead ──► negative point
    │   VΩ    │─── red lead   ──► positive point
    └─────────┘

STEP 3: Read display


CORRECT CONNECTION (Parallel):
═══════════════════════════════════════════════════════

    Voltmeter connects ACROSS the component
    
    ┌─────[RESISTOR]─────┐
    │                    │
    │   ┌─────────────┐  │
    │   │    METER    │  │
    │   └─────────────┘  │
    │                    │
    └────────────────────┘
    
    Voltmeter measures voltage DIFFERENCE


INCORRECT CONNECTION (Series - WRONG!):
═══════════════════════════════════════════════════════

    NEVER connect voltmeter in series!
    
    ┌─────[RESISTOR]─────[METER]──────┐
    │                                 │
    │                                 │
    └─────────────────────────────────┘
    
    Voltmeter has very high resistance (~10MΩ)
    Connected in series → almost no current → wrong reading!


VOLTMETER RULES:
├── ALWAYS connect in parallel (across the component)
├── NEVER connect in series (will disrupt circuit)
├── Observe polarity for DC (+ to +, - to -)
├── AC polarity doesn't matter
├── Start with highest range if unsure
└── Don't exceed meter's maximum voltage rating
```

## Voltage Safety

### Voltage Levels and Danger

```
VOLTAGE DANGER LEVELS

SAFE VOLTAGE (Generally safe to touch dry skin):
─────────────────────────────────────────────────
    < 30V AC        SELV (Safety Extra Low Voltage)
    < 60V DC        SELV (Safety Extra Low Voltage)
    
    Examples: 5V, 12V, 24V, 48V


LOW VOLTAGE (Requires protection):
─────────────────────────────────────────────────
    30V - 100V AC   Can be lethal under wet conditions
    60V - 150V DC   Can be lethal under wet conditions
    
    Examples: 120V AC (US wall), 48V-72V DC (some EV systems)


HIGH VOLTAGE (Extremely dangerous):
─────────────────────────────────────────────────
    >100V AC        Always lethal potential
    >150V DC        Always lethal potential
    
    Examples: 230V AC (EU wall), 400V DC (EV battery)


VOLTAGE vs CURRENT (Which kills?)

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │  Common myth: "Current kills, not voltage"                  │
    │                                                             │
    │  TRUTH: Both matter!                                        │
    │                                                             │
    │  ├── Voltage determines if current can flow                 │
    │  ├── Skin resistance: ~100kΩ dry, ~1kΩ wet                  │
    │  ├── Dry skin: 50V cannot push lethal current               │
    │  ├── Wet skin: 50V CAN push lethal current!                 │
    │  └── Ohm's Law: I = V / R                                   │
    │                                                             │
    │  Example (wet skin, R=1kΩ):                                 │
    │  ├── 12V → I = 12/1000 = 0.012A (12mA, painful)           │
    │  ├── 50V → I = 50/1000 = 0.05A (50mA, lethal potential)   │
    │  └── 120V → I = 120/1000 = 0.12A (120mA, definitely lethal)│
    │                                                             │
    └─────────────────────────────────────────────────────────────┘


EFFECTS OF VOLTAGE ON HUMAN BODY (wet skin):

    Voltage (AC)    Current       Effect
    ──────────────────────────────────────────────────────────
    5V-12V          <10 mA        Usually safe
    24V-30V         10-30 mA      Painful shock
    50V-60V         30-60 mA      "Can't let go", breathing difficulty
    120V+           >100 mA       Ventricular fibrillation (often fatal)
    230V+           >200 mA       Severe burns, cardiac arrest
```

### Safe Voltage Practices


## Voltage Drop Calculations

### Wire Sizing for Voltage Drop

```
VOLTAGE DROP CALCULATION

FORMULA:
    V_drop = I × R_wire
    
    R_wire = ρ × (2L / A)
    
    WHERE:
    V_drop = voltage drop (volts)
    I = current (amperes)
    R_wire = total wire resistance (ohms)
    ρ = resistivity of copper (1.68×10⁻⁸ Ω·m)
    L = one-way length (meters)
    A = cross-sectional area (m²)


ALLOWABLE VOLTAGE DROP:

    Application                    Max Drop
    ──────────────────────────────────────────
    Lighting circuits              3%
    Power circuits                 5%
    Sensitive electronics          1%
    Critical equipment             0.5%


EXAMPLE: 12V system, 5A load, 10m distance

    Step 1: Round trip length = 20m
    Step 2: Target max drop = 3% of 12V = 0.36V
    Step 3: Max resistance = V_drop / I = 0.36V / 5A = 0.072Ω
    Step 4: Required area = ρ × L / R = (1.68e-8 × 20) / 0.072
           = 3.36e-7 / 0.072 = 4.67e-6 m² = 4.67 mm²
    
    Solution: Use 6 mm² wire (AWG 10)


VOLTAGE DROP TABLE (Copper wire, 12V system):

    AWG    Area     Ω/100m    Current   Max Length (3% drop)
    ──────────────────────────────────────────────────────
    18     0.823    2.55      1A        14m
    16     1.31     1.60      2A        11m
    14     2.08     1.01      5A        7m
    12     3.31     0.634     10A       5m
    10     5.26     0.399     15A       5m
    8      8.37     0.251     20A       6m
    6      13.3     0.158     30A       8m
    
    Note: Higher voltage = longer possible distance!
    24V system can run 2× longer than 12V for same loss
```

## Common Voltage Problems and Solutions

### Problem 1: Low Voltage (Brownout)

```
SYMPTOMS:
├── Lights dim
├── Motors run slow or overheat
├── Electronics reset or malfunction
├── Battery-powered devices won't charge

CAUSES:
├── Undersized wiring (too thin, too long)
├── Weak battery (high internal resistance)
├── Undersized power supply (overloaded)
├── Poor connections (corrosion, loose)
├── High grid demand (utility brownout)

DIAGNOSIS:
├── Measure voltage at source (battery/PSU)
├── Measure voltage at load (compare)
├── Calculate expected voltage drop
├── Check all connections
├── Monitor voltage under load vs no load

SOLUTIONS:
├── Upgrade to thicker wire
├── Reduce distance (move supply closer)
├── Replace battery
├── Use higher current power supply
├── Clean/tighten connections
├── Add voltage regulator at load
├── Increase system voltage (12V → 24V)
```

### Problem 2: Voltage Spikes (Overvoltage)

```
SYMPTOMS:
├── Lights flash bright then burn out
├── Electronics damaged or destroyed
├── Power supply trips or shuts down
├── Strange behavior after power event

CAUSES:
├── Lightning strike (nearby)
├── Grid switching transients
├── Inductive load turn-off (motor, relay)
├── Faulty voltage regulator
├── Loose neutral (dangerous!)

DIAGNOSIS:
├── Requires oscilloscope to capture (too fast for multimeter)
├── Check for burn marks on PCBs
├── Test components for damage
├── Monitor with surge recorder

SOLUTIONS:
├── Surge protectors (MOVs, TVS diodes)
├── Uninterruptible Power Supply (UPS)
├── Line conditioner
├── Snubber circuits on inductive loads
├── Check and tighten neutral connection
├── Whole-house surge protector
```

### Problem 3: Floating Voltage

```
SYMPTOMS:
├── Multimeter shows erratic voltage
├── Touching circuit changes reading
├── Device works intermittently

CAUSES:
├── No ground reference
├── High impedance circuit
├── Capacitive coupling from adjacent wires
├── Open ground connection

DIAGNOSIS:
├── Connect a 10kΩ resistor from point to ground
├── If voltage drops significantly → floating
├── Check ground connections
├── Measure with oscilloscope (may show 50/60Hz pickup)

SOLUTIONS:
├── Add pull-down resistor to ground
├── Properly ground the circuit
├── Use differential measurement
├── Shield sensitive signals
├── Single-point grounding
```

## Summary

1. **Voltage** is electrical pressure that pushes current through a circuit

2. **Unit:** Volt (V) = 1 Joule per Coulomb = 1 Ampere × 1 Ohm

3. **Voltage is always measured between two points** (potential difference)

4. **Sources of voltage:** Batteries (chemical), power supplies (AC→DC), generators (mechanical), solar cells (light)

5. **Series circuits:** Voltage divides (V_total = V₁ + V₂ + ...)

6. **Parallel circuits:** Voltage is same across all branches

7. **Voltage divider rule:** V_n = V_total × (R_n / R_total)

8. **Voltage drop** occurs when current flows through resistance (V_drop = I × R)

9. **Measure voltage** by connecting voltmeter in PARALLEL (across component)

10. **Safety:** <48V generally safe; >48V potentially lethal

11. **Wet skin** dramatically lowers resistance → lower voltage becomes dangerous

12. **Wire sizing** must account for voltage drop (3-5% max typical)
