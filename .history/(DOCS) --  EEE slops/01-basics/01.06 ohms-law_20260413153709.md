# 01.06 - Ohm's Law

## What is Ohm's Law?

Ohm's Law is the fundamental relationship between voltage, current, and resistance in an electrical circuit. It states that the current through a conductor between two points is directly proportional to the voltage across the two points, and inversely proportional to the resistance between them.

The law is named after German physicist **Georg Simon Ohm** (1789-1854), who published it in 1827. At the time, his work was criticized and called "a web of naked fancies," but today it is recognized as the most important law in electrical engineering.

## The Three Forms of Ohm's Law

Ohm's Law can be expressed in three equivalent forms, depending on which quantity you want to find.

```
OHM'S LAW TRIANGLE

                ┌─────────────┐
                │      V      │
                │   (Volts)   │
                │             │
                │   ┌───┐     │
                │   │ × │     │
                │   └───┘     │
                │             │
                │  I  │  R    │
                │(A)  │ (Ω)   │
                └─────────────┘

    Cover the quantity you want to find.
    The remaining arrangement shows the formula.


THREE FORMULAS:

    1. V = I × R     (Find voltage)
    2. I = V / R     (Find current)
    3. R = V / I     (Find resistance)


HOW TO USE THE TRIANGLE:

    To find VOLTAGE (V):
    ┌─────────────┐
    │      V      │    V = I × R
    │             │
    │   ┌───┐     │
    │   │ × │     │
    │   └───┘     │
    │  I  │  R    │
    └─────────────┘

    To find CURRENT (I):
    ┌─────────────┐
    │      V      │    I = V / R
    │             │
    │   ┌───┐     │
    │   │ / │     │
    │   └───┘     │
    │  I  │  R    │
    └─────────────┘

    To find RESISTANCE (R):
    ┌─────────────┐
    │      V      │    R = V / I
    │             │
    │   ┌───┐     │
    │   │ / │     │
    │   └───┘     │
    │  I  │  R    │
    └─────────────┘
```

## Understanding the Proportionality

### Direct Proportionality (Voltage and Current)

```
V ∝ I (Voltage is directly proportional to Current)

When resistance is CONSTANT:
├── Double the voltage → Double the current
├── Halve the voltage → Halve the current
└── Voltage and current rise and fall together


GRAPH (Constant Resistance = 10Ω):

    Current (A)
        │
    1.0 ┼─────────────────●
        │               ╱
    0.8 ┼─────────────●
        │           ╱
    0.6 ┼─────────●
        │       ╱
    0.4 ┼─────●
        │   ╱
    0.2 ┼─●
        │╱
    0.0 ┼────────────────────────► Voltage (V)
        0    2    4    6    8    10
    
    Slope = 1/R = 0.1 Siemens (conductance)
    Straight line through origin = DIRECT proportion
```

### Inverse Proportionality (Current and Resistance)

```
I ∝ 1/R (Current is inversely proportional to Resistance)

When voltage is CONSTANT:
├── Double the resistance → Half the current
├── Halve the resistance → Double the current
└── Current decreases as resistance increases


GRAPH (Constant Voltage = 10V):

    Current (A)
        │
    5.0 ┼●
        │ ╲
    4.0 ┼  ●
        │   ╲
    3.0 ┼    ●
        │     ╲
    2.0 ┼      ●
        │       ╲
    1.0 ┼        ●
        │         ╲
    0.0 ┼──────────●────────────► Resistance (Ω)
        0    5    10    15    20
    
    Hyperbolic curve = INVERSE proportion
```

## Practical Examples (No Code)

### Example 1: Finding Voltage

```
PROBLEM:
A circuit has 0.5 amperes of current flowing through a 100 ohm resistor.
What is the voltage across the resistor?

GIVEN:
I = 0.5 A
R = 100 Ω
V = ?

FORMULA:
V = I × R

CALCULATION:
V = 0.5 A × 100 Ω = 50 V

ANSWER:
The voltage across the resistor is 50 volts.


CIRCUIT DIAGRAM:
    ┌─────────────────────────────────────┐
    │                                     │
    │   ┌─────[100Ω]─────┐                │
    │   │                │                │

    │  (+)              (-)               │
    │   │                │                │
    │   └────────────────┘                │
    │                                     │
    │   V = 50V    I = 0.5A               │

    │                                     │
    └─────────────────────────────────────┘
```

### Example 2: Finding Current

```
PROBLEM:
A 12 volt battery is connected to a 240 ohm resistor.
How much current flows?

GIVEN:
V = 12 V
R = 240 Ω
I = ?

FORMULA:
I = V / R

CALCULATION:
I = 12 V / 240 Ω = 0.05 A = 50 mA

ANSWER:
The current is 0.05 amperes (50 milliamperes).


VERIFICATION:
50 mA is safe for most LEDs (with proper resistor)
A typical LED needs 20 mA, so this is within range
```

### Example 3: Finding Resistance

```
PROBLEM:
A 9 volt battery produces 0.03 amperes (30 mA) through a circuit.
What is the total resistance?

GIVEN:
V = 9 V
I = 0.03 A
R = ?

FORMULA:
R = V / I

CALCULATION:
R = 9 V / 0.03 A = 300 Ω

ANSWER:
The total circuit resistance is 300 ohms.


CHECK:
300Ω is a standard resistor value (E24 series)
Commonly used with 9V batteries for LEDs
```

## Ohm's Law in Series Circuits

In a series circuit, Ohm's Law applies to each component AND the entire circuit.

```
SERIES CIRCUIT EXAMPLE

    ┌─────[R1]─────[R2]─────[R3]───────┐
    │      10Ω       20Ω       30Ω     │
    │                                  │
    │  (+)                          (-)
    │                                  │
    └─────────────[12V]───────────────┘

STEP 1: Find total resistance
R_total = R₁ + R₂ + R₃ = 10 + 20 + 30 = 60Ω

STEP 2: Find total current
I_total = V_total / R_total = 12V / 60Ω = 0.2A

STEP 3: Current is same everywhere in series
I₁ = I₂ = I₃ = I_total = 0.2A

STEP 4: Find voltage across each resistor (using Ohm's Law)
V₁ = I × R₁ = 0.2A × 10Ω = 2V
V₂ = I × R₂ = 0.2A × 20Ω = 4V
V₃ = I × R₃ = 0.2A × 30Ω = 6V

STEP 5: Verify Kirchhoff's Voltage Law
V_total = V₁ + V₂ + V₃ = 2V + 4V + 6V = 12V ✓

RESULTS SUMMARY:
┌──────────────┬──────────┬──────────┬──────────┐
│  Component   │  R (Ω)   │  I (A)   │  V (V)   │
├──────────────┼──────────┼──────────┼──────────┤
│  R1          │   10     │   0.2    │    2     │
│  R2          │   20     │   0.2    │    4     │
│  R3          │   30     │   0.2    │    6     │
│  Total       │   60     │   0.2    │   12     │
└──────────────┴──────────┴──────────┴──────────┘
```

## Ohm's Law in Parallel Circuits

In a parallel circuit, Ohm's Law applies to each branch AND the entire circuit.

```
PARALLEL CIRCUIT EXAMPLE

                      ┌─────[R1]─────┐
                      │     10Ω      │
                      │              │
              ┌───────┼─────[R2]─────┼───────┐
              │       │     20Ω      │       │
              │       │              │       │
              │       └─────[R3]─────┘       │
              │             30Ω              │
              │                              │
            (+)                            (-)
              │                              │
              └──────────[12V]───────────────┘

STEP 1: Find equivalent resistance
1/R_eq = 1/R₁ + 1/R₂ + 1/R₃
1/R_eq = 1/10 + 1/20 + 1/30
1/R_eq = 0.1 + 0.05 + 0.03333 = 0.18333
R_eq = 1 / 0.18333 = 5.45Ω

STEP 2: Find total current
I_total = V_total / R_eq = 12V / 5.45Ω = 2.2A

STEP 3: Voltage is same across all branches
V₁ = V₂ = V₃ = V_total = 12V

STEP 4: Find current through each resistor (Ohm's Law)
I₁ = V / R₁ = 12V / 10Ω = 1.2A
I₂ = V / R₂ = 12V / 20Ω = 0.6A
I₃ = V / R₃ = 12V / 30Ω = 0.4A

STEP 5: Verify Kirchhoff's Current Law
I_total = I₁ + I₂ + I₃ = 1.2 + 0.6 + 0.4 = 2.2A ✓

RESULTS SUMMARY:
┌──────────────┬──────────┬──────────┬──────────┐
│  Component   │  R (Ω)   │  I (A)   │  V (V)   │
├──────────────┼──────────┼──────────┼──────────┤
│  R1          │   10     │   1.2    │   12     │
│  R2          │   20     │   0.6    │   12     │
│  R3          │   30     │   0.4    │   12     │
│  Total       │  5.45    │   2.2    │   12     │
└──────────────┴──────────┴──────────┴──────────┘
```

## Ohm's Law in Series-Parallel Circuits

```
SERIES-PARALLEL CIRCUIT EXAMPLE

                      ┌─────[R2]─────┐
                      │     20Ω      │
              ┌───────┼─────[R3]─────┼───────┐
              │       │     30Ω      │       │
              │       └──────────────┘       │
              │                              │
              │         [R1]                 │
              │         10Ω                  │
              │                              │
            (+)                            (-)
              │                              │
              └──────────[24V]───────────────┘

STEP 1: Combine parallel resistors R2 and R3
1/R_parallel = 1/20 + 1/30 = 0.05 + 0.03333 = 0.08333
R_parallel = 12Ω

STEP 2: Now R1 is in series with R_parallel
R_total = R1 + R_parallel = 10Ω + 12Ω = 22Ω

STEP 3: Find total current
I_total = V_total / R_total = 24V / 22Ω = 1.091A

STEP 4: Current through R1 equals total current
I₁ = I_total = 1.091A

STEP 5: Find voltage across R1
V₁ = I₁ × R₁ = 1.091A × 10Ω = 10.91V

STEP 6: Voltage across parallel branch
V_parallel = V_total - V₁ = 24V - 10.91V = 13.09V

STEP 7: Current through R2 and R3 (same voltage)
I₂ = V_parallel / R₂ = 13.09V / 20Ω = 0.6545A
I₃ = V_parallel / R₃ = 13.09V / 30Ω = 0.4364A

STEP 8: Verify
I₁ = I₂ + I₃ = 0.6545 + 0.4364 = 1.0909A ✓

RESULTS SUMMARY:
┌──────────────┬──────────┬──────────┬──────────┐
│  Component   │  R (Ω)   │  I (A)   │  V (V)   │
├──────────────┼──────────┼──────────┼──────────┤
│  R1          │   10     │  1.091   │  10.91   │
│  R2          │   20     │  0.655   │  13.09   │
│  R3          │   30     │  0.436   │  13.09   │
│  Total       │   22     │  1.091   │   24     │
└──────────────┴──────────┴──────────┴──────────┘
```

## Voltage Divider Rule (Derived from Ohm's Law)

The voltage divider rule allows you to find voltage across a resistor in a series circuit without calculating current.

```
VOLTAGE DIVIDER FORMULA

For two resistors in series:

              R1
    V1 = V × ─────
             R1+R2

              R2
    V2 = V × ─────
             R1+R2

For multiple resistors:

              Rn
    Vn = V × ───────
             R_total


EXAMPLE:

    V_in = 12V
    
    ┌─────[R1]─────┐
    │     10Ω      │
    │              │
    │            V_out
    │              │
    └─────[R2]─────┘
          20Ω

    V_out = V_in × R2 / (R1 + R2)
    V_out = 12V × 20Ω / (10Ω + 20Ω)
    V_out = 12V × 20 / 30
    V_out = 12V × 0.6667 = 8V

PRACTICAL USE:
├── Volume controls (potentiometer as voltage divider)
├── Reference voltages for comparators
├── Sensor interfaces (photoresistor + fixed resistor)
├── Level shifting (5V to 3.3V for logic)
└── Biasing transistors
```

## Current Divider Rule (Derived from Ohm's Law)

The current divider rule allows you to find current through a parallel resistor without calculating voltage.

```
CURRENT DIVIDER FORMULA

For two resistors in parallel:

              R2
    I1 = I × ─────
             R1+R2

              R1
    I2 = I × ─────
             R1+R2

For multiple resistors:

              R_eq
    In = I × ─────
              Rn


EXAMPLE:

    I_total = 2A
    
              ┌─────[R1]─────┐
              │     10Ω      │
    I_total ──┤              ├── I1 + I2
              └─────[R2]─────┘
                    20Ω

    I1 = I_total × R2 / (R1 + R2)
    I1 = 2A × 20Ω / (10Ω + 20Ω)
    I1 = 2A × 20 / 30 = 2A × 0.6667 = 1.333A

    I2 = I_total × R1 / (R1 + R2)
    I2 = 2A × 10Ω / (10Ω + 20Ω)
    I2 = 2A × 10 / 30 = 2A × 0.3333 = 0.667A

    Check: 1.333A + 0.667A = 2A ✓

KEY INSIGHT:
├── Smaller resistor gets MORE current
├── Larger resistor gets LESS current
└── Current divides inversely to resistance ratio
```

## Ohm's Law for Power (Combined Formulas)

Ohm's Law combines with the power formula (P = V × I) to create useful variations.

```
POWER FORMULAS (Combined with Ohm's Law)

Starting with P = V × I:

Substitute V = I × R:
    P = (I × R) × I = I² × R

Substitute I = V / R:
    P = V × (V / R) = V² / R


THREE POWER FORMULAS:

    1. P = V × I     (Use when you know V and I)
    2. P = I² × R    (Use when you know I and R)
    3. P = V² / R    (Use when you know V and R)


OHM'S LAW + POWER WHEEL:

                    ┌─────────────┐
                    │      P      │
                    │   (Watts)   │
                    └─────────────┘
                          │
            ┌─────────────┼─────────────┐
            │             │             │
       ┌────┴────┐   ┌────┴────┐   ┌────┴────┐
       │    V    │   │    I    │   │    R    │
       │ (Volts) │   │ (Amps)  │   │ (Ohms)  │
       └────┬────┘   └────┬────┘   └────┬────┘
            │             │             │
            └─────────────┼─────────────┘
                          │
                    ┌─────┴─────┐
                    │  P = V×I  │
                    │  P = I²R  │
                    │  P = V²/R │
                    │  V = √(P×R)│
                    │  I = √(P/R)│
                    │  R = V²/P  │
                    └───────────┘


EXAMPLES:

1. Find power: V=12V, I=2A
   P = 12 × 2 = 24W

2. Find power: I=0.5A, R=100Ω
   P = (0.5)² × 100 = 0.25 × 100 = 25W

3. Find power: V=9V, R=300Ω
   P = (9)² / 300 = 81 / 300 = 0.27W

4. Find voltage: P=100W, R=4Ω
   V = √(100 × 4) = √400 = 20V

5. Find current: P=50W, R=8Ω
   I = √(50 / 8) = √6.25 = 2.5A

6. Find resistance: V=120V, P=60W (light bulb)
   R = (120)² / 60 = 14400 / 60 = 240Ω
```

## Limitations of Ohm's Law

Ohm's Law is not universal. It applies only to **ohmic materials**.

```
OHMIC vs NON-OHMIC MATERIALS

OHMIC (Follows Ohm's Law):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
    Current (A)
        │
    5.0 ┼─────────────────●
        │               ╱
    4.0 ┼─────────────●
        │           ╱
    3.0 ┼─────────●
        │       ╱
    2.0 ┼─────●
        │   ╱
    1.0 ┼─●
        │╱
    0.0 ┼────────────────────────► Voltage (V)
        0    2    4    6    8    10
    
    Straight line through origin
    Examples: Resistors, copper wire, nichrome


NON-OHMIC (Do NOT follow Ohm's Law):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    Current (A)              Current (A)
        │                        │
    5.0 ┼●                       │    ●
        │ ╲                      │   ╱
    4.0 ┼  ●                     │  ╱
        │   ╲                    │ ╱
    3.0 ┼    ●                   │●
        │     ╲                  │
    2.0 ┼      ●                 │
        │       ╲                │
    1.0 ┼        ●               │
        │         ╲              │
    0.0 ┼──────────●─────────►   └─────────────► Voltage
        0    2    4    6    8        0    2    4
    
    Diode (exponential)        Light bulb (heating effect)


NON-OHMIC MATERIALS:
├── Diodes and LEDs (exponential I-V curve)
├── Transistors (active region)
├── Light bulbs (filament resistance increases with heat)
├── Thermistors (resistance changes with temperature)
├── Varistors (voltage-dependent resistance)
├── Semiconductors in general
├── Electrolytes (batteries, body fluids)
└── Gases (neon bulbs, spark gaps)


WHY OHM'S LAW FAILS FOR THESE:
├── Resistance is not constant
├── R changes with V, I, T, or time
├── The V-I graph is not a straight line
└── The material has memory or threshold effects
```

## Common Applications of Ohm's Law

### Application 1: LED Current Limiting

```
LED RESISTOR CALCULATION

PROBLEM:
An LED requires 20 mA and drops 2V when on.
You have a 9V battery. What resistor do you need?

GIVEN:
V_supply = 9V
V_LED = 2V
I_LED = 0.020 A (20 mA)

CALCULATION:
Voltage across resistor = V_supply - V_LED = 9V - 2V = 7V
R = V_R / I = 7V / 0.020A = 350Ω

ANSWER:
Use 330Ω or 360Ω (standard values)
Power rating: P = I² × R = (0.02)² × 350 = 0.14W → use 1/4W


CIRCUIT:
    ┌─────[9V]─────┐
    │              │
    │             (+)
    │              │
    │            [LED]
    │              │
    │             (-)
    │              │
    │         [350Ω]
    │              │
    └──────────────┘
```

### Application 2: Finding Wire Size for Load

```
WIRE SIZING USING OHM'S LAW

PROBLEM:
A 12V system needs to deliver 10A to a load 10 meters away.
Allowable voltage drop is 0.5V (4.2%). What wire gauge?

GIVEN:
V_drop_allowed = 0.5V
I = 10A
Length = 10m (20m round trip: out and back)

CALCULATION:
Maximum resistance for round trip:
R_max = V_drop / I = 0.5V / 10A = 0.05Ω

Resistance per meter (round trip):
R_per_m = 0.05Ω / 20m = 0.0025 Ω/m = 2.5 mΩ/m

Per meter (one way): 1.25 mΩ/m

From wire table:
AWG 14: 8.28 mΩ/m (too high)
AWG 12: 5.21 mΩ/m (too high)
AWG 10: 3.28 mΩ/m (too high)
AWG 8:  2.06 mΩ/m (too high)
AWG 6:  1.30 mΩ/m (ACCEPTABLE)

ANSWER: Use AWG 6 wire (or thicker)


PRACTICAL TIP:
For long runs, increase voltage to reduce current.
Power = V × I, same power at higher V = lower I = thinner wire.
This is why power lines use VERY high voltage (hundreds of kV).
```

### Application 3: Finding a Short Circuit

```
USING OHM'S LAW TO FIND SHORTS

PROBLEM:
A power supply reads 0V output when connected to a board.
It should output 5V. You suspect a short circuit.

MEASUREMENT:
With power OFF, measure resistance between +5V and GND.

READING: 0.2Ω

CALCULATION (theoretical short circuit current):
I_short = V / R = 5V / 0.2Ω = 25A

INTERPRETATION:
├── 25A is massive for a 5V supply (typical supply is 1-2A)
├── This confirms a near-short circuit
├── The supply shuts down to protect itself
└── Find the 0.2Ω path (blown capacitor, solder bridge)


TROUBLESHOOTING:
├── Measure resistance between power and ground
├── Low resistance (< 10Ω) suggests short
├── Remove components one by one
├── When resistance jumps up, you found the culprit
└── Visual inspection for solder bridges or burned parts
```

### Application 4: Battery Internal Resistance

```
MEASURING BATTERY INTERNAL RESISTANCE

PROBLEM:
A 9V battery measures 9.3V with no load.
With a 100Ω load, voltage drops to 8.7V.
What is the internal resistance?

GIVEN:
V_no_load = 9.3V
V_load = 8.7V
R_load = 100Ω

CALCULATION:
Voltage drop inside battery = 9.3V - 8.7V = 0.6V
Current through load = V_load / R_load = 8.7V / 100Ω = 0.087A
Internal resistance = V_drop / I = 0.6V / 0.087A = 6.9Ω

CIRCUIT MODEL:
    ┌─────[Battery]─────┐
    │                   │
    │   Ideal 9.3V     │
    │      │           │
    │      └──[6.9Ω]───┤ (internal resistance)
    │                   │
    │      ┌──[100Ω]───┤ (load)
    │      │           │
    │      └───────────┘
    │                   │
    └───────────────────┘

INTERPRETATION:
├── New 9V battery: R_internal ≈ 1-2Ω
├── Used 9V battery: R_internal ≈ 5-10Ω
├── Dead 9V battery: R_internal > 20Ω
└── Car battery (new): R_internal ≈ 0.01Ω (10 mΩ)
```

| Resistance | R | Ohm (Ω) | R = V / I | 1Ω = 1V/A |
| Power | P | Watt (W) | P = V × I | 1W = 1J/s |

## Common Ohm's Law Problems and Solutions

### Problem 1: Resistor Overheating

```
SYMPTOM: Resistor is too hot to touch or smoking

DIAGNOSIS:
Calculate actual power: P = V²/R or P = I²×R
Compare to resistor power rating

EXAMPLE:
9V across 100Ω resistor
P = 81 / 100 = 0.81W
If using 1/4W (0.25W) resistor → OVERHEATING!

SOLUTIONS:
├── Use higher wattage resistor (1W or 2W)
├── Increase resistance (if circuit allows)
├── Reduce voltage
└── Use multiple resistors in series/parallel to share power

CALCULATION FOR MULTIPLE RESISTORS:
Two 200Ω resistors in parallel:
Each sees 9V, P_each = 81/200 = 0.405W
Still too high for 1/4W, but OK for 1/2W
```

### Problem 2: Voltage Too Low at Load

```
SYMPTOM: Device doesn't work or works poorly (dim light, slow motor)

DIAGNOSIS:
Measure voltage at load. Compare to expected.
Apply Ohm's Law to find voltage drop in wires.

EXAMPLE:
12V supply, 10m of AWG 18 wire (0.021Ω/m)
Load draws 5A

Wire resistance (round trip) = 20m × 0.021Ω/m = 0.42Ω
Voltage drop in wires = 5A × 0.42Ω = 2.1V
Voltage at load = 12V - 2.1V = 9.9V (too low!)

SOLUTIONS:
├── Use thicker wire (AWG 14 or 12)
├── Move supply closer to load
├── Increase supply voltage to compensate
├── Reduce load current (if possible)
└── Use higher voltage and local regulator

With AWG 12 wire (0.0052Ω/m):
Wire R = 20m × 0.0052Ω/m = 0.104Ω
V_drop = 5A × 0.104Ω = 0.52V
V_load = 12V - 0.52V = 11.48V (acceptable)
```

### Problem 3: Intermittent Operation

```
SYMPTOM: Device works sometimes, fails others

DIAGNOSIS:
Likely a loose connection or bad solder joint
Resistance changes when moved or heated

USE OHM'S LAW:
Measure resistance while wiggling wires/components
Look for resistance that jumps from near-zero to infinite

COMMON CULPRITS:
├── Cold solder joints (resistance varies)
├── Cracked PCB traces
├── Loose connector pins
├── Dirty switch contacts
├── Broken wire inside insulation

SOLUTION:
├── Re-solder suspicious joints
├── Bypass with jumper wire
├── Replace connectors
├── Clean contacts with contact cleaner
├── Replace damaged wires
```

## Summary

1. **Ohm's Law:** V = I × R (Voltage = Current × Resistance)

2. **Three forms:**
   - V = I × R (find voltage)
   - I = V / R (find current)
   - R = V / I (find resistance)

3. **Direct proportion:** V ∝ I (when R constant)

4. **Inverse proportion:** I ∝ 1/R (when V constant)

5. **Series circuits:** Same current, voltage divides

6. **Parallel circuits:** Same voltage, current divides

7. **Voltage divider:** Vn = V_total × (Rn / R_total)

8. **Current divider:** In = I_total × (R_eq / Rn)

9. **Power formulas:** P = V×I = I²R = V²/R

10. **Limitations:** Only for ohmic materials (not diodes, transistors, light bulbs)

11. **Practical uses:** LED resistors, wire sizing, finding shorts, battery testing

12. **Troubleshooting:** Use Ohm's Law to calculate expected values, then measure to find discrepancies
