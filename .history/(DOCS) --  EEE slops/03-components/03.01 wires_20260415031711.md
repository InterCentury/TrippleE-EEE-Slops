# 03.01 - Wires and Cables

## What are Wires and Cables?

Wires and cables are the fundamental building blocks of all electrical and electronic systems. They provide the conductive paths that allow electricity to flow from power sources to loads, between components, and throughout circuits.

A **wire** is a single conductor, typically made of copper or aluminum, that carries electrical current. A **cable** is a collection of two or more wires bundled together, often with additional insulation, shielding, or protective jackets.

The difference between wires and cables is important for proper selection. Understanding wire properties—gauge, insulation, current capacity, voltage rating, and environmental suitability—is essential for designing safe and reliable electrical systems.

### Solid vs Stranded Wire

```
SOLID WIRE

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
    │                                                             │
    │   Single solid conductor                                    │
    │                                                             │
    │   Advantages:                                               │
    │   ├── Lower cost                                            │
    │   ├── Easier to terminate (screw terminals, breadboards)    │
    │   ├── Less susceptible to corrosion                         │
    │   └── Lower resistance for same gauge (no air gaps)         │
    │                                                             │
    │   Disadvantages:                                            │
    │   ├── Stiff, hard to bend                                   │
    │   ├── Work-hardens and breaks if flexed repeatedly          │
    │   ├── Poor vibration resistance                             │
    │   └── Skin effect at high frequencies (minor)               │
    │                                                             │
    │   Best for:                                                 │
    │   ├── Permanent installations (building wiring)             │
    │   ├── Breadboards                                           │
    │   ├── Prototype boards                                      │
    │   └── Applications with no movement                         │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘


STRANDED WIRE

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░   │
    │                                                             │
    │   Many thin strands twisted together                        │
    │                                                             │
    │   Advantages:                                               │
    │   ├── Flexible, easy to bend                                │
    │   ├── Resists breakage from repeated flexing                │
    │   ├── Better vibration resistance                           │
    │   └── Better high-frequency performance (skin effect)       │
    │                                                             │
    │   Disadvantages:                                            │
    │   ├── Higher cost                                           │
    │   ├── Requires special terminals (ferrules for screw terms) │
    │   ├── Strands can break individually                        │
    │   └── Slightly higher resistance (air gaps)                 │
    │                                                             │
    │   Best for:                                                 │
    │   ├── Portable cords (extension cords, appliance cords)     │
    │   ├── Automotive wiring                                     │
    │   ├── Robotics and moving parts                             │
    │   ├── Test leads                                            │
    │   └── Any application with movement or vibration            │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘


STRANDING PATTERNS:

    Type              Description                    Flexibility
    ──────────────────────────────────────────────────────────────
    Solid             Single conductor               Rigid
    7-strand          1 center + 6 around            Moderate
    19-strand         1 + 6 + 12                     Flexible
    37-strand         1 + 6 + 12 + 18                Very flexible
    65-strand+        Many fine strands              Highly flexible
    Ultra-flexible    Hundreds of very fine strands  Extremely flexible
```

```

### Aluminum (Lightweight Alternative)

```
ALUMINUM CONDUCTOR PROPERTIES

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Why use aluminum:                                         │
    │                                                             │
    │   ├── Lightweight (1/3 density of copper)                   │
    │   ├── Lower cost (about 1/2 price of copper)                │
    │   ├── Good conductivity (61% of copper by volume)           │
    │   └── Better conductivity by weight (2× copper!)            │
    │                                                             │
    │   Resistivity: 2.65 × 10⁻⁸ Ω·m at 20°C                      │
    │   Temperature coefficient: +0.00403 /°C                     │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘


ALUMINUM WIRE CHALLENGES:

    Problem                     Solution
    ──────────────────────────────────────────────────────────────
    Oxide layer (insulating)    Use antioxidant paste
    Thermal expansion           Special lugs (Al/Cu rated)
    Creep (loosens connections) Periodic retorquing
    Galvanic corrosion (with Cu) Use bi-metal connectors
    Brittle (work hardens)      Avoid repeated bending
    Not solderable              Use crimp or compression


ALUMINUM APPLICATIONS:

    ├── Overhead power lines (ACSR - Aluminum Conductor Steel Reinforced)
    ├── Service entrance cables (larger gauges)
    ├── Feeder cables in buildings
    ├── Transformer windings (some)
    └── NOT recommended for branch circuits (problematic history)
```

### Derating Factors

```

    Ambient Temp     Derating Factor
    ──────────────────────────────────
    30°C (86°F)      1.00 (100%)
    35°C (95°F)      0.94 (94%)

    55°C (131°F)     0.67 (67%)
    60°C (140°F)     0.58 (58%)

    Derated = 30A × 0.75 = 22.5A maximum

    Number of Conductors    Derating Factor
    ────────────────────────────────────────
    1-3                     1.00 (100%)
    4-6                     0.80 (80%)
    7-9                     0.70 (70%)
    10-20                   0.50 (50%)
    21-30                   0.45 (45%)
    
    Example: 30A wire with 6 conductors in conduit
    Derated = 30A × 0.80 = 24A maximum


COMBINED DERATING:

    Both factors apply!
    
    Example: 30A wire, 50°C ambient, 6 conductors
    Temperature derate: 0.75
    Conductor count derate: 0.80
    Combined = 0.75 × 0.80 = 0.60
    
    Final ampacity = 30A × 0.60 = 18A

## Voltage Drop in Wires

### Calculation Method

```
VOLTAGE DROP FORMULA

    V_drop = I × R_wire × 2 (for round trip)
    
    R_wire = ρ × L / A
    
    WHERE:
    V_drop = voltage drop (volts)
    I = current (amperes)
    R_wire = resistance of one conductor (ohms)
    ρ = resistivity (1.68e-8 for copper)
    L = one-way length (meters)
    A = cross-sectional area (m²)


PERCENTAGE VOLTAGE DROP:

    % V_drop = (V_drop / V_source) × 100%


ALLOWABLE VOLTAGE DROP:

    Application                    Max % Drop
    ──────────────────────────────────────────
    Lighting circuits              3%
    Power circuits                 5%
    Sensitive electronics          1%
    Critical equipment             0.5%
    Motor starting                 10% (temporary)


SIMPLE FORMULA (Copper, 12V DC):

    Max length (m) = (A_mm² × V_source × %_drop) / (I × 0.04)
    
    Example: 2.5mm² wire, 12V, 5% drop, 5A load
    Max length = (2.5 × 12 × 5) / (5 × 0.04)
              = (150) / (0.2) = 750m (theoretical)
    But other factors limit (ampacity, cost, practicality)
```

## Wire Selection Guide

### How to Choose the Right Wire

```
SELECTION PROCESS

STEP 1: Determine current (I)
    Calculate or measure load current
    Add 25% safety margin
    Example: 8A load → design for 10A

STEP 2: Determine voltage (V)
    System voltage (12V, 24V, 120V, etc.)
    Affects voltage drop calculation

STEP 3: Determine length (L)
    Measure one-way distance
    Remember round trip for voltage drop

STEP 4: Determine acceptable voltage drop
    Critical: 1%
    Normal: 3-5%
    Non-critical: 10%

STEP 5: Calculate required gauge
    Use ampacity tables (thermal limits)
    Use voltage drop formula (distance limits)
    Use larger of the two results

STEP 6: Select insulation type
    Temperature requirement
    Environment (indoor/outdoor, wet/dry)
    Chemical exposure
    Flexibility needed

STEP 7: Check codes and standards
    NEC, CEC, IEC as applicable
    Local electrical codes


EXAMPLE SELECTION:

    Load: 10A, 24V, distance 20m, critical (1% drop)
    
    Ampacity: 10A × 1.25 = 12.5A minimum
    Table says: AWG 16 (10A) or AWG 14 (15A) → need AWG 14
    
    Voltage drop: 1% of 24V = 0.24V max
    Round trip length = 40m
    Max resistance = 0.24V / 10A = 0.024Ω total
    R per 100m = 0.024Ω / 40m × 100 = 0.06Ω/100m
    
    Table: AWG 6 = 0.158Ω/100m (too high)
    AWG 4 = 0.815Ω/100m? Wait, that's higher! (No, 0.815 is higher than 0.158)
    Let me correct: Smaller number = lower resistance
    AWG 4: 0.815Ω/100m? That can't be right.
    
    Correct values:
    AWG 14: 2.52Ω/100m
    AWG 12: 1.59Ω/100m
    AWG 10: 1.00Ω/100m
    AWG 8:  0.628Ω/100m
    AWG 6:  0.395Ω/100m
    AWG 4:  0.248Ω/100m
    AWG 2:  0.156Ω/100m
    
    For 0.06Ω/100m needed: AWG 2 is closest (0.156 is still too high)
    Wait, 0.156 > 0.06, so still not enough.
    
    Let me recalculate properly:
    
    Required R_total = 0.024Ω for 40m
    Required R per meter = 0.024Ω / 40m = 0.0006 Ω/m
    Required R per 100m = 0.0006 × 100 = 0.06 Ω/100m
    
    None of the above! Smallest is AWG 2 at 0.156 Ω/100m.
    0.156 > 0.06, so even AWG 2 gives:
    R_total = 0.156 Ω/100m × 40m = 0.0624Ω
    V_drop = 10A × 0.0624Ω = 0.624V (2.6% drop)
    
    For 1% drop (0.24V), this is impossible with copper at 20m!
    Need: higher voltage (48V) or shorter distance, or accept higher drop.
    
    LESSON: Voltage drop drives wire sizing at low voltages!
```

## Common Wire Problems

### Problem 1: Overheating

```
SYMPTOMS:
├── Wire insulation melting or discolored
├── Burning smell
├── Wire too hot to touch
├── Voltage drop excessive

CAUSES:
├── Current exceeds wire ampacity
├── Poor connection (high resistance)
├── Undersized wire for load
├── Bundled wires (reduced cooling)
├── High ambient temperature
├── Insulation damage (reduced rating)

DIAGNOSIS:
├── Measure current (compare to ampacity)
├── Feel for hot spots (indicate bad connections)
├── Check insulation for damage
├── Measure voltage drop (indicates high resistance)
├── Thermal imaging (professional)

SOLUTIONS:
├── Reduce load (add circuits, larger wire)
├── Replace with larger gauge wire
├── Improve cooling (separate bundled wires)
├── Repair or replace damaged insulation
├── Tighten loose connections
├── Clean corroded terminals
```

### Problem 2: Broken Wire (Open Circuit)

```
SYMPTOMS:
├── No continuity
├── Device doesn't work
├── Intermittent operation

CAUSES:
├── Repeated flexing (stranded wire fatigue)
├── Overheating (brittle copper)
├── Corrosion (green/black copper)
├── Mechanical damage (cut, crushed)
├── Poor manufacturing

DIAGNOSIS:
├── Visual inspection (look for break)
├── Continuity test with multimeter (beep mode)
├── Wiggle wire while testing (intermittent)
├── Measure resistance (should be near zero)

SOLUTIONS:
├── Cut out damaged section
├── Splice with proper connector
├── Replace entire wire (if badly damaged)
├── Use stranded wire where flexing occurs
├── Add strain relief
├── Use longer service loops (allows re-stripping)
```

### Problem 3: Intermittent Connection

```
SYMPTOMS:
├── Device works, then stops, then works
├── Tapping or moving wire changes operation
├── Voltage fluctuates

CAUSES:
├── Loose screw terminal
├── Crimp connector not fully crimped
├── Wire strands broken inside insulation
├── Corroded connection
├── Cold solder joint
├── Poor contact in connector

DIAGNOSIS:
├── Wiggle connections while measuring continuity
├── Check each termination point
├── Pull test on crimps (should not come apart)
├── Visual inspection (green corrosion, dark wire)
├── Measure voltage drop across connection

SOLUTIONS:
├── Tighten screw terminals
├── Re-crimp connectors (use proper tool)
├── Re-strip and re-terminate wire
├── Clean corrosion with contact cleaner
├── Re-solder cold joints
├── Replace damaged connectors
├── Use dielectric grease on connections
```

## Summary

1. **Wires** are single conductors; **cables** are multiple wires bundled together

2. **Solid wire** is stiff, cheap, good for permanent installation

3. **Stranded wire** is flexible, resists breakage, good for moving applications

4. **Copper** is the standard conductor (101% IACS conductivity)

5. **Aluminum** is lighter and cheaper but requires special handling (61% conductivity)

6. **AWG** (American Wire Gauge): larger number = thinner wire

7. **Metric wire** sized by cross-sectional area (mm²)

8. **Insulation types** determine temperature rating and environment suitability

9. **Wire colors** have standard meanings (green = ground, black/red = hot, white = neutral)

10. **Ampacity** depends on wire size, insulation, temperature, and installation method

11. **Voltage drop** = I × R × 2 (round trip); critical for low voltage systems

12. **Higher voltage** allows much longer wire runs (double voltage = quarter the loss)

13. **Always derate** for high ambient temperature and multiple conductors

14. **Common failures:** overheating, broken wire, intermittent connection

15. **Safety:** Never exceed ampacity; use proper colors; follow local codes
