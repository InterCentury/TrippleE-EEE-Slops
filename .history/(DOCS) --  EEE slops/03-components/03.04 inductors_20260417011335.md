# 03.04 - Inductors

## What is an Inductor?

An inductor is a passive two-terminal electronic component that stores electrical energy in a magnetic field. It consists of a wire wound into a coil. When current flows through the coil, a magnetic field is created around it. Changes in current cause changes in the magnetic field, which induces a voltage that opposes the change in current.

The inductor was discovered in the 1830s by Joseph Henry in the United States and independently by Michael Faraday in England. Faraday's experiments with coils and magnets led to the discovery of electromagnetic induction, the principle underlying all inductors, transformers, generators, and motors.

### Lenz's Law

Lenz's Law describes the direction of induced voltage.

```
LENZ'S LAW

    "The induced voltage opposes the change in current."

    V = -L × (dI / dt)

    WHERE:
    V = induced voltage (volts)
    L = inductance (henries)
    dI/dt = rate of change of current (amperes/second)
    Negative sign = opposition


EXAMPLES:

    Current change        Induced voltage opposes
    ──────────────────────────────────────────────────────────
    Increasing current    Creates voltage that tries to DECREASE current
    Decreasing current    Creates voltage that tries to INCREASE current
    Steady current        No induced voltage (inductor acts as wire)


PRACTICAL EFFECTS:

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   When switch closes:                                       │
    │                                                             │
    │   5V ──┬────[Switch]────[Inductor]────┬───                  │
    │        │                              │                     │
    │        │                              │                     │
    │        │    ┌──────────────────┐      │                     │
    │        │    │                  │      │                     │
    │        └────┤    Resistor      ├──────┘                     │
    │             │                  │                            │
    │             └──────────────────┘                            │
    │                                                             │
    │   Current rises SLOWLY (not instantly)                      │
    │   Inductor "resists" the change                             │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   When switch opens:                                        │
    │                                                             │
    │   Current wants to stop instantly                           │
    │   Inductor generates HIGH VOLTAGE to try to keep current    │
    │   Can cause arcing across switch!                           │
    │                                                             │
    │   Solution: Add "flyback diode" to protect circuit          │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘
```

## The Unit of Inductance: The Henry

### Definition

```
THE HENRY (H)

DEFINITION:
1 Henry = 1 Volt induced when current changes at 1 Ampere per second

1 H = 1 V / (1 A/s)


VISUALIZING 1 HENRY:

    An inductor with 1 Henry of inductance:
    ├── Current changing at 1A per second produces 1V across it
    ├── Very large for most electronics
    └── Most inductors are much smaller


COMMON INDUCTANCE PREFIXES:

    Prefix      Symbol      Value           Example
    ──────────────────────────────────────────────────
    Henry       H           1 H             Power supply filter
    Millihenry  mH          0.001 H         1-100 mH (common)
    Microhenry  µH          0.000001 H      1-1000 µH (very common)
    Nanohenry   nH          0.000000001 H   10-100 nH (RF, parasitic)


TYPICAL INDUCTANCE VALUES:

    Value           Use
    ──────────────────────────────────────────────────
    1-100 nH        Parasitic inductance (PCB traces)
    100 nH - 1 µH   RF chokes, high-frequency filters
    1-100 µH        Switching power supplies, filters
    100 µH - 1 mH   Audio filters, power supplies
    1-100 mH        Low-frequency filters, ballasts
    1-10 H          Power supply smoothing (large iron core)
```

## Inductor Types

### Air Core Inductors

No magnetic core; lowest inductance, highest frequency capability.

```
AIR CORE INDUCTOR

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   ┌─────────────────────────────────────────────────────┐   │
    │   │  ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐          │   │
    │   │  │   │ │   │ │   │ │   │ │   │ │   │ │   │          │   │
    │   │  └───┘ └───┘ └───┘ └───┘ └───┘ └───┘ └───┘          │   │
    │   │                                                     │   │
    │   │   Air core (no magnetic material)                   │   │
    │   │                                                     │   │
    │   └─────────────────────────────────────────────────────┘   │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘

    Characteristics:
    ├── No core losses (efficiency very high)
    ├── No saturation (inductance constant with current)
    ├── Very low inductance (nH to µH range)
    ├── Best for high frequency (RF, VHF, UHF)
    ├── Physically large for higher inductance
    └── Use: RF circuits, antennas, high-frequency filters


CALCULATING AIR CORE INDUCTANCE:

    L = (N² × µ₀ × A) / l

    WHERE:
    L = inductance (henries)
    N = number of turns
    µ₀ = permeability of free space (4π × 10⁻⁷ H/m)
    A = cross-sectional area of coil (m²)
    l = length of coil (m)

    For practical coils: L (µH) ≈ (N² × r²) / (9r + 10l)
    r = radius in inches, l = length in inches (approximate)
```

```
    │     70 ┼      ╲                                             │

```

### Temperature Rating

Maximum operating temperature before damage or performance degradation.

```
TEMPERATURE RATINGS

    Class       Max Temp     Core Type
    ──────────────────────────────────────────────────
    85°C        185°F        General ferrite
    105°C       221°F        Power ferrite, some iron
    125°C       257°F        High-temp ferrite
    155°C       311°F        Specialty materials
    200°C+      392°F+       Military, aerospace

    Effects of high temperature:
    ├── Core permeability changes (inductance drifts)
    ├── Core losses increase
    ├── Saturation current decreases
    ├── Insulation breakdown (varnish/magnet wire)
    └── Permanent damage if exceeded


TEMPERATURE RISE SELF-HEATING:

    P_loss = I_ripple² × DCR (DC resistance loss)
           + Core loss (hysteresis + eddy current)
    
    Temperature rise ≈ P_loss × R_thermal
    
    Typical temperature rise at rated current:
    ├── Small SMD: 20-40°C
    ├── Large SMD: 30-50°C
    ├── Through-hole: 40-60°C
    └── Toroidal: 30-50°C
```

## Inductors in Circuits

### Inductor Charging and Discharging (LR Time Constant)

Like capacitors, inductors have a time constant with resistors.

```
LR TIME CONSTANT

    τ = L / R

    WHERE:
    τ = time constant (seconds)
    L = inductance (henries)
    R = resistance (ohms)


CHARGING (Current through inductor):

    I_L(t) = I_final × (1 - e^(-t/τ))
    
    Time to reach certain percentage:
    ├── 1τ (63.2%)
    ├── 2τ (86.5%)
    ├── 3τ (95.0%)
    ├── 4τ (98.2%)
    └── 5τ (99.3%) (considered fully charged)


DISCHARGING (Current through inductor):

    I_L(t) = I_initial × e^(-t/τ)


GRAPHICAL VIEW (Opposite of capacitor!):

    Current (%)
        │
    100 ┼────────────────────────────────────────
        │                                      
     90 ┼                                      
        │                                      
     80 ┼                                      
        │                                      
     70 ┼                                      
        │                                      
     63 ┼─────────●                             (1τ)
        │         ╲                             
     50 ┼          ╲                            
        │           ╲                           
     37 ┼            ●                          (1τ discharge)
        │             ╲                         
     30 ┼              ╲                        
        │               ╲                       
     20 ┼                ╲                      
        │                 ╲                     
     10 ┼                  ╲                    
        │                   ╲                   
      5 ┼                    ╲                  
        │                     ╲                 
      0 ┼──────────────────────●───────────────► Time
        0    1τ    2τ    3τ    4τ    5τ


PRACTICAL EXAMPLES:

    Example 1: L = 10 mH, R = 100Ω
    τ = 0.01 / 100 = 0.0001 s = 0.1 ms
    
    After 0.1 ms: current reaches 63% of final
    After 0.5 ms: 99% of final

    Example 2: L = 100 µH, R = 10Ω
    τ = 0.0001 / 10 = 0.00001 s = 10 µs
    
    Very fast! Used in switching power supplies
```

### Inductors in Series

```
INDUCTORS IN SERIES (No mutual coupling)

    L_total = L₁ + L₂ + L₃ + ... + L_n


EXAMPLE:

    ┌───[10mH]───[22mH]───[47mH]───┐
    │                               │
    │                               │
    └─────────────[I]───────────────┘

    L_total = 10 + 22 + 47 = 79 mH


CHARACTERISTICS:
├── Total inductance > largest individual
├── Same current through all inductors
├── Voltage divides (V = L × dI/dt)
├── Adding inductors in series INCREASES total inductance
└── Use: Higher inductance than available


WITH MUTUAL COUPLING (transformers):

    L_total = L₁ + L₂ + 2M (aiding)
    L_total = L₁ + L₂ - 2M (opposing)
    
    M = k × √(L₁ × L₂)
    k = coupling coefficient (0 to 1)
```

### Inductors in Parallel

```
INDUCTORS IN PARALLEL (No mutual coupling)

    1/L_total = 1/L₁ + 1/L₂ + 1/L₃ + ... + 1/L_n


EXAMPLE:

              ┌─────[10mH]─────┐
              │                │
              ├─────[22mH]─────┤
              │                │
              ├─────[47mH]─────┤
              │                │
            (+)               (-)
              │                │
              └─────[I]────────┘

    1/L_total = 1/10 + 1/22 + 1/47
               = 0.1 + 0.04545 + 0.02128
               = 0.16673
    L_total = 1 / 0.16673 = 6.0 mH


CHARACTERISTICS:
├── Total inductance < smallest individual
├── Same voltage across all inductors
├── Current divides (I_total = I₁ + I₂ + I₃)
├── Adding inductors in parallel DECREASES total inductance
└── Use: Lower inductance than available, higher current capacity


CURRENT DIVISION:

    I_n = I_total × (L_total / L_n)
    
    (Inductors are opposite of resistors!)
```

### Inductive Reactance

Inductors oppose AC current with reactance proportional to frequency.

```
INDUCTIVE REACTANCE FORMULA

    XL = 2πfL

    WHERE:
    XL = inductive reactance (ohms)
    f = frequency (hertz)
    L = inductance (henries)


FREQUENCY EFFECT:

    f = 0 (DC):      XL = 0Ω (short circuit)
    f = 60 Hz:       XL = 2π × 60 × L
    f = 1 kHz:       XL = 2π × 1000 × L (16.7× higher)
    f = ∞:           XL = ∞Ω (open circuit)


PRACTICAL EXAMPLES:

    L = 100 µH (0.0001 H)
    
    f = 60 Hz:   XL = 2π × 60 × 0.0001 = 0.0377Ω (short)
    f = 1 kHz:   XL = 2π × 1000 × 0.0001 = 0.628Ω
    f = 100 kHz: XL = 2π × 100000 × 0.0001 = 62.8Ω
    f = 1 MHz:   XL = 2π × 1000000 × 0.0001 = 628Ω (open-ish)

    L = 10 mH (0.01 H)
    
    f = 60 Hz:   XL = 2π × 60 × 0.01 = 3.77Ω
    f = 1 kHz:   XL = 2π × 1000 × 0.01 = 62.8Ω
    f = 10 kHz:  XL = 2π × 10000 × 0.01 = 628Ω
    f = 100 kHz: XL = 2π × 100000 × 0.01 = 6280Ω (open)


PHASE RELATIONSHIP:

    In a pure inductor: Current LAGS voltage by 90°
    
    Voltage:     ╱╲    ╱╲
                ╱  ╲  ╱  ╲
               ╱    ╲╱    ╲
    Current:        ╱╲    ╱╲
                  ╱  ╲  ╱  ╲
                 ╱    ╲╱    ╲
    
    "ELI the ICE man" memory aid:
    ELI: Voltage (E) leads Current (I) in an Inductor (L)
    ICE: Current leads Voltage in a Capacitor
```

## Inductor Applications

### Filtering (Smoothing)

Inductors smooth current in power supplies (opposite of capacitors).

```
LC FILTER (Pi filter)

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   AC in ──► Rectifier ──┬────[L]────┬───┬──► DC out        │
    │                         │           │   │                  │
    │                        ┌┴┐         ┌┴┐ ┌┴┐                 │
    │                        │ │         │ │ │ │                 │
    │                        │C1│        │L2│ │C2│                │
    │                        │ │         │ │ │ │                 │
    │                        └┬┘         └┬┘ └┬┘                 │
    │                         │           │   │                  │
    │   GND ──────────────────┴───────────┴───┴──────────────────│
    │                                                             │
    └─────────────────────────────────────────────────────────────┘

    Why use inductor instead of capacitor?
    ├── Inductor smooths CURRENT (capacitor smooths VOLTAGE)
    ├── Inductor stores energy in magnetic field
    ├── Better for high-current, low-voltage supplies
    └── Combined LC filter is very effective


INDUCTOR INPUT FILTER (Choke input):

    Advantages over capacitor input:
    ├── Better voltage regulation
    ├── Lower peak currents (easier on rectifiers)
    ├── Higher efficiency at high current
    └── Used in high-power supplies (tube amps, industrial)

    Disadvantages:
    ├── Requires minimum load (otherwise voltage rises)
    ├── Larger and heavier
    └── More expensive
```

### Switching Power Supplies (DC-DC Converters)

Inductors are essential in modern switching regulators.

```
BUCK CONVERTER (Step-down)

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Vin ──┬────[Switch]────┬────[L]────┬─── Vout              │
    │         │                │           │                     │
    │         │                │          ┌┴┐                    │
    │         │                │          │ │ C                  │
    │         │               ╱ ╲         │ │                    │
    │         │              ╱   ╲        └┬┘                    │
    │         │             │ Diode │       │                     │
    │         │              ╲   ╱        │                     │
    │         │               ╲ ╱         │                     │
    │         │                │          │                     │
    │         └────────────────┴──────────┴─────────────────────│
    │                                                             │
    │   GND                                                      │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘

    How it works:
    ├── Switch ON: Current flows through L, energy stored
    ├── Switch OFF: Diode conducts, L releases energy
    ├── Output voltage = Vin × duty cycle
    └── Inductor size determines ripple current


INDUCTOR SELECTION FOR BUCK CONVERTER:

    L = (Vin - Vout) × Vout / (Vin × f × ΔI)

    WHERE:
    ΔI = ripple current (typically 20-40% of I_out)
    f = switching frequency

    Example: Vin=12V, Vout=5V, I_out=2A, f=500kHz, ΔI=0.5A
    L = (12-5) × 5 / (12 × 500000 × 0.5)
    L = 35 / (12 × 250000) = 35 / 3,000,000 = 11.7 µH
    Use 10µH or 15µH
```

### Tuned Circuits (LC Resonators)

Inductors and capacitors together create resonant circuits.

```
LC RESONANT CIRCUIT

    Parallel LC (tank circuit):
    
    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   ┌─────[L]─────┐                                          │
    │   │             │                                          │
    │   │    ┌────┐   │                                          │
    │   │    │    │   │                                          │
    │   └────┤ C  ├───┘                                          │
    │        │    │                                              │
    │        └────┘                                              │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘

    Resonant frequency:
    f_res = 1 / (2π√(LC))

    At resonance:
    ├── Impedance peaks (parallel LC) → high Z
    ├── Impedance minimum (series LC) → low Z
    ├── Energy oscillates between L and C
    └── Used for tuning radios, oscillators, filters


EXAMPLE: AM radio tuning

    Desired frequency: 1 MHz (1000 kHz)
    Choose C = 200 pF
    
    L = 1 / (4π²f²C)
    L = 1 / (39.5 × (1e6)² × 200e-12)
    L = 1 / (39.5 × 1e12 × 200e-12)
    L = 1 / (39.5 × 200) = 1 / 7900 = 126 µH


EXAMPLE: FM radio tuning

    Desired frequency: 100 MHz
    Choose L = 0.1 µH
    
    C = 1 / (4π²f²L)
    C = 1 / (39.5 × (1e8)² × 0.1e-6)
    C = 1 / (39.5 × 1e16 × 1e-7)
    C = 1 / (39.5 × 1e9) = 1 / 3.95e10 = 25 pF
```

### Flyback (Snubber) Circuits

Inductors generate high voltage when switched off; snubbers protect switches.

```
FLYBACK DIODE (Freewheeling diode)

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Without diode:                    With diode:             │
    │                                                             │
    │   5V ──┬────[Switch]────[L]────┐    5V ──┬────[Sw]────[L]──┐│
    │        │                       │         │                 ││

    │                                                             │
    │   When switch opens:                   Current recirculates │
    │   High voltage spike!                  through diode safely │
    │   (can destroy transistor)             (voltage clamped)    │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘

    Flyback voltage without diode:
    V_peak = I_initial × √(L / C_stray)
    
    Example: I=1A, L=1mH, C_stray=100pF
    V_peak = 1 × √(0.001 / 100e-12) = 1 × √(1e7) = 3162V!
    
    Diode clamps to ~0.7V + supply voltage

    │            │   │R│ │                                        │
    │            │   │ │ │                                        │
    │            │   └┬┘ │                                        │
    │            │    │  │                                        │
    │            │   ┌┴┐ │                                        │
    │            │   │ │ │                                        │
    │            │   │C│ │                                        │
    │            │   │ │ │                                        │
    │            │   └┬┘ │                                        │
    │            │    │  │                                        │
    │            └────┴──┘                                        │
    │                 │                                           │
    │                GND                                          │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘

    Snubber absorbs inductive kickback energy
    Protects switch from voltage spikes
    Used in relay drivers, motor controllers, switching supplies
```

## Measuring Inductors

### Using an LCR Meter

```
INDUCTANCE MEASUREMENT

STEP 1: Power OFF circuit
    ┌─────────────────────────────────────────────────────────────┐
    │  WARNING: Inductors can hold energy!                        │
    │  Short inductor leads to discharge (if any stored energy)  │
    └─────────────────────────────────────────────────────────────┘

STEP 2: Set LCR meter to inductance mode (L)

STEP 3: Select test frequency
    ├── Low frequency (100Hz, 1kHz) for power inductors
    ├── High frequency (100kHz) for RF inductors
    └── Match frequency to application

STEP 4: Connect leads (polarity doesn't matter)

STEP 5: Read value


INTERPRETING RESULTS:

    Within tolerance:     Good
    Low reading:          Shorted turns (partial short)
    High reading:         Core damage, open circuit (some meters)
    "OL" or very high:    Open circuit (broken wire)
    Very low (<1Ω DCR):   Short circuit (touching turns)


TESTING WITHOUT LCR METER (Oscilloscope method):

    1. Build LR circuit with known resistor
    2. Apply square wave
    3. Measure time constant τ = L/R
    4. L = τ × R
    
    Example: R=1kΩ, τ=10µs
    L = 10e-6 × 1000 = 0.01 H = 10 mH
```

## Common Inductor Problems

### Problem 1: Open Circuit (Broken Wire)

```
SYMPTOMS:
├── No inductance (meter reads "OL")
├── Circuit loses filtering
├── Switching supply fails (output voltage wrong)
├── No current in that branch

CAUSES:
├── Overcurrent (wire melted)
├── Physical damage (cracked core, cut wire)
├── Corrosion (tinned wire turns black)
├── Poor solder connection at lead
├── Vibration fatigue (wire breaks at lead)

DIAGNOSIS:
├── Visual inspection (broken wire, burn mark)
├── Continuity test (should be low resistance)
├── LCR meter: "OL" or very high reading
├── Measure DCR: Should be low (ohms), not infinite

SOLUTIONS:
├── Replace inductor
├── For toroids: can rewind (if accessible)
├── Use thicker wire for high current
├── Add strain relief for leads
├── Use conformal coating to prevent corrosion
```

### Problem 2: Shorted Turns (Partial Short)

```
SYMPTOMS:
├── Inductance lower than specified
├── Q (quality factor) low (lossy)
├── Circuit detuned (resonant frequency off)
├── Inductor gets hot even with low current

CAUSES:
├── Overvoltage (insulation breakdown between turns)
├── Mechanical damage (wire insulation scraped)
├── Manufacturing defect
├── Excessive temperature (melted enamel)
├── Contamination (conductive material between turns)

DIAGNOSIS:
├── Measure inductance: lower than expected
├── Measure DCR: may be lower than spec (shorted turn reduces resistance)
├── Q meter: low Q indicates losses
├── Thermal imaging: hot spot indicates short

SOLUTIONS:
├── Replace inductor (shorted turns not repairable)
├── Use higher voltage rating insulation
├── Use toroidal instead of bobbin wound (less layer-to-layer stress)
├── Add varnish impregnation (fills gaps)
├── Derate current to prevent overheating
```

### Problem 3: Core Saturation

```
SYMPTOMS:
├── Inductance drops at high current
├── Current spikes increase
├── Circuit efficiency drops
├── Inductor gets hot (core loss)
├── Switching supply unstable

CAUSES:
├── Peak current exceeds Isat rating
├── DC bias too high
├── Core too small for application
├── Wrong core material (low saturation flux density)

DIAGNOSIS:
├── Measure inductance at various DC bias currents
├── Observe current waveform on oscilloscope
├── Inductance drop >20% indicates saturation
├── Calculate peak current in circuit

SOLUTIONS:
├── Use inductor with higher Isat rating
├── Increase inductance (lower ripple current)
├── Use gapped core (ferrite with gap, iron powder)
├── Use larger core size
├── Reduce peak current in circuit
├── Use core material with higher B_sat (iron powder vs ferrite)

SATURATION CURRENT COMPARISON:

    Core Type           Saturation Behavior        Isat vs Size
    ──────────────────────────────────────────────────────────
    Ferrite (no gap)    Very sharp, low Isat       Poor
    Ferrite (gapped)    Sharp, medium Isat         Good
    Iron powder         Soft (gradual), high Isat  Very good
    MPP (Molyperm)      Soft, very high Isat       Excellent
    Air core            Never saturates            Low inductance
```

### Problem 4: Excessive Noise (Audible Whine)

```
SYMPTOMS:
├── Inductor makes high-pitched whine (5-20 kHz)
├── Buzzing or clicking sound
├── Sound changes with load

CAUSES:
├── Magnetostriction (core expands/contracts at switching frequency)
├── Loose core halves (E-cores)
├── Loose wire (vibration against core)
├── Operating frequency in audible range (<20 kHz)
├── Current ripple too high

DIAGNOSIS:
├── Identify frequency (low hum = 120Hz, high whine = switching freq)
├── Press on inductor with non-conductive tool (sound changes? = loose)
├── Change load (sound may change frequency)
├── Listen at different input voltages

SOLUTIONS:
├── Increase switching frequency above 20 kHz (inaudible)
├── Varnish impregnation (glues windings and core)
├── Use toroidal inductor (inherently quiet)
├── Add mechanical damping (silicone, epoxy)
├── Reduce ripple current (increase inductance)
├── Tighten core clamps (for E-cores)

WHEN TO WORRY:
├── Sudden change in noise level (pre-failure indicator)
├── Noise accompanied by overheating
├── Arcing or sizzling sounds (dangerous!)
├── Otherwise, annoying noise usually not harmful
```

## Summary

1. **Inductors** store energy in a magnetic field; oppose changes in current

2. **Unit:** Henry (H) = 1V induced at 1A/s change

3. **Lenz's Law:** Induced voltage opposes change in current (V = -L × dI/dt)

4. **Core types:**
   - Air core: No saturation, very low L, best for RF (GHz)
   - Ferrite core: High L for size, saturates sharply, 10kHz-10MHz
   - Iron core: Very high L, low frequency (<1kHz), heavy
   - Toroidal: Low EMI, efficient, hard to wind

5. **Current rating (Isat):** Current where inductance drops 10-30%

6. **DC resistance (DCR):** Causes I²R loss, heating, voltage drop

7. **Self-resonant frequency (SRF):** Frequency where parasitic C resonates with L

8. **Time constant:** τ = L/R (current reaches 63% in 1τ, 99% in 5τ)

9. **Series:** L_total = L₁ + L₂ + ... (increases inductance)

10. **Parallel:** 1/L_total = 1/L₁ + 1/L₂ + ... (decreases inductance)

11. **Inductive reactance:** XL = 2πfL (increases with frequency)

12. **Phase:** Current lags voltage by 90° (ELI the ICE man)

13. **Energy storage:** E = ½LI² (joules)

14. **Resonance:** f = 1/(2π√(LC)) (LC tank circuit)

15. **Flyback diode** protects switches from inductive kickback (V = L × dI/dt)

16. **Common failures:** Open wire, shorted turns, core saturation, audible noise
