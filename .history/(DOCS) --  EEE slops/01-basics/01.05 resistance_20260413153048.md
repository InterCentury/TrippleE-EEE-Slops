# 01.05 - Resistance

## What is Resistance?

Resistance is the opposition to the flow of electric current. It is the electrical equivalent of friction in a mechanical system. Every material, component, and wire resists the flow of electrons to some degree.

The word "resistance" comes from the Latin *"resistere"*, meaning "to stand back" or "to oppose." In electronics, resistance is what converts electrical energy into heat, light, or motion.

## The Nature of Resistance

### Atomic-Level View

Resistance occurs when moving electrons collide with atoms in a conductor.

```

Without voltage (random motion):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    Atom          Atom          Atom          Atom
     │             │             │             │
    [Cu] ←─e⁻──  [Cu]  ──e⁻→   [Cu]  ←─e⁻──  [Cu]
     │             │             │             │
     │             │             │             │
     
    Electrons move randomly, occasional collisions
    Net current = ZERO


With voltage applied (drift with collisions):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    VOLTAGE ──►
    
    Atom         Atom         Atom         Atom
     │            │            │            │
    [Cu] ──e⁻→   [Cu] ──e⁻→   [Cu] ──e⁻→   [Cu]
     │      ╲     │      ╲     │      ╲     │
     │       ╲    │       ╲    │       ╲    │
     │        ◄───┘        ◄───┘        ◄───┘
     
    Electrons drift forward but constantly collide
    Collisions = RESISTANCE = Heat generation


WHAT HAPPENS DURING A COLLISION:

    Before collision:          After collision:
    
    Electron ──►               Electron ──► (slower)
       │                            │
       │      Atom                  │      Atom
       │     [Cu]                   │     [Cu] (vibrating)
       │      │                     │      │
       │      │                     │      │
       ▼      ▼                     ▼      ▼
       
    Electron transfers energy     Atom vibrates more (HEAT!)
    to the atom
```

                └─────────────────┘
    
    ┌─────────────────────────────────────────┐
    │  High resistance = less current         │

ANALOGY SUMMARY:

    ─────────────────────────────────────

## The Unit of Resistance: The Ohm (Ω)

### Definition

```
THE OHM (Ω)

DEFINITION:
1 Ohm = Resistance that allows 1 Ampere of current 
        when 1 Volt is applied

FORMULA: R = V / I


VISUALIZING 1 OHM:

    1 Volt ──► ┌─────┐ ──► 1 Ampere
               │ 1Ω  │
               └─────┘
    
    A 1Ω resistor with 1V across it passes 1A


COMMON RESISTANCE VALUES:

    ┌─────────────────────────────────────────────┐
    │                                             │
    │  0.01 Ω  ──  Short piece of copper wire     │
    │  0.1 Ω   ──  Longer wire, connector         │
    │  1 Ω     ──  Current sensing resistor       │
    │  10 Ω    ──  LED current limiter (bright)   │
    │  100 Ω   ──  LED current limiter (normal)   │
    │  220 Ω   ──  Common LED resistor (5V)       │
    │  330 Ω   ──  Common LED resistor (9V)       │
    │  470 Ω   ──  Common LED resistor (12V)      │
    │  1 kΩ    ──  Pull-up resistor               │
    │  4.7 kΩ  ──  Common for transistors         │
    │  10 kΩ   ──  Common for potentiometers      │
    │  100 kΩ  ──  High impedance input           │
    │  1 MΩ    ──  Very high resistance           │

### 2. Length

```
RESISTANCE vs LENGTH (Directly Proportional)

    Longer wire = More resistance
    Shorter wire = Less resistance

    FORMULA: R ∝ L (R increases as L increases)


VISUAL EXAMPLE:

    1 meter of wire:
    ┌────────────────────────────────────────────────┐
    │  [=====]                                       │
    │  R = 0.1 Ω                                     │
    └────────────────────────────────────────────────┘
    
    2 meters of wire:
    ┌────────────────────────────────────────────────┐
    │  [=====][=====]                                │
    │  R = 0.2 Ω  (2× longer = 2× resistance)        │
    └────────────────────────────────────────────────┘
    
    10 meters of wire:
    ┌────────────────────────────────────────────────┐
    │  [=====][=====][=====][=====][=====]           │
    │  [=====][=====][=====][=====][=====]           │
    │  R = 1.0 Ω  (10× longer = 10× resistance)      │
    └────────────────────────────────────────────────┘


PRACTICAL EXAMPLE:

    Question: A 100m spool of 22 AWG copper wire has 
              5.3 Ω resistance. What is the resistance 
              of 250m of the same wire?
    
    Solution: R ∝ L
              R₂ / R₁ = L₂ / L₁
              R₂ = R₁ × (L₂ / L₁)
              R₂ = 5.3 Ω × (250 / 100)
              R₂ = 5.3 Ω × 2.5 = 13.25 Ω
```

### 3. Cross-Sectional Area (Thickness)

```
RESISTANCE vs AREA (Inversely Proportional)

    Thicker wire = Less resistance
    Thinner wire = More resistance

    FORMULA: R ∝ 1/A (R decreases as A increases)


VISUAL EXAMPLE:

    Thin wire (small area):
    ┌────────────────────────────────────────────────┐
    │  .                                             │
    │  .  (cross section = 0.5 mm²)                  │
    │  .                                             │
    │  R = 0.2 Ω per meter                           │

    Thick wire (large area):
    ┌────────────────────────────────────────────────┐
    │  ████                                          │
    │  ████  (cross section = 2.0 mm²)               │
    │  ████                                          │
    │  R = 0.05 Ω per meter (4× less)                │
    └────────────────────────────────────────────────┘


PRACTICAL EXAMPLE:

    Question: A wire with 1 mm² area has 0.1 Ω/m 
              resistance. What is the resistance per 
              meter of wire with 4 mm² area?
    
    Solution: R ∝ 1/A
              R₂ / R₁ = A₁ / A₂

### 4. Temperature

```
RESISTANCE vs TEMPERATURE

For most metals (positive temperature coefficient):
    Hotter = More resistance
    Colder = Less resistance

For semiconductors and carbon (negative temperature coefficient):
    Hotter = Less resistance
    Colder = More resistance


METAL BEHAVIOR (Copper, Aluminum, etc.):

    Resistance
        │
        │                                   
        │                              ╱
        │                           ╱
        │                        ╱
        │                     ╱
        │                  ╱
        │               ╱
        │            ╱
        │         ╱
        │      ╱
        │   ╱
        │╱
        └────────────────────────────────► Temperature
        
        -273°C    0°C     20°C    100°C    200°C
        (Absolute Zero)


WHY METALS INCREASE RESISTANCE WITH HEAT:

    Cold metal:                 Hot metal:
    
    Atoms:  ○──○──○──○         Atoms:  ○  ○  ○  ○
            │  │  │  │                 │  │  │  │
            ○──○──○──○                 ○  ○  ○  ○
            │  │  │  │                 │  │  │  │
            ○──○──○──○                 ○  ○  ○  ○
            
    Atoms vibrate little       Atoms vibrate a lot
    Electrons flow easily      Electrons collide more
    LOW RESISTANCE             HIGH RESISTANCE


TEMPERATURE COEFFICIENT FORMULA:

    R₂ = R₁ × [1 + α(T₂ - T₁)]
    
    WHERE:
    R₂ = Resistance at temperature T₂
    R₁ = Resistance at temperature T₁
    α = Temperature coefficient of resistance
    T₂ = Final temperature (°C)
    T₁ = Initial temperature (°C)


TYPICAL α VALUES (per °C):

    Material        α (20°C)        Behavior
    ───────────────────────────────────────────
    Copper          +0.00393        Increases with heat
    Aluminum        +0.00403        Increases with heat
    Silver          +0.00380        Increases with heat
    Iron            +0.00500        Increases with heat
    Tungsten        +0.00450        Increases with heat
    Nichrome        +0.00017        Nearly constant
    Carbon          -0.00050        Decreases with heat
    Silicon         -0.07500        Decreases with heat (strong)


PRACTICAL EXAMPLES:

    Example 1: Copper wire at 20°C has 10 Ω.
               What is resistance at 100°C?
    
    R₂ = 10 Ω × [1 + 0.00393 × (100 - 20)]
    R₂ = 10 Ω × [1 + 0.00393 × 80]
    R₂ = 10 Ω × [1 + 0.3144]
    R₂ = 10 Ω × 1.3144 = 13.14 Ω
    
    (31.4% increase!)


    Example 2: Light bulb filament (Tungsten)
               Cold resistance: 10 Ω at 20°C
               Hot resistance: 144 Ω at 2500°C
    
    R₂ = 10 Ω × [1 + 0.0045 × (2500 - 20)]
    R₂ = 10 Ω × [1 + 0.0045 × 2480]
    R₂ = 10 Ω × [1 + 11.16]
    R₂ = 10 Ω × 12.16 = 121.6 Ω (close to 144 Ω)
    
    This is why bulbs have high inrush current!
    (Cold filament = low resistance = surge current)
```

## The Resistance Formula (Putting It All Together)

```
COMPLETE RESISTANCE FORMULA

         ρ × L
    R = ───────
           A

WHERE:
R = Resistance (Ω)
ρ = Resistivity (Ω·m) - material property
L = Length (m)
A = Cross-sectional area (m²)


PRACTICAL CALCULATION EXAMPLE:

Problem: Calculate resistance of 100m of 1.5 mm² copper wire.

Given:
ρ (copper) = 1.68 × 10⁻⁸ Ω·m
L = 100 m
A = 1.5 mm² = 1.5 × 10⁻⁶ m²

Solution:
R = (1.68 × 10⁻⁸ × 100) / (1.5 × 10⁻⁶)
R = (1.68 × 10⁻⁶) / (1.5 × 10⁻⁶)
R = 1.68 / 1.5 = 1.12 Ω

Answer: 100m of 1.5 mm² copper wire has 1.12 Ω resistance


CHECK WITH WIRE TABLE:
Standard 1.5 mm² copper wire is approximately AWG 15-16
Resistance per 100m is typically 1.2-1.3 Ω ✓
```

## Resistor Power Rating

Resistors convert electrical energy into heat. Power rating tells you how much heat they can safely dissipate.

```
POWER RATING AND PHYSICAL SIZE

    Power Rating    Size (approx)    Typical Use
    ──────────────────────────────────────────────
    1/8 W (0.125W)  Tiny (2mm)       Surface mount, very low power
    1/4 W (0.25W)   Small (6mm)      Most common for hobby/Arduino
    1/2 W (0.5W)    Medium (9mm)     Brighter LEDs, small loads
    1 W             Large (11mm)     Power supplies, audio
    2 W             Larger (15mm)    Higher power circuits
    5 W             Big (22mm)       Power supplies, amplifiers
    10 W+           Very large       Industrial, heaters


POWER CALCULATION:

    P = V × I
    P = I² × R
    P = V² / R


EXAMPLE: Is a 1/4 W resistor safe?

    Circuit: 9V battery, 220Ω resistor
    
    Calculate actual power:
    P = V² / R = (9 × 9) / 220 = 81 / 220 = 0.368 W
    
    0.368 W > 0.25 W rating!
    
    RESULT: 1/4 W resistor would overheat (too hot!)
    SOLUTION: Use 1/2 W or 1 W resistor


WHAT HAPPENS WHEN YOU EXCEED POWER RATING:

    Slightly over (0.3W in 0.25W):
    └── Resistor gets very hot, may discolor, shortens life
    
    2× over rating:
    └── Resistor burns, smoke, changes value
    
    5× over rating:
    └── Resistor catches fire, opens circuit
    
    WAY over (short circuit):
    └── Resistor explodes/vaporizes (literally!)

ALWAYS use resistor with power rating ABOVE calculated power.
(Add 50-100% safety margin)
```

## Resistance in Series and Parallel

### Series Resistance

```
RESISTORS IN SERIES

    ┌─────┐    ┌─────┐    ┌─────┐
    │ R1  │    │ R2  │    │ R3  │
    │ 10Ω │    │ 20Ω │    │ 30Ω │
    └──┬──┘    └──┬──┘    └──┬──┘
       │          │          │
       └──────────┴──────────┘
                    │
                 Total = ?

FORMULA:
    R_total = R₁ + R₂ + R₃ + ... + R_n


EXAMPLE:
    R_total = 10 + 20 + 30 = 60Ω


KEY PROPERTIES:
├── Total resistance is LARGER than any individual resistor
├── Current is same through all resistors
├── Voltage divides across resistors (voltage divider)
└── Adding resistors in series INCREASES total resistance
```

### Parallel Resistance

```
RESISTORS IN PARALLEL

              ┌─────┐
          ┌───┤ R1  ├───┐
          │   │ 10Ω │   │
          │   └─────┘   │
          │             │
          │   ┌─────┐   │
          ├───┤ R2  ├───┤
          │   │ 20Ω │   │
          │   └─────┘   │
          │             │
          │   ┌─────┐   │
          └───┤ R3  ├───┘
              │ 30Ω │
              └─────┘
                
              Total = ?

FORMULA (two resistors):
    R_total = (R₁ × R₂) / (R₁ + R₂)

FORMULA (three or more):
    1/R_total = 1/R₁ + 1/R₂ + 1/R₃ + ... + 1/R_n


EXAMPLE (three resistors):
    1/R_total = 1/10 + 1/20 + 1/30
    1/R_total = 0.1 + 0.05 + 0.0333 = 0.1833
    R_total = 1 / 0.1833 = 5.45Ω


KEY PROPERTIES:
├── Total resistance is SMALLER than the smallest resistor
├── Voltage is same across all resistors
├── Current divides through resistors (current divider)
└── Adding resistors in parallel DECREASES total resistance


SPECIAL CASE: Equal resistors in parallel
    R_total = R / n
    
    Example: Four 100Ω resistors in parallel
    R_total = 100 / 4 = 25Ω
```

### Series-Parallel Combinations

```
SERIES-PARALLEL EXAMPLE

    ┌─────────────────────────────────────────────┐
    │                                             │
    │      R1=10Ω                                 │
    │   ┌───┬───┐                                 │
    │   │   │   │                                 │
    │   │   R2=20Ω    R3=30Ω                      │
    │   │   │        │                            │
    │   │   └───┬────┼────┐                       │
    │   │       │    │    │                       │
    │   └───────┘    └────┼────┐                  │
    │                     │    │                  │
    │                     R4=40Ω                  │
    │                     │    │                  │
    │                     └────┘                  │
    │                                             │
    └─────────────────────────────────────────────┘

STEP-BY-STEP SOLUTION:

Step 1: Combine R2 and R3 (parallel)
    1/R23 = 1/20 + 1/30 = 0.05 + 0.0333 = 0.0833
    R23 = 12Ω

Step 2: Combine R23 with R4 (series)
    R234 = R23 + R4 = 12 + 40 = 52Ω

Step 3: Combine R1 with R234 (parallel)
    1/R_total = 1/10 + 1/52 = 0.1 + 0.01923 = 0.11923
    R_total = 8.39Ω

ANSWER: Total resistance = 8.39Ω
```

## Measuring Resistance

### Using an Ohmmeter / Multimeter

```
HOW TO MEASURE RESISTANCE

STEP 1: Power OFF!
    ┌─────────────────────────────────────────────┐
    │  WARNING: Never measure resistance on a    │
    │  live circuit! The meter applies its own   │
    │  small voltage/current to make the         │
    │  measurement. Live voltage will damage     │
    │  the meter or give wrong readings.         │
    └─────────────────────────────────────────────┘

STEP 2: Isolate the resistor
    ├── Remove resistor from circuit (best)
    ├── Or measure before soldering
    └── In-circuit measurements may be affected by other components

STEP 3: Set multimeter to Ω (resistance)
    ├── Choose auto-range or appropriate range
    ├── Start with highest range if unsure
    └── 200Ω range for small resistors, 200kΩ for large

STEP 4: Connect leads
    ┌─────────┐
    │   COM   │─── black lead ──► resistor one end
    │   VΩ    │─── red lead  ──► resistor other end
    └─────────┘
    
    Polarity doesn't matter for resistance!

STEP 5: Read display
    ├── "0.0" = short circuit (0Ω)
    ├── "OL" or "1   " = open circuit (infinite)
    └── Numeric value = resistance in selected range


IN-CIRCUIT MEASUREMENT TIPS:

    Component you want to measure:
           R1
        ┌─┬─┐
        │ │ │
        └─┼─┘
          │
    ┌─────┼─────┐
    │     │     │
    │  ┌──┴──┐  │
    │  │ R2  │  │
    │  └─────┘  │
    └───────────┘
    
    Problem: R2 is in parallel with R1!
    Solution: Desolder one leg of R1, or measure before soldering
```

## Common Resistance Problems and Solutions

### Problem 1: Open Circuit (Infinite Resistance)

```
SYMPTOMS:
├── No current flow
├── Component doesn't work
├── Meter reads "OL" or very high

CAUSES:
├── Broken wire/trace
├── Burned out resistor (overheated)
├── Cold solder joint
├── Switch in OFF position

SOLUTION:
├── Visually inspect for damage
├── Check continuity with beeper mode
├── Replace burned components
├── Re-solder bad joints
```

### Problem 2: Short Circuit (Zero Resistance)

```
SYMPTOMS:
├── Excessive current
├── Power supply shuts down (overcurrent)
├── Wires get hot
├── Meter reads 0.0Ω or very low

CAUSES:
├── Solder bridge between pads
├── Wire insulation melted (bare wires touching)
├── Component failed shorted
├── Wrong connection (bypassing resistor)

SOLUTION:
├── Visual inspection
├── Check for unintended connections
├── Remove solder bridges
├── Separate touching wires
├── Replace shorted components
```

### Problem 3: Incorrect Resistance Value

```
SYMPTOMS:
├── Circuit works but not as expected
├── LED too dim or too bright
├── Wrong voltage at test points

CAUSES:
├── Wrong resistor installed (color code misread)
├── Resistor drifted in value (old/burned)
├── Measured in-circuit (other components affect reading)
├── Tolerance too wide for precision circuit

SOLUTION:
├── Double-check color codes
├── Measure out of circuit
├── Use 1% metal film for precision
├── Replace drifted resistors
```

## Quick Reference Table

| Term | Symbol | Unit | Formula | Description |
|------|--------|------|---------|-------------|
| Resistance | R | Ohm (Ω) | R = V/I | Opposition to current |
| Resistivity | ρ | Ohm-meter (Ω·m) | ρ = R×A/L | Material property |
| Conductance | G | Siemens (S) | G = 1/R | Ease of flow (opposite) |
| Temperature Coefficient | α | per °C | α = ΔR/(R×ΔT) | Resistance change with temp |
| Series Resistance | R_total | Ω | R₁ + R₂ + ... | Adds together |
| Parallel Resistance | R_total | Ω | 1/(1/R₁+1/R₂+...) | Less than smallest |

## Summary

1. **Resistance** is opposition to current flow (electrical friction)

2. **Unit:** Ohm (Ω) = 1V / 1A

3. **Four factors** determine resistance:
   - Material (resistivity ρ)
   - Length (L) - longer = more resistance
   - Area (A) - thicker = less resistance
   - Temperature (T) - metals: hotter = more resistance

4. **Formula:** R = ρ × L / A

5. **Resistor types:**
   - Fixed: Carbon, metal film, wirewound
   - Variable: Potentiometer, trimmer, rheostat

6. **Color code:** 4 bands (Digit, Digit, Multiplier, Tolerance)

7. **Power rating:** Must exceed I²R losses (use 2× safety margin)

8. **Series:** R_total = R₁ + R₂ + ... (increases)

9. **Parallel:** 1/R_total = 1/R₁ + 1/R₂ + ... (decreases)

10. **Common faults:**
    - Open circuit (infinite Ω) = no current
    - Short circuit (0 Ω) = too much current
    - Wrong value = incorrect circuit behavior

11. **Measure resistance** ONLY with power OFF

12. **Temperature effects** matter for precision circuits and high-power designs
