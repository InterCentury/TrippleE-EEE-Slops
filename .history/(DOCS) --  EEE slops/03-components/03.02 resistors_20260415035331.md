# 03.02 - Resistors

## What is a Resistor?

A resistor is a passive two-terminal electrical component that implements electrical resistance as a circuit element. Resistors oppose the flow of electric current, converting electrical energy into heat. They are the most common electronic component, found in virtually every electronic circuit.

The resistor was discovered independently by several inventors in the early 19th century, with Sir Charles Wheatstone (of Wheatstone bridge fame) being one of the early developers. Today, resistors are manufactured in countless varieties, from tiny surface-mount chips smaller than a grain of sand to large power resistors that can handle thousands of watts.

## Basic Resistor Properties

### What Resistors Do

Resistors serve several essential functions in electronic circuits:

```

1. LIMIT CURRENT:
   ┌─────────────────────────────────────────────────────────────┐
   │                                                             │
   │   Without resistor:          With resistor:                 │
   │                                                             │
   │   5V ──┬────[LED]────┐       5V ──┬────[220Ω]────[LED]────┐ │

   ┌─────────────────────────────────────────────────────────────┐
   │                                                             │

   │                                                             │

   │                                                             │

   ┌─────────────────────────────────────────────────────────────┐
   │                                                             │
   │   Pull-up:                   Pull-down:                     │
   │                                                             │
   │   5V ──┬────[10kΩ]────┬───   ┌───┬────[10kΩ]──── 0V         │
   │        │              │      │   │                          │
   │        │    ┌────┐    │      │ ┌┴┐ │                        │
   │        │    │ IC │    │      │ │IC│ │                       │

   │                                                             │
   └─────────────────────────────────────────────────────────────┘

4. CURRENT SENSING:
   ┌─────────────────────────────────────────────────────────────┐
   │                                                             │
   │   Current ──► ┌────[0.1Ω]────┐                              │
   │               │              │                              │
   │               │    ┌────┐    │                              │
   │               │    │ADC │    │                              │
   │               │    └────┘    │                              │
   │               │              │                              │
   │               └──────────────┘                              │
   │                                                             │
   │   V = I × R = I × 0.1Ω                                      │
   │   Measure voltage to determine current                      │
   │                                                             │
   └─────────────────────────────────────────────────────────────┘

5. TIMING (with capacitor):
   ┌─────────────────────────────────────────────────────────────┐
   │                                                             │
   │   5V ──┬────[R]────┬──── Capacitor ── 0V                    │
   │        │           │                                        │
   │        │    ┌────┐ │                                        │
   │        │    │ IC │ │                                        │
   │        │    └────┘ │                                        │
   │        │           │                                        │
   │        └───────────┘                                        │
   │                                                             │
   │   RC time constant: τ = R × C                               │
   │   Controls timing in oscillators, delays, filters           │
   │                                                             │
   └─────────────────────────────────────────────────────────────┘
```

### Special Resistors

```
SPECIAL RESISTOR TYPES

THERMISTOR (Temperature-dependent):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    NTC (Negative Temperature Coefficient):
    ├── Resistance DECREASES as temperature INCREASES
    ├── Use: Temperature sensing, inrush current limiting
    └── Common: 10kΩ at 25°C (resistance drops to ~1kΩ at 100°C)

    PTC (Positive Temperature Coefficient):
    ├── Resistance INCREASES as temperature INCREASES
    ├── Use: Overcurrent protection, self-resetting fuses
    └── Common: Polyfuse (resistance jumps from <1Ω to >100Ω)


PHOTORESISTOR (Light-dependent - LDR):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Light → resistance decreases                              │
    │   Dark → resistance increases                               │
    │                                                             │
    │   Common: 10kΩ to 1MΩ (dark), 100Ω to 1kΩ (light)          │
    │   Use: Light sensors, automatic night lights               │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘


VARISTOR (Voltage-dependent - VDR):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    ├── Low voltage → very high resistance (open)
    ├── High voltage spike → low resistance (shorts)
    ├── Use: Surge protection (MOV - Metal Oxide Varistor)
    └── Common: 130V, 275V, 320V (clamping voltages)


RESISTOR NETWORKS (Arrays):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## Resistor Power Rating

### Power Dissipation

Resistors convert electrical energy into heat. Power rating indicates maximum heat dissipation.

```
POWER FORMULAS FOR RESISTORS

    P = V × I
    P = I² × R
    P = V² / R


STANDARD POWER RATINGS:

    Rating        Size (through-hole)    Typical Use
    ──────────────────────────────────────────────────────────
    1/8W (0.125W) 2mm diameter          Very small, signal
    1/4W (0.25W)  3mm diameter          Most common (Arduino)
    1/2W (0.5W)   4mm diameter          Brighter LEDs
    1W            6mm diameter          Power circuits
    2W            8mm diameter          Higher power
    5W            15×5mm rectangular    Power supplies
    10W+          Large, often with heatsink Industrial


EXAMPLE: Is 1/4W resistor safe for this circuit?

    Circuit: 9V battery, 220Ω resistor (LED current limiter)
    
    Calculate actual power:
    P = V² / R = (9 × 9) / 220 = 81 / 220 = 0.368W
    
    0.368W > 0.25W rating → TOO HIGH!
    
    Solutions:
    ├── Use 1/2W resistor (0.5W rating)
    ├── Use higher resistance (less current, less power)
    ├── Use two resistors in series (each dissipates less)
    └── Use two resistors in parallel (each dissipates less)


PARALLEL RESISTORS TO SHARE POWER:

    Two 440Ω resistors in parallel = 220Ω equivalent
    Each resistor: P = (9V)² / 440Ω = 81 / 440 = 0.184W
    0.184W < 0.25W → SAFE!


SERIES RESISTORS TO SHARE POWER:

    Two 110Ω resistors in series = 220Ω equivalent
    Each resistor: V across each = 4.5V
    P = (4.5)² / 110 = 20.25 / 110 = 0.184W → SAFE!
```

### Temperature Rise

```
TEMPERATURE RISE vs POWER

    Approximate temperature rise above ambient:
    
    Power      Temperature Rise (typical)
    ──────────────────────────────────────
    1/8W       20-30°C
    1/4W       30-40°C
    1/2W       40-50°C
    1W         50-70°C
    2W         60-80°C
    5W         80-100°C
    
    Example: 1/4W resistor dissipating 0.2W in 25°C room
    Temperature = 25°C + (0.2/0.25 × 40°C) = 25 + 32 = 57°C
    (Hot but safe for most components)


WHEN RESISTORS GET TOO HOT:

    1× power rating: Hot but safe (rated temperature)
    1.5× rating: Very hot, may discolor, shortened life
    2× rating: Smoking, charring, value changes
    3× rating: Glowing red, catastrophic failure
    5× rating: Explodes/vaporizes (literally!)

    Always use resistor with power rating ABOVE calculated power.
    Recommended safety margin: 50-100%
```

## Resistor Networks (Series and Parallel)

### Resistors in Series

```
SERIES RESISTOR FORMULA

    R_total = R₁ + R₂ + R₃ + ... + R_n


EXAMPLE:

    ┌─────[100Ω]─────[220Ω]─────[330Ω]─────┐
    │                                       │
    │                                       │
    └─────────────────[9V]─────────────────┘

    R_total = 100 + 220 + 330 = 650Ω
    I = V / R = 9V / 650Ω = 0.0138A = 13.8mA


CHARACTERISTICS:
├── Total resistance > largest individual resistor
├── Current same through all resistors
├── Voltage divides across resistors (voltage divider)
├── Adding resistors INCREASES total resistance
├── Power adds: P_total = P₁ + P₂ + P₃
└── If one resistor fails open → all stop working


WHEN TO USE SERIES:
├── Need higher resistance than available
├── Voltage divider applications
├── Sharing power (each resistor dissipates less)
└── Creating non-standard values
```

### Resistors in Parallel

```
PARALLEL RESISTOR FORMULA

    1/R_total = 1/R₁ + 1/R₂ + 1/R₃ + ... + 1/R_n

    For two resistors: R_total = (R₁ × R₂) / (R₁ + R₂)


EXAMPLE:

              ┌─────[100Ω]─────┐
              │                │
              ├─────[220Ω]─────┤
              │                │
              ├─────[330Ω]─────┤
              │                │
            (+)               (-)
              │                │
              └─────[9V]───────┘

    1/R_total = 1/100 + 1/220 + 1/330
               = 0.01 + 0.004545 + 0.003030
               = 0.017575
    R_total = 1 / 0.017575 = 56.9Ω
    
    I_total = V / R_total = 9V / 56.9Ω = 0.158A = 158mA


CHARACTERISTICS:
├── Total resistance < smallest individual resistor
├── Voltage same across all resistors
├── Current divides (current divider)
├── Adding resistors DECREASES total resistance
├── Power adds: P_total = P₁ + P₂ + P₃
└── If one resistor fails open → others still work


WHEN TO USE PARALLEL:
├── Need lower resistance than available
├── Current divider applications
├── Sharing power (each resistor dissipates less)
├── Redundancy (one fails, circuit still works)
└── Creating non-standard values


EQUAL RESISTORS IN PARALLEL:

    R_total = R / n
    
    Example: Four 100Ω resistors in parallel
    R_total = 100 / 4 = 25Ω
```

### Series-Parallel Combinations

```
SERIES-PARALLEL EXAMPLE

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │      R1=100Ω                                               │
    │   ┌───┬───┐                                                │
    │   │   │   │                                                │
    │   │   R2=220Ω   R3=330Ω                                   │
    │   │   │        │                                           │
    │   │   └───┬────┼────┐                                      │
    │   │       │    │    │                                      │
    │   └───────┘    └────┼────┐                                 │
    │                     │    │                                 │
    │                     R4=470Ω                               │
    │                     │    │                                 │
    │                     └────┘                                 │
    │                     │                                      │
    │                   (+)  (-)                                │
    │                     │                                      │
    │                   [12V]                                   │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘


SOLUTION STRATEGY:

Step 1: Combine R2 and R3 (parallel)
    1/R23 = 1/220 + 1/330 = 0.004545 + 0.003030 = 0.007575
    R23 = 132Ω

Step 2: Combine R23 with R4 (series)
    R234 = R23 + R4 = 132 + 470 = 602Ω

Step 3: Combine R1 with R234 (parallel)
    1/R_total = 1/100 + 1/602 = 0.01 + 0.001661 = 0.011661
    R_total = 85.7Ω

Step 4: Total current
    I_total = V / R_total = 12V / 85.7Ω = 0.140A = 140mA

Step 5: Current through R1
    V_R1 = I_total × R1? No, that's wrong (R1 not in series)
    Actually: V_R1 = V_total = 12V (parallel branch)
    I_R1 = V_R1 / R1 = 12V / 100Ω = 0.120A = 120mA

Step 6: Current through R234 branch
    I_R234 = I_total - I_R1 = 140 - 120 = 20mA

Step 7: Voltage across R23 and R4
    V_R4 = I_R234 × R4 = 0.020A × 470Ω = 9.4V
    V_R23 = V_total - V_R4 = 12 - 9.4 = 2.6V

Step 8: Current through R2 and R3
    I_R2 = V_R23 / R2 = 2.6V / 220Ω = 0.0118A = 11.8mA
    I_R3 = V_R23 / R3 = 2.6V / 330Ω = 0.00788A = 7.9mA
    
    Check: 11.8 + 7.9 = 19.7mA ≈ 20mA ✓


RESULTS SUMMARY:

    Resistor    R (Ω)    I (mA)    V (V)     P (mW)
    ────────────────────────────────────────────────
    R1          100      120       12.0      1440
    R2          220      11.8      2.6       30.7
    R3          330      7.9       2.6       20.5
    R4          470      20.0      9.4       188
    Total       85.7     140       12.0      1680
```

## Measuring Resistors

### Using an Ohmmeter / Multimeter

```
HOW TO MEASURE RESISTANCE

STEP 1: Power OFF!
    ┌─────────────────────────────────────────────────────────────┐
    │  WARNING: Never measure resistance on a live circuit!      │
    │  The meter applies its own small voltage/current to make   │
    │  the measurement. Live voltage will damage the meter or    │
    │  give wrong readings.                                      │
    └─────────────────────────────────────────────────────────────┘

STEP 2: Isolate the resistor (best practice)
    ├── Remove resistor from circuit (desolder one leg)
    ├── Or measure before soldering
    └── In-circuit measurements affected by parallel paths

STEP 3: Set multimeter to Ω (resistance mode)
    ├── Choose auto-range or appropriate range
    ├── 200Ω range for small resistors
    ├── 20kΩ range for kΩ resistors
    ├── 2MΩ range for MΩ resistors
    └── "OL" means over-limit (too high for range)

STEP 4: Connect leads (polarity doesn't matter for resistance)

STEP 5: Read display
    ├── "0.0" = short circuit (0Ω)
    ├── "OL" or "1   " = open circuit (infinite)
    └── Numeric value = resistance in selected range


IN-CIRCUIT MEASUREMENT:

    Problem: Other components in parallel affect reading
    
    Example: Measuring R1 in this circuit
              R1 (100Ω)
           ┌─┬─┐
           │ │ │
           └─┼─┘
             │
        ┌────┼────┐
        │    │    │
        │  ┌─┴─┐  │
        │  │ R2│  │  (R2 in parallel with R1)
        │  │220│  │
        │  └───┘  │
        └─────────┘
    
    Solution: Desolder one leg of R1
    Or measure known good board for comparison


TESTING RESISTOR WITH MULTIMETER:

    Good resistor: Reading within tolerance (e.g., 220Ω ±5% = 209-231Ω)
    Open resistor: "OL" or infinite reading
    Shorted resistor: 0Ω or very low reading
    Drifted resistor: Reading outside tolerance (e.g., 250Ω for 220Ω)
```

## Common Resistor Problems

### Problem 1: Open Resistor

```
SYMPTOMS:
├── No current in that branch
├── Voltage across resistor = supply voltage (no drop)
├── Circuit malfunction (LED won't light, etc.)

CAUSES:
├── Overheating (exceeded power rating)
├── Mechanical damage (cracked body)
├── Manufacturing defect
├── Old age (carbon composition drifts up until open)

DIAGNOSIS:
├── Visual inspection: burn marks, cracking
├── Multimeter: reads "OL" or infinite
├── Measure voltage across resistor (should be 0V if OK)
├── If voltage = supply voltage, resistor likely open

SOLUTIONS:
├── Replace with same resistance and higher power rating
├── Check why it failed (overcurrent? undersized?)
├── Use metal film instead of carbon for reliability
├── Add fuse or protection circuit
```

### Problem 2: Resistor Drift (Value Change)

```
SYMPTOMS:
├── Circuit works but not as designed
├── LED too dim or too bright
├── Timing circuit off (too fast or slow)
├── Voltage divider output wrong

CAUSES:
├── Overheating (temporary or permanent)
├── Old age (carbon composition drifts)
├── Moisture absorption
├── Mechanical stress (cracked body)

DIAGNOSIS:
├── Measure resistance (out of circuit)
├── Compare to color code/tolerance
├── Example: 220Ω ±5% measures 280Ω (27% high)

SOLUTIONS:
├── Replace resistor
├── Use tighter tolerance (1% instead of 5%)
├── Use metal film (more stable than carbon)
├── Reduce power dissipation (use higher wattage)
├── Add cooling (ventilation, heatsink)
```

### Problem 3: Noisy Resistor

```
SYMPTOMS:
├── Audio equipment has hiss or crackle
├── Sensitive measurements fluctuate
├── Oscilloscope shows noise on DC

CAUSES:
├── Carbon composition resistors (inherently noisy)
├── Damaged resistor (internal arcing)
├── Loose end cap connections
├── High voltage causing internal discharge

DIAGNOSIS:
├── Substitute with metal film resistor
├── Measure with oscilloscope (AC coupled)
├── Tap resistor while listening (mechanical noise)

SOLUTIONS:
├── Replace carbon comp with metal film
├── Use wirewound for lowest noise (but inductive)
├── Use higher wattage (runs cooler, less noise)
├── For audio: use metal film or bulk metal foil
```

### Problem 4: Incorrect Value Installed

```
SYMPTOMS:
├── Circuit doesn't work at all
├── Works but wrong (LED too bright, timer wrong)
├── Smoke or overheating

CAUSES:
├── Color code misread
├── Wrong resistor grabbed from bin
├── SMD code misinterpreted
├── Tolerance not considered (5% vs 1%)

DIAGNOSIS:
├── Double-check color code
├── Measure before soldering (always!)
├── Compare to schematic
├── Check with known good circuit

SOLUTIONS:
├── Always measure resistors before use
├── Organize resistors by value (labeled bins)
├── Use magnifier for color code reading
├── For SMD: use meter (codes can be ambiguous)
├── Keep 1% resistors separate from 5% (different appearance)
```

## Summary

1. **Resistors** oppose current flow, converting electrical energy to heat

2. **Key functions:** Current limiting, voltage division, pull-up/pull-down, current sensing, timing

3. **Unit:** Ohm (Ω) = 1V / 1A

4. **Fixed resistor types:**
   - Carbon composition (obsolete, noisy, drifts)
   - Carbon film (general purpose, 5% tolerance)
   - Metal film (precision, 1% tolerance, low noise)
   - Metal oxide (high temperature, flameproof)
   - Wirewound (high power, inductive)
   - SMD chip (modern, tiny, automated assembly)

5. **Variable resistors:**
   - Potentiometer (3-terminal, voltage divider)
   - Rheostat (2-terminal, variable resistance)
   - Trimmer (small, set once)

6. **Color code:** 4-band (digit, digit, multiplier, tolerance) or 5-band (3 digits for precision)

7. **Power rating** must exceed I²R or V²/R (use 2× safety margin)

8. **Series:** R_total = R₁ + R₂ + ... (increases resistance)

9. **Parallel:** 1/R_total = 1/R₁ + 1/R₂ + ... (decreases resistance)

10. **E-series values:** E12 (10%), E24 (5%), E96 (1%) - not every value exists

11. **Measure resistance** with power OFF, preferably out of circuit

12. **Common failures:** Open circuit (burned), drifted value (overheated), noisy (carbon comp)

13. **Higher wattage resistors** are physically larger (dissipate more heat)

14. **SMD resistors** use number codes (e.g., 472 = 4.7kΩ) or EIA-96 system

15. **Always measure before soldering** - color codes can be misread!
