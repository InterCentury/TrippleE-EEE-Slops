# 02.01 - DC Electricity

## What is DC (Direct Current) ?

Direct Current (DC) is the unidirectional flow of electric charge. Unlike alternating current (AC) which periodically reverses direction, DC flows consistently in one direction from the negative terminal to the positive terminal (electron flow) or from positive to negative (conventional current).

The term "direct current" was popularized by Thomas Edison during the early days of electrical power distribution. While AC won the "War of Currents" for grid power, DC remained essential for batteries, electronics, and an increasing number of modern applications including renewable energy and electric vehicles.

## The Nature of DC Flow

### Electron Movement in DC

In a DC circuit, electrons move steadily from the negative terminal (excess electrons) to the positive terminal (deficiency of electrons).

```
DC ELECTRON FLOW (Physical Reality)

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   BATTERY                        COPPER WIRE                │
    │                                                             │
    │   ┌─────┐                        ┌───┐   ┌───┐   ┌───┐      │
    │   │  -  │ ═══════════════════════│ e │═══│ e │═══│ e │═══   │
    │   │     │   Electron flow        └─┬─┘   └─┬─┘   └─┬─┘      │
    │   │  +  │     direction            │       │       │        │
    │   └─────┘     ────────►            ▼       ▼       ▼        │
    │                                                             │
    │   Electron flow: (-) ──────────────────────────────► (+)    │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘


DC CHARACTERISTICS:

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Voltage         Current         Power                     │
    │                                                             │
    │    5V ┼──────      1A ┼──────      5W ┼──────               │
    │        │               │               │                    │
    │        │               │               │                    │
    │        │               │               │                    │
    │        │               │               │                    │

KEY PROPERTIES:

    ├── Polarity is fixed (+ and - never swap)
    ├── Magnitude is constant (ideally)
    ├── Direction is constant (one way only)
    ├── Frequency = 0 Hz
    └── Can be stored in batteries
```

### Conventional Current vs Electron Flow

Two conventions exist for describing DC direction. Both are correct; they just use different reference points.

```
CONVENTIONAL CURRENT vs ELECTRON FLOW

    CIRCUIT:                    BATTERY ──► RESISTOR ──► BATTERY
                                                   
    CONVENTIONAL CURRENT (Benjamin Franklin's assumption):
    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Positive charges move from positive to negative           │
    │                                                             │
    │   (+) ──────────────────────────────────────────────► (-)   │
    │                                                             │
    │   Used in: All circuit diagrams, formulas, textbooks        │
    │   Arrow direction: Points from + to -                       │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘


    ELECTRON FLOW (Physical reality, discovered after Franklin):
    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Electrons move from negative to positive                  │
    │                                                             │
    │   (-) ──────────────────────────────────────────────► (+)   │
    │                                                             │
    │   Used in: Semiconductor physics, vacuum tubes, battery chem│
    │   Arrow direction: Points from - to + (on schematics, rare) │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘


    COMPARISON TABLE:

    Aspect                    Conventional          Electron Flow
    ──────────────────────────────────────────────────────────────
    Charge carrier            Positive (+ holes)    Electrons (-)
    Direction                 + to -                - to +
    Used in diagrams          YES (standard)        Rare
    Ohm's Law                 V = I × R             Same formula
    Diode symbol arrow        Points to -           Actually electron flow
    Transistor diagrams       PNP/NPN symbols       Based on holes/electrons
    Battery internal          Ions move both ways   Electrons move
    Which is "correct"?       Convention only       Physical reality


PRACTICAL ADVICE:

    ├── For circuit analysis: Use conventional current (everyone does)
    ├── For understanding semiconductors: Know electron flow exists
    ├── For battery chemistry: Electrons flow - to + inside
    ├── For troubleshooting: Either works (just be consistent)
    └── Don't let this confuse you! Both give same numerical results.
```

## Ideal vs Real DC

### Ideal DC Source

An ideal DC source maintains constant voltage regardless of load current.

```
IDEAL DC SOURCE

    Symbol:
        ┌─────┐
        │  +  │─── Vout (constant)
        │     │
        │  -  │─── GND
        └─────┘

    Characteristics:
    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   V/I Characteristic:                                       │
    │                                                             │
    │   Voltage                                                   │
    │     │                                                       │
    │  5V ┼───────────────────────                                │
    │     │                      │                                │
    │     │                      │                                │
    │     │                      │                                │
    │    0┼──────────────────────┼────► Current                   │
    │      0                    10A                               │
    │                                                             │
    │   Perfectly flat (no voltage drop with load)                │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘

    Properties:
    ├── Internal resistance = 0 Ω
    ├── Output voltage independent of load
    ├── Can supply infinite current (theoretically)
    ├── No power loss inside source
    └── Does not exist in reality (only approximation)
```

### Real DC Source (with Internal Resistance)

All real DC sources have internal resistance that causes voltage to drop under load.

```
REAL DC SOURCE (Battery Model)

    Circuit model:
        ┌─────────────────────────────────────────────────────────┐
        │                                                         │
        │      ┌───────┐        ┌─────┐                           │
        │      │       │        │     │                           │
        │      │  V    │────┬───┤  R  ├──── Vout                  │
        │      │ ideal │    │   │ int │                           │
        │      │       │    │   │     │                           │
        │      └───────┘    │   └─────┘                           │
        │                   │                                     │
        │                   │   ┌─────┐                           │
        │                   └───┤     │                           │
        │                       │Load │                           │
        │                       │     │                           │
        │                       └─────┘                           │
        │                           │                             │
        │                          GND                            │
        └─────────────────────────────────────────────────────────┘

    Voltage drops as current increases:
    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   V_out = V_ideal - (I × R_internal)                        │
    │                                                             │
    │   Example: 9V battery with 2Ω internal resistance           │
    │                                                             │
    │   No load (0A):     V_out = 9.0V                            │
    │   100mA load:       V_out = 9.0 - (0.1 × 2) = 8.8V          │
    │   500mA load:       V_out = 9.0 - (0.5 × 2) = 8.0V          │
    │   1A load:          V_out = 9.0 - (1 × 2) = 7.0V            │
    │   4.5A load (short): V_out = 0V (all voltage dropped inside)│
    │                                                             │
    └─────────────────────────────────────────────────────────────┘

    V/I Characteristic (real source):
    Voltage
        │
    9V  ┼●
        │  ╲
    8V  ┼    ●
        │      ╲
    7V  ┼        ●
        │          ╲
    6V  ┼            ●
        │              ╲
    5V  ┼                ●
        │                  ╲
    0V  ┼────────────────────●──► Current
        0    0.5   1.0   1.5   2.0A

    Slope = -R_internal (negative because voltage drops)
```

### Power Supplies (AC to DC Converters)

Power supplies convert AC from the wall outlet into regulated DC.

```

LINEAR POWER SUPPLY:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    ├── Simple design
    ├── Fast transient response
    └── No switching noise

    Disadvantages:
    ├── Inefficient (40-60%)
    ├── Heavy (big transformer)
    ├── Large (needs heat sinks)
    └── Only steps down (no step-up)

    Uses: Audio equipment, lab supplies, sensitive analog


SWITCHING POWER SUPPLY (SMPS):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    Disadvantages:
    ├── Noisy (switching ripple)
    ├── Complex design
    ├── Slower transient response
    └── Electromagnetic interference (EMI)

    Uses: Phone chargers, computers, TVs, most electronics
```

### DC Generators (Dynamos)

DC generators produce DC directly using a commutator.

```
DC GENERATOR BASICS

    Basic principle: Rotating coil in magnetic field produces AC
    Commutator converts AC to DC by reversing connections each half-cycle

    Simplified diagram:
    
              N ───────────────────────────────── S
                │                              │
                │     ┌─────────────────┐      │
                │     │   [Coil]        │      │
                │     │                 │      │
                │     │    ┌────┐       │      │
                │     │    │Comm│       │      │
                │     │    └─┬──┘       │      │
                │     │      │          │      │
                │     └──────┼──────────┘      │
                │            │                 │
                └────────────┼─────────────────┘
                             │
                        Brushes (DC out)


    Types of DC generators:

    Separately excited:
    ├── Field powered by separate source
    ├── Voltage can be controlled independently
    └── Used in lab and precision applications

    Shunt wound:
    ├── Field in parallel with armature
    ├── Constant voltage characteristic
    └── Used for battery charging

    Series wound:
    ├── Field in series with armature
    ├── High torque at low speed
    └── Used for cranes, electric trains

    Compound wound:
    ├── Both series and shunt fields
    ├── Best of both types
    └── Used for general industrial


DC GENERATOR vs ALTERNATOR + RECTIFIER:

    Most "DC generators" today are actually AC alternators with rectifiers:
    ├── Alternator produces AC (simpler, no commutator)
    ├── Diode bridge converts to DC
    ├── More reliable, cheaper, more efficient
    └── Used in cars (12V), wind turbines, bicycles
```

### DC Power Distribution

```
DC DISTRIBUTION SYSTEMS

TRADITIONAL (Low voltage, short distance):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    Battery ──► Fuse ──► Wire ──► Load
    
    Limitations:
    ├── Voltage drop limits distance
    ├── Thick wire needed for high current
    └── Best for local distribution (<10 meters)

    Example: 12V, 10A load, 10m wire (round trip 20m)
    Using AWG 14 (2.5mm²): R = 20m × 0.008 Ω/m = 0.16Ω
    Voltage drop = 10A × 0.16Ω = 1.6V (13% loss - too high!)
    Need AWG 8 (8.4mm²): R = 20m × 0.0021 Ω/m = 0.042Ω
    Voltage drop = 10A × 0.042Ω = 0.42V (3.5% loss - acceptable)


MODERN (Higher voltage, longer distance):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    Higher voltage = Lower current = Less voltage drop
    
    48V system, 10A load (same 480W power):
    Voltage drop (AWG 14, 20m): 10A × 0.16Ω = 1.6V (3.3% loss)
    Same wire works fine!

    Benefits of higher voltage DC:
    ├── Longer distance possible
    ├── Thinner wire (saves copper)
    ├── Lower I²R losses
    └── Smaller voltage drop percentage


HVDC (High Voltage DC Transmission):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    For very long distance (hundreds of km):
    ├── ±500kV to ±800kV typical
    ├── Lower losses than AC (no skin effect, no reactive power)
    ├── Can connect asynchronous AC grids
    └── Used for undersea cables, long overland lines

    Example: ±800kV HVDC line, 1000km, 5000MW
    Losses: ~3% per 1000km (AC would be 10-15%)
```

## DC Circuit Analysis

### Series DC Circuits

In a series DC circuit, components are connected end-to-end forming a single path.

```
SERIES DC CIRCUIT

    ┌─────[R1]─────[R2]─────[R3]─────┐
    │    10Ω       20Ω       30Ω     │
    │                                │
    └────────────[12V]───────────────┘


RULES FOR DC SERIES CIRCUITS:

    1. Current is the same everywhere
       I_total = I₁ = I₂ = I₃

    2. Voltage divides across components
       V_total = V₁ + V₂ + V₃

    3. Resistance adds
       R_total = R₁ + R₂ + R₃

    4. Power adds
       P_total = P₁ + P₂ + P₃


CALCULATION EXAMPLE:

    Given: 12V battery, R₁=10Ω, R₂=20Ω, R₃=30Ω

    Step 1: Total resistance
    R_total = 10 + 20 + 30 = 60Ω

    Step 2: Total current
    I = V / R_total = 12V / 60Ω = 0.2A

    Step 3: Voltage across each resistor
    V₁ = I × R₁ = 0.2 × 10 = 2V
    V₂ = I × R₂ = 0.2 × 20 = 4V
    V₃ = I × R₃ = 0.2 × 30 = 6V

    Step 4: Verify
    V_total = 2V + 4V + 6V = 12V ✓


CHARACTERISTICS:

    ├── One path for current
    ├── If one component fails open → all stop working
    ├── Adding more resistors increases total resistance
    ├── Adding more resistors decreases total current
    └── Voltage divider: V_n = V_total × (R_n / R_total)
```

### Parallel DC Circuits

In a parallel DC circuit, components are connected across the same voltage source.

```
PARALLEL DC CIRCUIT

              ┌─────[R1]─────┐
              │     10Ω      │
              │              │
              ├─────[R2]─────┤
              │     20Ω      │
              │              │
              ├─────[R3]─────┤
              │     30Ω      │
              │              │
             (+)            (-)
              │              │
              └─────[12V]────┘


RULES FOR DC PARALLEL CIRCUITS:

    1. Voltage is the same across all branches
       V_total = V₁ = V₂ = V₃

    2. Current divides among branches
       I_total = I₁ + I₂ + I₃

    3. Resistance adds reciprocally
       1/R_total = 1/R₁ + 1/R₂ + 1/R₃

    4. Power adds
       P_total = P₁ + P₂ + P₃


CALCULATION EXAMPLE:

    Given: 12V battery, R₁=10Ω, R₂=20Ω, R₃=30Ω

    Step 1: Total resistance
    1/R_total = 1/10 + 1/20 + 1/30 = 0.1 + 0.05 + 0.0333 = 0.1833
    R_total = 1 / 0.1833 = 5.45Ω

    Step 2: Current through each resistor
    I₁ = V / R₁ = 12 / 10 = 1.2A
    I₂ = V / R₂ = 12 / 20 = 0.6A
    I₃ = V / R₃ = 12 / 30 = 0.4A

    Step 3: Total current
    I_total = 1.2 + 0.6 + 0.4 = 2.2A

    Step 4: Verify with Ohm's Law
    I_total = V / R_total = 12 / 5.45 = 2.2A ✓


CHARACTERISTICS:

    ├── Multiple paths for current
    ├── If one component fails open → others still work
    ├── Adding more resistors decreases total resistance
    ├── Adding more resistors increases total current
    └── Current divider: I_n = I_total × (R_eq / R_n)
```

### Series-Parallel DC Circuits

Most real circuits combine series and parallel elements.

```
SERIES-PARALLEL DC CIRCUIT

                      ┌─────[R2]─────┐
                      │     20Ω      │
              ┌───────┼─────[R3]─────┼───────┐
              │       │     30Ω      │       │
              │       └──────────────┘       │
              │                              │
              │         [R1]                 │
              │         10Ω                  │
              │                              │
            (+)                            (-)
              │                              │
              └──────────[24V]───────────────┘


SOLUTION STRATEGY:

    Step 1: Combine parallel resistors first
    R_parallel (R2 and R3) = (20 × 30) / (20 + 30) = 600 / 50 = 12Ω

    Step 2: Now series combination
    R_total = R1 + R_parallel = 10 + 12 = 22Ω

    Step 3: Total current
    I_total = V / R_total = 24 / 22 = 1.091A

    Step 4: Voltage across R1
    V1 = I_total × R1 = 1.091 × 10 = 10.91V

    Step 5: Voltage across parallel branch
    V_parallel = V_total - V1 = 24 - 10.91 = 13.09V

    Step 6: Current through R2 and R3
    I2 = V_parallel / R2 = 13.09 / 20 = 0.6545A
    I3 = V_parallel / R3 = 13.09 / 30 = 0.4364A


RESULTS:

    Component    R (Ω)    I (A)     V (V)     P (W)
    ────────────────────────────────────────────────
    R1           10       1.091     10.91     11.90
    R2           20       0.6545    13.09     8.57
    R3           30       0.4364    13.09     5.71
    Total        22       1.091     24        26.18
```

## DC Waveforms and Quality

### Types of DC Waveforms

```
DC WAVEFORM CLASSIFICATION

PURE DC (Ideal):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    Voltage
        │
    5V  ┼─────────────────────────► Time
        │
    
    Characteristics:
    ├── Zero ripple
    ├── Zero noise
    ├── Perfectly constant
    └── Only exists in theory/batteries (at rest)


RIPPLED DC (Unfiltered rectifier):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    Voltage
        │    ╱╲    ╱╲
    5V  ┼──╱  ╲──╱  ╲─────────────► Time
        │ ╱    ╲╱    ╲
        │╱              ╲
    
    Characteristics:
    ├── Periodic variation (ripple)
    ├── Frequency = 2× line frequency (100/120 Hz)
    ├── Ripple voltage = peak-to-peak variation
    └── Reduced with larger filter capacitors


NOISY DC (Switching supply):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    Voltage
        │ ┌─┐  ┌─┐
    5V  ┼─┘ └──┘ └────────────────► Time
        │   (high frequency spikes)
    
    Characteristics:
    ├── High frequency noise (kHz to MHz)
    ├── Caused by switching MOSFETs
    ├── Requires filtering for sensitive circuits
    └── Measured as ripple + noise


REGULATED DC (Linear regulator):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    Voltage
        │
    5V  ┼─────────────────────────► Time
        │   (very flat, clean)
    
    Characteristics:
    ├── Very low ripple (μV to mV)
    ├── Low noise
    ├── Used for sensitive analog and audio
    └── Inefficient (drops excess voltage as heat)


DC WITH TRANSIENTS (Load changes):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    Voltage
        │
    5V  ┼──────────┬───────────────► Time
        │          │
    4.8V┼          └───
        │      (voltage dip when load turns on)
    
    Characteristics:
    ├── Temporary voltage drop (load step)
    ├── Caused by source impedance
    ├── Recovered after transient
    └── Reduced with low ESR capacitors
```

### Measuring DC Quality

```
DC QUALITY METRICS

RIPPLE VOLTAGE:
    V_ripple_pp = V_max - V_min (peak-to-peak)
    V_ripple_rms = V_ripple_pp / (2√3) for triangle ripple
    
    Acceptable levels:
    ├── Battery: <1 mV
    ├── Linear supply: <5 mV
    ├── Switching supply: <50 mV
    ├── Computer PSU: <120 mV
    └── Phone charger: <200 mV


LOAD REGULATION:
    Load Regulation = (V_no_load - V_full_load) / V_full_load × 100%
    
    Example: 5.00V no load, 4.90V at 2A
    Load regulation = (5.00 - 4.90) / 4.90 × 100% = 2.04%


LINE REGULATION:
    Line Regulation = ΔV_out / ΔV_in × 100%
    
    Example: 5.00V at 100V AC in, 5.01V at 240V AC in
    Line regulation = (0.01 / 5.00) × 100% = 0.2%


TRANSIENT RESPONSE:
    ├── Settling time: How fast voltage recovers after load change
    ├── Overshoot: Voltage above nominal during recovery
    ├── Undershoot: Voltage below nominal during dip
    └── Good supplies: <50µs settling, <5% overshoot


MEASUREMENT TOOLS:

    Multimeter (DCV):
    ├── Shows average DC voltage
    ├── Hides ripple and noise
    └── Good for steady-state only

    Oscilloscope (AC coupled):
    ├── Shows ripple and noise
    ├── Can measure peak-to-peak
    └── Best for quality assessment

    AC voltmeter (True RMS):
    ├── Measures ripple voltage (AC component)
    ├── Set to AC mode while measuring DC
    └── Subtract from DC reading for clean DC value
```

## Common DC Problems and Solutions

### Problem 1: Voltage Sag Under Load

```
SYMPTOM:
    Device works fine on battery, but fails when connected to supply
    Voltage drops significantly when load turns on

DIAGNOSIS:
    Measure voltage with no load, then with full load
    
    No load:    12.0V
    Full load:  10.5V (1.5V drop)
    
    I_load = 5A
    R_source = V_drop / I = 1.5V / 5A = 0.3Ω (high!)

CAUSES:
    ├── Weak battery (high internal resistance)
    ├── Undersized power supply
    ├── Long thin wires
    ├── Poor connections or corroded terminals
    └── Loose crimps or solder joints

SOLUTIONS:
    ├── Replace battery
    ├── Use larger power supply
    ├── Thicker wire (lower resistance)
    ├── Clean connections
    └── Move supply closer to load
```

### Problem 2: Excessive Ripple

```
SYMPTOM:
    Audio equipment hums (100/120 Hz)
    Digital circuits behave erratically
    Oscilloscope shows sawtooth wave on DC

DIAGNOSIS:
    Measure AC voltage across DC output (multimeter in AC mode)
    
    Good supply: <10 mV AC
    Poor supply: >100 mV AC

CAUSES:
    ├── Filter capacitor too small
    ├── Dried out electrolytic capacitor (high ESR)
    ├── Rectifier diode failed (half-wave instead of full)
    ├── Excessive load current
    └── Poor transformer regulation

SOLUTIONS:
    ├── Increase filter capacitance (C = I_load / (f_ripple × V_ripple))
    ├── Replace aging capacitors
    ├── Check diode bridge with multimeter (diode mode)
    ├── Reduce load or add second supply
    └── Add post-regulator (7805, LM317, etc.)

CALCULATION:
    Required capacitance for given ripple:
    C = I_load / (f_ripple × V_ripple)
    
    Example: I=2A, f=120Hz, V_ripple=0.1V
    C = 2 / (120 × 0.1) = 2 / 12 = 0.1667F = 166,700µF
```

### Problem 3: Reverse Polarity

```
SYMPTOM:
    Device doesn't work
    Smoke or burning smell (if no protection)
    Multimeter shows negative voltage

DIAGNOSIS:
    Check voltage with multimeter (red to +, black to -)
    If reading negative, polarity is reversed

CAUSES:
    ├── Battery installed backwards
    ├── Power supply connected backwards
    ├── Wrong adapter (center positive vs center negative)
    └── Wire color coding incorrect

SOLUTIONS (Prevention):
    ├── Reverse polarity protection diode (series):
        ┌─────[Diode]─────┐
        │                 │
       (+)               Load
        │                 │
        └─────────────────┘
        (Diode conducts only when polarity correct)
    
    ├── Reverse polarity protection diode (shunt + fuse):
        ┌─────────────────┐
        │                 │
       (+)               Load
        │      ┌─────┐    │
        └──────┤Diode├────┘
               └─────┘
        (Reverse polarity shorts supply, blows fuse)
    
    ├── MOSFET reverse protection (low voltage drop)
    ├── Polarized connectors (only fit one way)
    └── Color coding (red = positive, black = negative)

SOLUTIONS (After damage):
    ├── Replace burned components
    ├── Check all ICs (often damaged by reverse voltage)
    ├── Add protection before powering again
    └── Always double-check polarity before applying power
```

### Problem 4: Ground Loops

```
SYMPTOM:
    Hum in audio systems
    Erratic readings in measurement equipment
    Unexpected current paths

DIAGNOSIS:
    Measure voltage between "ground" points
    Should be 0V, but might be mV or even volts!

DIAGRAM (Ground loop):
    
    Supply A ──┬── Load 1 ──┬── Supply B
               │            │
               │            │
               └── Load 2 ──┘
    
    Current flows in ground loop (unintended path)

CAUSES:
    ├── Multiple ground connections
    ├── Different ground potentials
    ├── Long ground wires
    └── Shared return paths

SOLUTIONS:
    ├── Star grounding (all grounds meet at one point)
    ├── Isolated supplies (no common ground)
    ├── Differential signaling (balanced audio)
    ├── Ground lift switches (isolate shield)
    └── Single-point grounding (best practice)

STAR GROUND (Correct):
    
    All loads ──┬── Common ground point ──┬── Supply
                │                         │
                └─────────────────────────┘
    
    No loops! All grounds meet at one physical point.
```

## Summary

1. **Direct Current (DC)** flows in one direction with constant polarity

2. **Two conventions:** Conventional current (+ to -) and electron flow (- to +)

3. **Ideal DC** has zero internal resistance, constant voltage, infinite current capability

4. **Real DC sources** have internal resistance causing voltage drop under load (V_out = V_ideal - I×R_int)

5. **DC sources include:** Batteries (primary/secondary), power supplies (linear/switching), solar cells, DC generators

6. **Battery chemistries:** Alkaline (1.5V), Li-ion (3.7V), Lead-acid (2.1V/cell), NiMH (1.2V)

7. **DC voltage standards:** 5V (USB), 12V (automotive), 24V (industrial), 48V (telecom), 400V (EV/solar)

8. **Series circuits:** I constant, V divides, R_total = ΣR_n

9. **Parallel circuits:** V constant, I divides, 1/R_total = Σ(1/R_n)

10. **DC quality metrics:** Ripple, load regulation, line regulation, transient response

11. **Common problems:** Voltage sag, excessive ripple, reverse polarity, ground loops

12. **Higher voltage DC** enables longer distance transmission with lower losses (HVDC for hundreds of km)
