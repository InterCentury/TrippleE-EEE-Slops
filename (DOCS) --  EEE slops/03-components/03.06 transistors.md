# 03.06 - Transistors

## What is a Transistor?

A transistor is a three-terminal semiconductor device that can amplify or switch electronic signals. It is the fundamental building block of all modern electronics, from microprocessors to power amplifiers. Transistors can act as electronically controlled switches or variable resistors, making them essential for digital logic, amplification, and regulation.

The transistor was invented in 1947 at Bell Labs by John Bardeen, Walter Brattain, and William Shockley. This invention revolutionized electronics, replacing bulky vacuum tubes and enabling the miniaturization that led to integrated circuits, microprocessors, and the digital age. The three inventors received the Nobel Prize in Physics in 1956.

## Basic Transistor Types

## BJT Operating Regions

### Cutoff Region

Transistor is OFF (no collector current).

```
CUTOFF REGION

    Conditions:
    ├── V_BE < 0.6V (NPN) or V_EB < 0.6V (PNP)
    ├── Base-emitter junction NOT forward biased
    ├── Base current I_B = 0 (or negligible)
    └── Collector current I_C = 0 (or leakage only, nA-µA)


    Circuit behavior:
    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Vcc ──┬────[Rc]────┬─── Vout (high = Vcc)                 │
    │         │            │                                      │
    │         │          ┌─┴─┐                                    │
    │         │          │   │                                    │
    │         │          │ NPN                                    │
    │         │          │   │                                    │
    │         │          └─┬─┘                                    │
    │         │            │                                      │
    │         │            │                                      │
    │         │            └──────────────┬─────────────────────  │
    │         │                           │                       │
    │         │    ┌────[Rb]────┬─── V_in (low, <0.6V)            │
    │         │    │            │                                 │
    │         │    │            │                                 │
    │         │    │            └───────────────────────────────  │
    │         │    │                                              │
    │   GND ──┴────┴────────────────────────────────────────────  │
    │                                                             │
    │   V_in < 0.6V → Transistor OFF → V_out = Vcc                │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘

    Use: Logic 0 output (open switch)
```

### Active (Linear) Region

Transistor amplifies; collector current proportional to base current.

```
ACTIVE REGION

    Conditions:
    ├── V_BE ≈ 0.6V-0.7V (NPN)
    ├── Base-emitter junction forward biased
    ├── Base-collector junction reverse biased
    ├── I_C = β × I_B (linear relationship)
    └── V_CE > V_CE(sat) (typically >0.3V)


    CURRENT RELATIONSHIPS:

    I_E = I_B + I_C
    I_C = β × I_B
    I_C = α × I_E
    β = α / (1 - α)
    
    WHERE:
    β (beta) = DC current gain (h_FE), typical 50-500
    α (alpha) = common-base gain, typical 0.95-0.99


    TRANSFER CHARACTERISTIC:

    I_C (mA)
        │
     50 ┼                                           ●
        │                                          ╱
     40 ┼                                        ●
        │                                       ╱
     30 ┼                                     ●
        │                                    ╱
     20 ┼                                  ●
        │                                 ╱
     10 ┼                               ●
        │                              ╱
      0 ┼─────────────────────────────●───► I_B (µA)
        0   100  200  300  400  500  600
    
        I_C = β × I_B (straight line through origin)


    EXAMPLE (β = 100):

    I_B = 10µA → I_C = 100 × 10µA = 1mA
    I_B = 50µA → I_C = 100 × 50µA = 5mA
    I_B = 100µA → I_C = 100 × 100µA = 10mA


    Use: Linear amplifiers (class A, AB, B)
```

### Saturation Region

Transistor is fully ON (switch closed).

```
SATURATION REGION

    Conditions:
    ├── V_BE > 0.6V (NPN)
    ├── Both junctions forward biased
    ├── I_C = I_C(sat) (maximum, limited by external circuit)
    ├── V_CE(sat) ≈ 0.2V (small transistor) to 1.0V (power transistor)
    └── β_forced = I_C(sat) / I_B (much less than β)


    CIRCUIT BEHAVIOR:

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Vcc ──┬────[Rc]────┬─── Vout (low, ≈0.2V)                 │
    │         │            │                                      │
    │         │          ┌─┴─┐                                    │
    │         │          │   │                                    │
    │         │          │ NPN                                    │
    │         │          │   │                                    │
    │         │          └─┬─┘                                    │
    │         │            │                                      │
    │         │            │                                      │
    │         │            └──────────────┬─────────────────────  │
    │         │                           │                       │
    │         │    ┌────[Rb]────┬─── V_in (high, >0.6V)           │
    │         │    │            │                                 │
    │         │    │            │                                 │
    │         │    │            └───────────────────────────────  │
    │         │    │                                              │
    │   GND ──┴────┴────────────────────────────────────────────  │
    │                                                             │
    │   V_in > 0.6V with enough I_B → Transistor SATURATED        │
    │   V_out = V_CE(sat) ≈ 0.2V (near GND)                       │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘

    SATURATION CONDITION:

    I_B > I_C(sat) / β_min
    
    Example: Rc = 1kΩ, Vcc = 10V
    I_C(sat) = Vcc / Rc = 10V / 1kΩ = 10mA
    
    If β_min = 100:
    I_B(required) > 10mA / 100 = 0.1mA = 100µA
    
    Use I_B = 200µA (2× margin for guaranteed saturation)


    Use: Logic 1 output (closed switch), LED driver, relay driver
```

## BJT as a Switch

The most common use of BJTs in digital circuits.

```
NPN TRANSISTOR SWITCH

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Vcc (+5V)                                                 │
    │      │                                                      │
    │      └────[Rc]────┬─── Output to load (e.g., LED, relay)    │
    │                   │                                         │
    │                   │                                         │
    │                 ┌─┴─┐                                       │
    │                 │   │                                       │
    │             ┌───┤NPN│                                       │
    │             │   │   │                                       │
    │             │   └─┬─┘                                       │
    │             │     │                                         │
    │    ┌───┐    │     │                                         │
    │    │   │    │     │                                         │
    │    │ Rb│    │     │                                         │
    │    │   │    │     │                                         │
    │    └─┬─┘    │     │                                         │
    │      │      │     │                                         │
    │   Control   │     │                                         │
    │   Input     │     │                                         │
    │   (0V or    │     │                                         │
    │    5V)      │     │                                         │
    │      │      │     │                                         │
    │      └──────┘     │                                         │
    │                   │                                         │
    │   GND ────────────┴──────────────────────────────────────── │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘


OPERATION:

    Control Input    Transistor    Output (Collector)
    ──────────────────────────────────────────────────────────────
    0V (logic 0)     OFF (cutoff)  Vcc (logic 1, LED off)
    5V (logic 1)     ON (saturation) V_CE(sat) ≈ 0.2V (logic 0, LED on)


DESIGN EXAMPLE (LED Driver):

    Vcc = 5V, LED V_F = 2.0V, I_LED = 20mA
    Control = 5V logic (microcontroller output)
    β_min = 100

    Step 1: Calculate Rc
    V_Rc = Vcc - V_LED - V_CE(sat)
    V_Rc = 5V - 2.0V - 0.2V = 2.8V
    Rc = V_Rc / I_LED = 2.8V / 0.02A = 140Ω (use 150Ω)

    Step 2: Calculate required I_B
    I_C(sat) = I_LED = 20mA
    I_B(min) = I_C(sat) / β_min = 20mA / 100 = 0.2mA = 200µA
    Use I_B = 1mA (5× margin for guaranteed saturation)

    Step 3: Calculate Rb
    V_Rb = V_control - V_BE(sat)
    V_Rb = 5V - 0.7V = 4.3V
    Rb = V_Rb / I_B = 4.3V / 0.001A = 4.3kΩ (use 4.7kΩ or 3.9kΩ)

    Step 4: Verify
    I_B = 4.3V / 4.7kΩ = 0.915mA
    I_C(max) = β × I_B = 100 × 0.915mA = 91.5mA (much >20mA, saturated)
    LED current limited by Rc to 20mA


PNP TRANSISTOR SWITCH (High-side switch):

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Vcc (+5V) ──┬────[Rc]────┬─── Output to load (e.g., LED)  │
    │               │            │                                │
    │               │          ┌─┴─┐                              │
    │               │          │   │                              │
    │               │          │PNP│                              │
    │               │          │   │                              │
    │               │          └─┬─┘                              │
    │               │            │                                │
    │               │    ┌───┐   │                                │
    │               │    │   │   │                                │
    │               │    │ Rb│   │                                │
    │               │    │   │   │                                │
    │               │    └─┬─┘   │                                │
    │               │      │     │                                │
    │               │      └─────┼─────────── Control Input       │
    │               │            │           (0V to turn ON)      │
    │   GND ────────┴────────────┴─────────────────────────────── │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘

    Operation:
    ├── Control = 0V → Transistor ON (load powered)
    ├── Control = 5V → Transistor OFF (load off)
    └── Use when load must be connected to ground (common ground)
```

## BJT as an Amplifier

Transistors can amplify small signals.

```
COMMON EMITTER AMPLIFIER

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Vcc (+12V)                                                │
    │      │                                                      │
    │      ├────[Rc]────┬─── Vout                                 │
    │      │            │                                         │
    │      │            │                                         │
    │      │          ┌─┴─┐                                       │
    │      │          │   │                                       │
    │      ├────[Rb1]─┤NPN│                                       │
    │      │          │   │                                       │
    │      │          └─┬─┘                                       │
    │      │            │                                         │
    │      │            ├────[Re]────┬─── GND                     │
    │      │            │            │                            │
    │      ├────[Rb2]───┘           ┌┴┐                           │
    │      │                        │ │ Ce (bypass)               │
    │      │                        └┬┘                           │
    │      │                         │                            │
    │  GND ┴─────────────────────────┴──────────────────────────  │
    │                                                             │
    │   Vin ──┬────[C1]────┬─── Base (coupling)                   │
    │         │            │                                      │
    │         │            │                                      │
    │   GND ──┴────────────┴───────────────────────────────────── │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘

    Voltage gain: Av ≈ -Rc / Re (with bypass cap, Av ≈ -Rc / re)

    Example: Rc = 4.7kΩ, Re = 470Ω (bypassed)
    re = 25mV / I_C ≈ 25Ω (for I_C ≈ 1mA)
    Av ≈ -4700 / 25 = -188 (inverting amplifier!)


GAIN CALCULATION:

    Without bypass capacitor (Re not bypassed):
    Av = -Rc / Re
    
    With bypass capacitor (Re bypassed for AC):
    Av = -Rc / re
    re = 25mV / I_C (at room temperature)
    
    Example: I_C = 1mA → re = 25Ω
    Rc = 4.7kΩ → Av = -4700 / 25 = -188 (very high gain!)


INPUT IMPEDANCE:

    Zin = Rb1 ∥ Rb2 ∥ (β × re)
    
    Example: Rb1=100kΩ, Rb2=22kΩ, β=100, re=25Ω
    β×re = 100×25 = 2.5kΩ
    Zin = 100k ∥ 22k ∥ 2.5k ≈ 2.2kΩ
```

## MOSFET Operating Regions

### Cutoff Region

MOSFET is OFF (no drain current).

```
CUTOFF REGION (N-Channel Enhancement)

    Conditions:
    ├── V_GS < V_GS(th) (threshold voltage, typically 1-4V)
    ├── No channel formed between source and drain
    ├── I_D = 0 (or leakage only, nA)
    └── Transistor acts as open circuit


    V_GS(th) examples:
    ├── Logic level MOSFET: 1-2.5V
    ├── Standard MOSFET: 2-4V
    └── High voltage MOSFET: 3-5V


    Use: Switch OFF, logic 0
```

### Triode (Ohmic) Region

MOSFET acts as a voltage-controlled resistor.

```
TRIODE REGION

    Conditions:
    ├── V_GS > V_GS(th)
    ├── V_DS < V_GS - V_GS(th)
    ├── Channel fully formed but not pinched off
    ├── I_D = K × [(V_GS - V_GS(th)) × V_DS - V_DS²/2]
    └── For small V_DS: I_D ≈ K × (V_GS - V_GS(th)) × V_DS (linear)


    R_DS(on) = V_DS / I_D (depends on V_GS)
    
    Example: Logic level MOSFET, V_GS=5V
    R_DS(on) ≈ 0.05Ω (very low resistance!)


    Use: Switch ON (saturation), voltage-controlled resistor
```

### Saturation (Active) Region

MOSFET acts as a current source (amplifier).

```
SATURATION REGION

    Conditions:
    ├── V_GS > V_GS(th)
    ├── V_DS ≥ V_GS - V_GS(th)
    ├── Channel pinched off at drain end
    ├── I_D = (K/2) × (V_GS - V_GS(th))²
    └── I_D independent of V_DS (ideal current source)


    TRANSFER CHARACTERISTIC:

    I_D (mA)
        │
     50 ┼                                           ●
        │                                          ╱
     40 ┼                                        ●
        │                                       ╱
     30 ┼                                     ●
        │                                    ╱
     20 ┼                                  ●
        │                                 ╱
     10 ┼                               ●
        │                              ╱
      0 ┼─────────────────────────────●───► V_GS (V)
        2    3    4    5    6    7    8
    
        I_D = (K/2) × (V_GS - V_GS(th))² (parabolic)


    TRANSCONDUCTANCE (gm):

    gm = ΔI_D / ΔV_GS = K × (V_GS - V_GS(th))
    
    Example: K = 0.5 A/V², V_GS - V_GS(th) = 2V
    gm = 0.5 × 2 = 1 S (siemens) = 1000 mS
    (1 siemens = 1 ampere per volt)


    Use: Linear amplifiers
```

## MOSFET as a Switch

MOSFETs are superior to BJTs for switching due to very low R_DS(on).

```
             Vcc (+12V)
                         |
                         |
                    [ LOAD/MOTOR ]
                         |
                         | D (Drain)
                       __█__
             Control   | █ | (N-Channel MOSFET)
             Input >---| █ |
             (0V/5V)   | █ |
                       ¯¯█¯¯
                         | S (Source)
                         |
                        GND
             
    Operation:
    ├── Control = 0V → MOSFET OFF (load off)
    ├── Control = 5V → MOSFET ON (load powered)
    └── No base current needed! (gate draws ~0 current)


DESIGN EXAMPLE (LED Driver with MOSFET):

    Vcc = 12V, LED V_F = 3.0V, I_LED = 100mA
    Control = 5V (microcontroller)
    MOSFET: Logic level, R_DS(on) = 0.2Ω at V_GS=5V

    Step 1: Calculate resistor (if needed)
    V_R = Vcc - V_LED - V_DS(on)
    V_R = 12V - 3.0V - (0.1A × 0.2Ω) = 12 - 3.0 - 0.02 = 8.98V
    R = V_R / I_LED = 8.98V / 0.1A = 89.8Ω (use 100Ω)
    
    (MOSFET alone cannot limit current! Use resistor or constant current)


    Power in MOSFET: P = I_D² × R_DS(on) = 0.1² × 0.2 = 0.002W (tiny!)


ADVANTAGES OVER BJT:

    ├── No base current (gate draws ~0, easy to drive from logic)
    ├── Very low voltage drop (R_DS(on) can be <0.01Ω)
    ├── Higher efficiency (less heat)
    ├── Faster switching (no minority carrier storage)
    ├── Can be paralleled easily (positive temp coefficient)
```

## Important BJT Parameters

```
DC CURRENT GAIN (h_FE or β)

    β = I_C / I_B
    
    Typical values:
    ├── Small signal (2N3904, 2N2222): 100-300
    ├── Power (TIP31, TIP41): 20-100
    ├── Darlington (TIP120): 1000+ (two transistors inside)
    └── High voltage (MJE13007): 10-40

    β varies with:
    ├── Collector current (highest at mid-range)
    ├── Temperature (increases with temperature)
    ├── Manufacturing (wide variation between parts)
    └── Voltage (slight variation)


COLLECTOR-EMITTER SATURATION VOLTAGE (V_CE(sat))

    V_CE(sat) when transistor is fully ON
    
    Typical values:
    ├── Small signal: 0.2V - 0.3V at low current
    ├── Power: 0.5V - 1.5V at high current
    ├── Darlington: 1.5V - 2.5V (two V_BE drops)
    └── Lower V_CE(sat) = less power loss = better


COLLECTOR CURRENT (I_C)

    Maximum continuous collector current
    
    Examples:
    ├── 2N3904: 200mA
    ├── 2N2222: 600mA
    ├── BC547: 100mA
    ├── TIP31: 3A
    ├── TIP41: 6A
    └── 2N3055: 15A


POWER DISSIPATION (P_D)

    P_D = V_CE × I_C (plus base power, negligible)
    
    Maximum power (without heatsink):
    ├── TO-92 (2N3904): 0.625W
    ├── TO-220 (TIP31): 2W (without heatsink), 40W (with heatsink)
    └── TO-3 (2N3055): 3W (without), 115W (with)

    Rule: Keep P_D below rating (derate at high temperature)
```

## Important MOSFET Parameters

```
THRESHOLD VOLTAGE (V_GS(th))

    Gate voltage where MOSFET begins to conduct
    
    Typical values:
    ├── Logic level: 1V - 2.5V (use with 3.3V or 5V logic)
    ├── Standard: 2V - 4V (needs 5V-10V gate drive)
    ├── High voltage: 3V - 5V
    └── Depletion mode: negative (conducts at V_GS=0)


DRAIN-SOURCE ON RESISTANCE (R_DS(on))

    Resistance when fully enhanced (V_GS high)
    
    Typical values:
    ├── Small signal (2N7000): 5Ω
    ├── Logic level (IRLZ44N): 0.022Ω (22mΩ)
    ├── Standard (IRFZ44N): 0.0175Ω (17.5mΩ)
    ├── High voltage (IRF740): 0.55Ω
    └── Very low R_DS(on) (IRLS3034): 0.0017Ω (1.7mΩ)!
    
    R_DS(on) increases with temperature (positive temp coefficient)
    → MOSFETs parallel easily (current sharing)


DRAIN CURRENT (I_D)

    Maximum continuous drain current
    
    Examples:
    ├── 2N7000: 200mA
    ├── BS170: 500mA
    ├── IRLZ44N: 47A
    ├── IRFZ44N: 49A
    └── IRLS3034: 343A!


GATE CHARGE (Q_g)

    Charge required to turn MOSFET on/off
    
    Typical values:
    ├── Small signal: 1-5 nC
    ├── Power MOSFET: 20-100 nC
    ├── Large MOSFET: 100-300 nC
    └── Affects switching speed (higher Q_g = slower switching)


GATE THRESHOLD VOLTAGE vs TEMPERATURE:

    V_GS(th) decreases with temperature (negative temp coefficient)
    But R_DS(on) increases with temperature (positive temp coefficient)
    Overall: MOSFETs are thermally stable (unlike BJTs)
```

## Biasing and Thermal Stability

### BJT Biasing

Proper biasing ensures the transistor operates in the desired region.

```
VOLTAGE DIVIDER BIAS (Most stable)

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Vcc                                                      │
    │    │                                                       │
    │    ├────[R1]────┬─── V_B                                   │
    │    │            │                                          │
    │    │            │                                          │
    │    │          ┌─┴─┐                                        │
    │    │          │   │                                        │
    │    ├────[R2]──┤ NPN│                                        │
    │    │          │   │                                        │
    │    │          └─┬─┘                                        │
    │    │            │                                          │
    │    │            ├────[Re]────┬─── GND                      │
    │    │            │            │                             │
    │    │            │            │                             │
    │    │            │            │                             │
    │    │            └────────────┘                             │
    │    │                                                       │
    │   GND                                                      │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘

    Design equations:
    V_B = Vcc × R2 / (R1 + R2)
    V_E = V_B - V_BE (≈ V_B - 0.7V)
    I_E = V_E / Re
    I_C ≈ I_E
    V_C = Vcc - I_C × Rc
    V_CE = V_C - V_E


    RULE OF THUMB (Good stability):
    ├── I_R1,R2 ≈ 10 × I_B
    ├── V_E ≈ 1V to 2V (for temperature stability)
    └── Re provides negative feedback (stabilizes I_C)


    TEMPERATURE STABILITY:

    Without Re: I_C increases with temperature (thermal runaway!)
    With Re:   I_C increase → V_E increase → V_BE decrease → I_B decrease
               → I_C decreases (negative feedback, stable!)
```

### Thermal Runaway (BJT)

BJTs have a positive temperature coefficient → can destroy themselves.

```
THERMAL RUNAWAY EXPLANATION

    Temperature ↑ → β ↑ and V_BE ↓
                     ↓
    I_C increases (for fixed I_B or V_BE)
                     ↓
    Power dissipation ↑ (P = V_CE × I_C)
                     ↓
    Temperature ↑ (more heating)
                     ↓
    (cycle repeats until transistor destroys itself!)


PREVENTION:

    ├── Use emitter resistor (Re) for negative feedback
    ├── Derate power at high temperature
    ├── Use adequate heatsinking
    ├── Use MOSFETs (positive temp coefficient of R_DS(on))
    └── Parallel BJTs only with emitter resistors (current sharing)


EMITTER RESISTOR CALCULATION:

    Choose V_E ≈ 1V to 2V (compromise between stability and efficiency)
    Re = V_E / I_C
    
    Example: I_C = 100mA, choose V_E = 1V
    Re = 1V / 0.1A = 10Ω
    
    Power in Re: P = I_C² × Re = 0.01 × 10 = 0.1W (use 1/4W)
```

## Transistor Testing

### Testing BJT with Multimeter

```
BJT TESTING (Diode mode)

    NPN TRANSISTOR:

    Base to Emitter (forward):    0.6V - 0.7V (diode mode)
    Base to Emitter (reverse):    OL (open)
    Base to Collector (forward):  0.6V - 0.7V (diode mode)
    Base to Collector (reverse):  OL (open)
    Collector to Emitter:         OL (both directions, unless breakdown)


    PNP TRANSISTOR:

    Base to Emitter (forward):    0.6V - 0.7V (but red on base, black on emitter)
    (polarity opposite of NPN)


    GOOD TRANSISTOR READINGS:

    Measurement                    NPN (red on base)    NPN (black on base)
    ────────────────────────────────────────────────────────────────────
    Base-Emitter forward           0.6-0.7V            OL
    Base-Emitter reverse           OL                   0.6-0.7V
    Base-Collector forward         0.6-0.7V            OL
    Base-Collector reverse         OL                   0.6-0.7V
    Collector-Emitter              OL                   OL


    FAILURE MODES:

    Shorted (any junction 0V):     Replace transistor
    Open (any junction OL):        Replace transistor
    Leaky (reverse reads low):     Replace transistor
    Low gain (β test fails):       Replace transistor


    β (h_FE) TEST (if meter has transistor socket):

    Insert transistor in correct socket (EBC or ECB)
    Meter displays β (current gain)
    Compare to datasheet (typical 100-300 for small signal)
    If β too low (<20), transistor may be damaged or wrong type
```

### Testing MOSFET with Multimeter

```
MOSFET TESTING (Diode mode)

    N-CHANNEL ENHANCEMENT MOSFET:

    Gate to Source:               OL (open, insulated)
    Gate to Drain:                OL (open, insulated)
    Drain to Source (forward):    0.4V - 0.7V (body diode)
    Drain to Source (reverse):    OL (unless body diode conducts)


    TESTING PROCEDURE:

    Step 1: Measure body diode (Drain to Source)
    ├── Red on Source, Black on Drain → OL (should be open)
    ├── Red on Drain, Black on Source → 0.4V-0.7V (body diode drop)
    └── If shorted (0V) or open (OL both ways) → MOSFET bad

    Step 2: Check Gate insulation
    ├── Gate to Source: should be OL (infinite resistance)
    ├── Gate to Drain: should be OL
    └── If low resistance → gate oxide damaged (ESD) → replace

    Step 3: Turn on MOSFET (test switching)
    ├── Connect 10kΩ resistor from Gate to Drain (or Gate to Source)
    ├── Apply 5V-12V between Gate and Source
    ├── Measure Drain-Source resistance (should be very low, <1Ω)
    └── If still high → MOSFET not turning on (damaged)


    FAILURE MODES:

    Gate-Source short:            ESD damage (replace)
    Drain-Source short:           Overcurrent/overvoltage (replace)
    Drain-Source open:            Overcurrent (bond wire melted)
    High R_DS(on):                Partial damage (replace)
```

## Common Transistor Problems

### Problem 1: Overheating (BJT)

```
SYMPTOMS:
├── Transistor too hot to touch
├── Case discolored (brown/black)
├── Circuit malfunctions (gain drops at high temp)
├── Thermal runaway (current increases with temperature)

CAUSES:
├── Excessive collector current (load too heavy)
├── Insufficient heatsinking
├── Poor thermal contact (no thermal paste)
├── V_CE too high (wrong operating point)
├── Oscillation (unintended, causes extra dissipation)

DIAGNOSIS:
├── Measure I_C (should be ≤ rating)
├── Measure V_CE (should be within safe operating area)
├── Check heatsink temperature (should be warm, not hot)
├── Thermal imaging (find hot spots)
├── Oscilloscope check for oscillation (RF, >1MHz)

SOLUTIONS:
├── Add heatsink (or larger heatsink)
├── Use thermal paste between transistor and heatsink
├── Reduce I_C (use larger transistor or parallel devices)
├── Improve ventilation (fan, larger enclosure)
├── Add emitter resistor (for thermal stability)
├── Check biasing (V_CE may be too high)
├── Add base-stopper resistor (prevents oscillation)
```

### Problem 2: Transistor Won't Turn Off (BJT)

```
SYMPTOMS:
├── Output stays high (or low) regardless of input
├── Leakage current when should be off
├── Circuit draws current even in "off" state

CAUSES:
├── Base resistor too small (excess base current)
├── Base drive not pulled to 0V (floating base)
├── Collector-emitter leakage (damaged transistor)
├── Saturation too deep (excess stored charge)
├── Temperature too high (increases leakage)

DIAGNOSIS:
├── Measure V_BE when input is low (should be <0.5V)
├── Measure I_B when input is low (should be near 0)
├── Measure I_CEO (collector leakage with base open)
├── Test transistor out of circuit

SOLUTIONS:
├── Add pull-down resistor from base to emitter (1kΩ-10kΩ)
├── Reduce base resistor (calculate correct value)
├── Add Baker clamp (Schottky diode from collector to base)
├── Replace transistor if leaky
├── For fast switching: use speed-up capacitor across Rb
├── Use MOSFET instead (no leakage issues)
```

### Problem 3: MOSFET Won't Turn On

```
SYMPTOMS:
├── Load stays off even when gate driven high
├── V_DS remains high (near Vcc)
├── No current through MOSFET

CAUSES:
├── Gate voltage too low (< V_GS(th))
├── Wrong MOSFET type (enhancement vs depletion)
├── Gate resistor too large (slow charging, but should eventually turn on)
├── Gate driver insufficient current (for large MOSFETs)
├── MOSFET damaged (gate-source short or open)

DIAGNOSIS:
├── Measure V_GS (should be > V_GS(th) by 2-3V)
├── For 5V logic: use logic-level MOSFET (V_GS(th) <2.5V)
├── Check gate drive waveform (oscilloscope)
├── Test MOSFET out of circuit

SOLUTIONS:
├── Increase gate voltage (use gate driver IC)
├── Use logic-level MOSFET for 3.3V/5V logic
├── Reduce gate resistor (10Ω-100Ω for switching)
├── Add gate driver (TC4420, MIC4422, etc.)
├── For high current: use bootstrap circuit for high-side
├── Replace damaged MOSFET
```

### Problem 4: Oscillation (Parasitic)

```
SYMPTOMS:
├── Output waveform has high-frequency ringing (>1MHz)
├── Transistor gets hot even with light load
├── Circuit works but inefficient
├── Radio interference (EMI)

CAUSES:
├── Long leads on base/gate (acts as antenna)
├── Poor layout (feedback from output to input)
├── Missing base/gate stopper resistor
├── Capacitive load on output
├── Inductive load without snubber

DIAGNOSIS:
├── Oscilloscope probe at base/gate (use short ground lead)
├── Look for high-frequency oscillation (10-100MHz typical)
├── Vary load (oscillation may change frequency)
├── Touch base/gate with finger (oscillation may stop)

SOLUTIONS:
├── Add base stopper resistor (100Ω-1kΩ in series with base/gate)
├── Keep leads short (especially base/gate)
├── Add ferrite bead on base/gate lead
├── Add snubber (RC) across collector/drain
├── Improve layout (star ground, short traces)
├── Add small capacitor (100pF) from base/gate to emitter/source
├── Use proper decoupling (0.1µF ceramic near transistor)
```

## Summary

1. **Transistor** is a three-terminal semiconductor device for amplification and switching

2. **BJT (Bipolar Junction Transistor):** Current-controlled (I_C = β × I_B)

3. **FET (Field Effect Transistor):** Voltage-controlled (I_D = f(V_GS))

4. **NPN BJT:** Collector more positive than Emitter; Base current flows into Base

5. **PNP BJT:** Emitter more positive than Collector; Base current flows out of Base

6. **BJTs have three regions:** Cutoff (OFF), Active (amplify), Saturation (ON)

7. **N-MOSFET:** Drain more positive than Source; positive V_GS turns ON

8. **P-MOSFET:** Source more positive than Drain; negative V_GS turns ON

9. **MOSFET regions:** Cutoff (OFF), Triode (switch ON), Saturation (amplify)

10. **BJTs as switches:** I_B > I_C(sat)/β_min ensures saturation (V_CE(sat) ≈ 0.2V)

11. **MOSFETs as switches:** V_GS > V_GS(th) + margin; R_DS(on) very low (mΩ)

12. **Darlington pair:** β_total = β1 × β2 (very high gain, higher V_CE(sat))

13. **Push-pull output:** NPN + PNP for sourcing and sinking current

14. **H-bridge:** Four transistors for bidirectional motor control

15. **Thermal runaway (BJTs):** Use emitter resistor for negative feedback

16. **MOSFET advantages:** No gate current, low R_DS(on), easy to parallel

17. **Testing:** Diode mode for BJT (0.6V B-E, B-C); body diode for MOSFET

18. **Common failures:** Overheating, won't turn off, won't turn on, oscillation
