# 02.04 - Transformers

## What is a Transformer?

A transformer is a static electrical device that transfers electrical energy between two or more circuits through electromagnetic induction. It changes voltage and current levels while keeping frequency constant. Transformers are the reason AC power dominates the world—they made long-distance electrical transmission practical.

The transformer was invented in the 1880s, with key contributions from Ottó Bláthy, Miksa Déri, Károly Zipernowsky (the "ZBD" transformer), and later improved by William Stanley and Nikola Tesla. The ability to step voltage up for transmission and down for safe use made AC the winner of the "War of Currents."

## Basic Principle of Operation

### Electromagnetic Induction

Transformers work because a changing magnetic field induces voltage in a nearby conductor.

```
FARADAY'S LAW OF INDUCTION

    V = -N × dΦ/dt

    WHERE:
    V = induced voltage (volts)
    N = number of turns in coil
    dΦ/dt = rate of change of magnetic flux (webers/second)


MUTUAL INDUCTION (Transformer Action):

    Primary coil ──► Magnetic field ──► Secondary coil
    (input)          (through core)     (output)

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   PRIMARY                    SECONDARY                      │
    │   COIL                       COIL                           │
    │   ┌─┐                        ┌─┐                            │
    │   │ │                        │ │                            │
    │   │ │      IRON CORE         │ │                            │
    │   │ │     ┌─────────┐        │ │                            │
    │   │ │────►│         ├───────►│ │                            │
    │   │ │     │  FLUX   │        │ │                            │
    │   │ │◄────│         ├───────◄│ │                            │
    │   │ │     └─────────┘        │ │                            │
    │   │ │                        │ │                            │
    │   └─┘                        └─┘                            │
    │    │                          │                             │
    │   AC in                       AC out                        │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘

    AC in primary creates CHANGING magnetic field
    Changing field induces voltage in secondary
    Output frequency = input frequency (same!)
```

### Step-by-Step Operation

```
TRANSFORMER OPERATION SEQUENCE

STEP 1: AC voltage applied to primary coil
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    V_primary (AC) ──► Current flows in primary ──► Magnetic field created

    The field strength changes with the AC waveform:
     ╱╲    ╱╲
    ╱  ╲  ╱  ╲   ← AC current
    ╱    ╲╱    ╲


STEP 2: Magnetic field couples through the core
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Primary coil's magnetic field expands and collapses       │
    │   Iron core "conducts" magnetic flux to secondary coil      │
    │   Core concentrates and guides the magnetic field           │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘


STEP 3: Changing field induces voltage in secondary
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    Faraday's Law: V_secondary = -N_secondary × dΦ/dt

    More turns = more induced voltage
    Faster field change = more induced voltage


STEP 4: Output voltage appears at secondary terminals
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    V_secondary / V_primary = N_secondary / N_primary

    This is the TURNS RATIO!
```

## The Turns Ratio

The turns ratio is the most important parameter of any transformer.

```
TURNS RATIO FORMULA

    a = N_primary / N_secondary = V_primary / V_secondary = I_secondary / I_primary

    WHERE:
    a = turns ratio (primary:secondary)
    N = number of turns
    V = voltage
    I = current


THREE CASES:

STEP-DOWN TRANSFORMER (a > 1):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    N_primary > N_secondary
    V_secondary = V_primary × (N_secondary / N_primary)  [LOWER]
    I_secondary = I_primary × (N_primary / N_secondary)  [HIGHER]

    Example: 1000 turns primary, 100 turns secondary
    a = 1000/100 = 10
    120V primary → 12V secondary
    1A primary → 10A secondary

    Use: Power supplies, battery chargers, doorbells


STEP-UP TRANSFORMER (a < 1):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    N_primary < N_secondary
    V_secondary = V_primary × (N_secondary / N_primary)  [HIGHER]
    I_secondary = I_primary × (N_primary / N_secondary)  [LOWER]

    Example: 100 turns primary, 1000 turns secondary
    a = 100/1000 = 0.1
    12V primary → 120V secondary
    10A primary → 1A secondary

    Use: Power transmission, ignition systems, neon signs


ISOLATION TRANSFORMER (a = 1):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    N_primary = N_secondary
    V_secondary = V_primary
    I_secondary = I_primary

    Example: 500 turns primary, 500 turns secondary
    a = 1
    120V primary → 120V secondary
    1A primary → 1A secondary

    Use: Safety isolation, ground breaking, medical equipment


TURNS RATIO VISUALIZATION:

    Step-Down (10:1):        Step-Up (1:10):         Isolation (1:1):
    ┌───[10T]───┐            ┌───[1T]────┐           ┌───[1T]───┐
    │           │            │           │           │          │
    │  120V     │  12V       │  12V      │  120V     │ 120V     │ 120V
    │  1A       │  10A       │  10A      │  1A       │ 1A       │ 1A
    │           │            │           │           │          │
    └───[1T]────┘            └───[10T]───┘           └───[1T]───┘

    Thicker wire on    Thicker wire on      Same wire both
    secondary (high    primary (high        sides (equal current)
    current)           current)
```

## Transformer Construction

### Core Types

The core provides a path for magnetic flux and determines transformer characteristics.

```
CORE MATERIALS

SILICON STEEL LAMINATIONS (Power transformers):

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Thin sheets stacked together:                             │
    │   ┌──────────────────────────────────────────────┐          │
    │   │══════════════════════════════════════════════│ Sheet 1  │
    │   │──────────────────────────────────────────────│ Sheet 2  │
    │   │══════════════════════════════════════════════│ Sheet 3  │
    │   │──────────────────────────────────────────────│ Sheet 4  │
    │   └──────────────────────────────────────────────┘          │
    │                                                             │
    │   Each lamination insulated from neighbors                  │
    │   Prevents eddy currents (reduces losses)                   │
    │   Typical thickness: 0.3-0.5 mm                             │
    │   Frequency: 50/60 Hz                                       │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘


FERRITE CORE (High frequency):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Powdered iron oxide ceramic material                      │
    │   ┌─────┐                                                   │
    │   │▒▒▒▒▒│  High electrical resistance                       │
    │   │▒▒▒▒▒│  Low eddy current losses                          │
    │   │▒▒▒▒▒│  Good for 10 kHz - 1 MHz                          │
    │   └─────┘                                                   │
    │                                                             │
    │   Used in: Switch-mode power supplies, flyback transformers │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘


TOROIDAL CORE (Donut shape):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │         ┌─────────────────────┐                             │
    │        /                       \                            │
    │       │                         │                           │
    │       │     ┌─────────────┐     │                           │
    │       │     │  Core       │     │                           │

    │                                                             │

CORE TYPE COMPARISON:

    Core Type          Efficiency   Size   Cost    EMI    Frequency
    ──────────────────────────────────────────────────────────────
```

```
    │   Windings layered like onion skins                         │
    │   High voltage winding outermost (for insulation)           │
    │                                                             │

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    Primary and secondary interleaved:
    
    P1 ─ S1 ─ P2 ─ S2 ─ P3 ─ S3
    
    Advantages:
    ├── Lower leakage inductance
    ├── Better high frequency response
    ├── Used in switch-mode power supplies
    └── More complex to manufacture


MULTIPLE SECONDARIES:

    One primary, multiple secondaries:

    All share the same magnetic flux
    Used in: Power supplies with multiple outputs
```

## Transformer Types and Applications

### Power Transformers

Power transformers handle high voltages and power levels for electrical distribution.

```
POWER TRANSFORMER HIERARCHY

    GENERATION (11kV - 25kV)
         │
         ▼
    STEP-UP TRANSFORMER
    (11kV → 115kV - 765kV)
         │
         ▼
    TRANSMISSION (115kV - 765kV)
    (Hundreds of kilometers)
         │
         ▼
    STEP-DOWN TRANSFORMER (Substation)
    (765kV → 115kV → 13.8kV - 34.5kV)
         │
         ▼
    DISTRIBUTION (13.8kV - 34.5kV)
    (Few kilometers)
         │
         ▼
    DISTRIBUTION TRANSFORMER (Pole-mounted)
    (13.8kV → 120V/240V)
         │
         ▼
    END USER (120V/240V)


TYPICAL POWER TRANSFORMER SPECIFICATIONS:

    Type              Power      Primary     Secondary    Cooling
    ──────────────────────────────────────────────────────────────
    Pole mount        10-500kVA   13.8kV      120/240V     Oil (passive)
    Pad mount         100-5000kVA 34.5kV      120/240V     Oil (passive)
    Substation        5-500MVA    115-765kV   13.8-34.5kV  Oil + fans
    Generator step-up 50-1500MVA  11-25kV     115-765kV    Oil + fans + pumps
    Distribution      0.1-5MVA    13.8-34.5kV 120V-480V    Oil or dry
```

### Isolation Transformers

Isolation transformers provide electrical separation between input and output.

```
ISOLATION TRANSFORMER PURPOSE

    PRIMARY SIDE                    SECONDARY SIDE
    (Connected to wall)             (Connected to equipment)
    
     Live ──┬──                       ┌── Neutral (isolated)
            │                         │
     ┌──────┴──────┐              ┌───┴──────┐
     │  Primary    │    IRON      │ Secondary│
     │  Coil       │◄══ CORE ═══► │  Coil    │
     └─────────────┘              └──────────┘
            │                         │
    Neutral─┴──                       └── Live (isolated)
    
    NO direct electrical connection between primary and secondary
    Energy transferred magnetically only


WHY USE ISOLATION TRANSFORMER:

    Safety:
    ├── Breaks ground loops
    ├── Prevents shock from single contact (no reference to earth)
    ├── Required for medical equipment (patient safety)
    └── Protects oscilloscope when measuring live circuits

    Noise reduction:
    ├── Blocks common-mode noise
    ├── Filters high frequency interference
    └── Used in sensitive audio and measurement


TESTING WITH ISOLATION TRANSFORMER:

    WORKING ON LIVE CIRCUITS (with extreme caution):
    ├── Oscilloscope ground is isolated from earth
    ├── Can measure across any two points without shorting
    ├── STILL DANGEROUS (current can still kill)
    └── Only for trained professionals


SPECIAL TYPES:

    Medical isolation transformer:
    ├── Extra low leakage current (<100 µA)
    ├── Shielded primary-secondary
    ├── Hospital grade outlets
    └── Used in operating rooms, ICUs

    Ultra-isolation transformer:
    ├── Faraday shield between windings
    ├── Capacitive coupling eliminated
    ├── <0.001 pF coupling capacitance
    └── Used in sensitive lab equipment
```

### Autotransformer

An autotransformer uses a single winding with a tap, making it smaller and cheaper.

```
AUTOTRANSFORMER CONSTRUCTION

    Standard transformer:              Autotransformer:
    
    ┌───[P]───┐                        ┌────────────────────────┐
    │         │                        │                        │
    │  120V   │  12V                   │  120V in               │
    │         │                        │  │                     │
    └───[S]───┘                        │  └──[Tap]──120V out    │
                                       │  │                     │
    Two separate windings              │  └──[Tap]──12V out     │
    (more copper, larger)              │  │                     │
                                       │  │                     │
                                       │  Single winding        │
                                       └────────────────────────┘
                                       (less copper, smaller)


VOLTAGE RELATIONSHIP (Autotransformer):

    V_secondary = V_primary × (N_tap / N_total)

    Example: 120V primary, 120 total turns
    Tap at 12 turns: V_out = 120 × (12/120) = 12V
    Tap at 60 turns: V_out = 120 × (60/120) = 60V
    Tap at 120 turns: V_out = 120V (isolation lost!)


ADVANTAGES OF AUTOTRANSFORMER:

    ├── Smaller and lighter (20-50% less material)
    ├── Cheaper (less copper and core)
    ├── Higher efficiency (less loss)
    ├── Better voltage regulation
    └── Can be continuously variable (variac)


DISADVANTAGES:

    ├── NO isolation (primary and secondary connected!)
    ├── Dangerous if used incorrectly
    ├── Fault on secondary affects primary
    └── Not for medical or safety applications


TYPICAL USES:

    ├── Variac (variable AC supply)
    ├── Motor starting (reduced voltage)
    ├── Voltage adjustment (brownout compensation)
    ├── Railway power (25-0-25kV for trains)
    └── NOT for medical or consumer safety isolation


SAFETY WARNING:

    ┌─────────────────────────────────────────────────────────────┐
    │  WARNING: Autotransformers do NOT provide isolation!        │
    │                                                             │
    │  The output is directly connected to the input.             │
    │  Touching either output terminal can be lethal!             │
    │  Use only with isolation transformer for safety.            │
    └─────────────────────────────────────────────────────────────┘
```

### Instrument Transformers

Special transformers for measurement and protection.

```
CURRENT TRANSFORMER (CT)

    Purpose: Measure high current safely
    Primary: Single turn (wire through core)
    Secondary: Many turns (5A typical)

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   Wire carrying high current                                │
    │   (1000A)                                                   │
    │        │                                                    │
    │        │     ┌─────────────────────┐                        │
    │        └────►│                     │                        │
    │             │   TOROIDAL CORE      │                        │
    │             │                      │                        │
    │        ┌────►│                     │                        │
    │        │     └─────────────────────┘                        │
    │        │              │                                     │
    │        │              │                                     │
    │        │           ┌──┴──┐                                  │
    │        │           │ 5A  │ to meter                         │
    │        │           └─────┘                                  │
    │        │                                                    │
    │   Return path for current                                   │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘

    Turns ratio: 1000A primary : 5A secondary = 200:1 ratio
    N_secondary = 200 turns (for 200:1 CT)


    CRITICAL RULE: Never open-circuit CT secondary!
    ┌─────────────────────────────────────────────────────────────┐
    │  If secondary opens while primary has current:              │
    │  ├── Primary current still flows                            │
    │  ├── No opposing secondary current                          │
    │  ├── Core saturates                                         │
    │  ├── Very high voltage appears on secondary (thousands V!)  │
    │  └── Danger to personnel and equipment!                     │
    └─────────────────────────────────────────────────────────────┘


VOLTAGE TRANSFORMER (VT) or Potential Transformer (PT):

    Purpose: Measure high voltage safely
    Primary: Many turns (connected across high voltage)
    Secondary: Few turns (120V typical)

    Example: 13.8kV primary, 120V secondary
    Ratio = 13800 / 120 = 115:1
    
    Secondary voltage = 120V when primary = 13.8kV
    Used with voltmeters, relays, protection systems

    CRITICAL RULE: Never short-circuit VT secondary!
    ├── Would draw huge current from primary
    ├── Could destroy transformer
    └── Secondary must never be shorted
```

### Audio and RF Transformers

Specialized transformers for signal applications.

```
AUDIO TRANSFORMERS

    Uses:
    ├── Microphone inputs (impedance matching)
    ├── Tube amplifier output (impedance matching)
    ├── Line level isolation (hum reduction)
    ├── Balanced to unbalanced (balun)
    └── Loudspeaker crossovers


    AUDIO OUTPUT TRANSFORMER (Tube amp):
    
    Primary: High impedance (2kΩ - 10kΩ) to match tubes
    Secondary: Low impedance (4Ω, 8Ω, 16Ω) to match speakers
    Turns ratio: √(Z_primary / Z_secondary)
    
    Example: 5000Ω primary, 8Ω secondary
    Ratio = √(5000 / 8) = √625 = 25:1
    (25 turns primary for each 1 turn secondary)


    CHARACTERISTICS:
    ├── Must have flat frequency response (20 Hz - 20 kHz)
    ├── Low distortion (especially at low frequencies)
    ├── High quality core (silicon steel, often grain-oriented)
    └── Often expensive (good ones cost $50-$500)


RF TRANSFORMERS:

    Uses:
    ├── Antenna matching
    ├── Impedance transformation at RF
    ├── Baluns (balanced to unbalanced)
    ├── Mixer circuits
    └── Feedback oscillators


    TYPES:
    ├── Air core (VHF/UHF, <100nH)
    ├── Ferrite core (HF, 1-50 MHz)
    ├── Toroidal (low EMI, good coupling)
    └── Transmission line (very wide bandwidth)


    WIDE BAND RF TRANSFORMER:
    ├── Often wound on ferrite toroid
    ├── Bifilar or trifilar windings
    ├── 1:1, 1:4, 1:9, 4:1 impedance ratios
    └── Used in baluns, mixers, amplifiers
```

## Transformer Ratings and Specifications

### Voltage Rating

The maximum voltage that can be applied without insulation breakdown.

```
VOLTAGE RATING

    Determined by:
    ├── Insulation thickness between windings
    ├── Insulation between layers
    ├── Core insulation (for laminations)
    ├── Terminal spacing
    └── Creepage and clearance distances


    TYPICAL VOLTAGE RATINGS:

    Type                    Primary     Secondary
    ──────────────────────────────────────────────
    Wall wart (US)          120V        3-24V
    Wall wart (EU)          230V        3-24V
    Isolation transformer   120V/240V   120V/240V
    Distribution pole       13.8kV      120/240V
    Substation              115kV-765kV 13.8kV-34.5kV
    Audio input             Line level  Line level
    Current transformer     Any         Limited (600V insulation)


    INSULATION CLASSES:
```

### Power Rating (VA)

The apparent power rating in volt-amperes (VA) or kilovolt-amperes (kVA).

```
POWER RATING (VA)

    For single-phase transformer:
    VA = V_primary × I_primary = V_secondary × I_secondary

    For three-phase transformer:
    VA = √3 × V_line × I_line


    EXAMPLE CALCULATION:

    Transformer: 120V primary, 12V secondary, 5A secondary
    VA = 12V × 5A = 60VA
    
    Primary current = VA / V_primary = 60 / 120 = 0.5A


    EFFICIENCY:

    η = (P_out / P_in) × 100%

    Small transformer:  η = 70-90%
    Large transformer:  η = 95-99%
    Losses: Copper loss (I²R) + Core loss (hysteresis + eddy current)


    TEMPERATURE RISE:

    Typical temperature rise at rated power:
    ├── Small (<50VA): 40-50°C above ambient
    ├── Medium (50-500VA): 30-40°C above ambient
    ├── Large (>500VA): 20-30°C above ambient
    └── Ambient usually 40°C maximum


    DERATING:

    For higher ambient temperature, reduce VA:
    
    Ambient    Derating Factor
    ──────────────────────────
    40°C       100%
    50°C       80%
    60°C       60%
    70°C       40%
```

### Frequency Rating

Transformers are designed for specific frequencies.

```
FREQUENCY EFFECTS

    CORE SATURATION:
    B_max ∝ V / (f × N × A_core)
    
    Lower frequency → Higher flux density → Risk of saturation
    Higher frequency → Lower flux density → Can use smaller core


    FREQUENCY RESPONSE:

    Power transformers:
    ├── 50 Hz: Designed for 50 Hz ±5%
    ├── 60 Hz: Designed for 60 Hz ±5%
    ├── 400 Hz: Aircraft and military
    └── Can run 60 Hz transformer on 50 Hz? MAYBE at reduced voltage


    60 Hz TRANSFORMER ON 50 Hz:

    V_max(50Hz) = V_rated(60Hz) × (50/60) = V_rated × 0.833
    
    Example: 120V 60Hz transformer on 50Hz
    Max voltage = 120 × 0.833 = 100V
    
    Applying 120V at 50Hz would saturate core → overheating!


    FREQUENCY LIMITATIONS:

    Frequency     Core Material    Application
    ──────────────────────────────────────────────
    50-400 Hz     Silicon steel    Power distribution
    400-4000 Hz   Thin silicon steel Aircraft, military
    10-100 kHz    Ferrite          Switch-mode supplies
    100 kHz-1 MHz Ferrite, powdered iron RF power supplies
    1-100 MHz     Ferrite, air core RF circuits, baluns
    >100 MHz      Air core, ferrite beads VHF/UHF circuits
```

## Transformer Losses

No transformer is perfect. Losses appear as heat.

```
TYPES OF LOSSES

CORE LOSSES (Fixed, depend on voltage and frequency):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    1. HYSTERESIS LOSS:
       ┌─────────────────────────────────────────────────────────────┐
       │                                                             │
       │   Energy lost in repeatedly magnetizing and demagnetizing   │
       │   the core material.                                        │
       │                                                             │
       │   P_hyst = k_h × f × B_maxⁿ × volume                        │
       │                                                             │
       │   k_h = hysteresis constant                                 │
       │   n = Steinmetz exponent (1.5-2.5)                          │
       │                                                             │
       │   Reduced by: Grain-oriented silicon steel                  │
       │                                                             │
       └─────────────────────────────────────────────────────────────┘

    2. EDDY CURRENT LOSS:
       ┌─────────────────────────────────────────────────────────────┐
       │                                                             │
       │   Circulating currents induced in core material.            │
       │                                                             │
       │   P_eddy = k_e × f² × B_max² × t² × volume                  │
       │                                                             │
       │   t = lamination thickness                                  │
       │                                                             │
       │   Reduced by: Thin laminations (0.3-0.5 mm)                 │
       │              Insulated laminations                          │
       │              Higher resistivity material                    │
       │                                                             │
       └─────────────────────────────────────────────────────────────┘


COPPER LOSSES (Variable, depend on load current):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    1. I²R LOSS (Resistive):
       P_cu = I_primary² × R_primary + I_secondary² × R_secondary
       
       Reduced by: Larger wire gauge
                  Lower resistivity copper (99.9% pure)
                  Shorter winding length (better core design)

    2. SKIN EFFECT LOSS (High frequency only):
       At high frequencies, current flows on surface of wire
       Effectively increases resistance
       
       Reduced by: Litz wire (many insulated strands)
                  Multiple parallel windings


STRAY LOSSES:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    ├── Leakage flux inducing currents in tank/windings
    ├── Dielectric losses in insulation
    └── Typically 0.5-2% of rated power


TYPICAL LOSS DISTRIBUTION (Power transformer):

    Loss Type          Percent of Total Loss
    ──────────────────────────────────────────
    Copper loss (I²R)  40-60% (depends on load)
    Hysteresis         15-25%
    Eddy current       15-25%
    Stray              5-10%


EFFICIENCY AT DIFFERENT LOADS:

    Load %        Efficiency (Large transformer)
    ────────────────────────────────────────────
    0% (no load)  0% (all losses, no output)
    25%           96%
    50%           98%
    75%           98.5%
    100%          98-99% (peak efficiency)
    125% (overload) 96-97%
```

## Transformer Testing

### Polarity Test

Determines the relative polarity of windings.

```
POLARITY TEST

    PURPOSE: Identify which secondary terminal is in phase with primary
    IMPORTANT FOR: Paralleling transformers, three-phase connections

    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    │   PRIMARY            SECONDARY                              │
    │   H1 ──┬──            X1 ──┬──                              │
    │        │                   │                                │
    │        │     ┌─────┐       │                                │
    │        └────►│     ├───────┘                                │
    │              │     │                                        │
    │        ┌────►│     ├───────┐                                │
    │        │     └─────┘       │                                │
    │   H2 ──┴──                   X2 ──┴──                       │
    │                                                             │
    │   Connect H1 to X1                                          │
    │   Measure voltage H2 to X2                                  │
    │                                                             │
    │   If V_H2-X2 = V_primary - V_secondary: ADDITIVE polarity   │
    │   If V_H2-X2 = V_primary - V_secondary: SUBTRACTIVE         │
    │                                                             │
    └─────────────────────────────────────────────────────────────┘


    RESULT INTERPRETATION:

    Additive polarity (H1 to X1):
    └── V_H2-X2 = V_primary + V_secondary
    └── Standard for distribution transformers

    Subtractive polarity (H1 to X1):
    └── V_H2-X2 = V_primary - V_secondary
    └── Standard for some instrument transformers
```

### Turns Ratio Test

Verifies the transformer's voltage ratio.

```
TURNS RATIO TEST

    EQUIPMENT: Turns ratio meter (or voltmeter + AC source)

    PROCEDURE:
    1. Apply safe AC voltage to primary (e.g., 10V)
    2. Measure primary voltage (V_primary_actual)
    3. Measure secondary voltage (V_secondary_actual)
    4. Calculate ratio = V_primary_actual / V_secondary_actual
    5. Compare to nameplate ratio


    ACCEPTABLE TOLERANCES:

    Application                 Tolerance
    ──────────────────────────────────────────
    General purpose             ±5%
    Distribution transformer    ±1-2%
    Instrument transformer      ±0.1-0.3%
    Audio matching              ±1-3%


    INTERPRETATION:

    Ratio too high:
    ├── Shortened turns in secondary
    ├── Open turns in primary
    └── Core problems

    Ratio too low:
    ├── Shortened turns in primary
    ├── Open turns in secondary
    └── Misconnection (wrong tap)
```

### Insulation Resistance Test

Checks the quality of insulation between windings and core.

```
MEGGER TEST (Insulation Resistance)

    EQUIPMENT: Megohmmeter (Megger)

    TEST POINTS:
    1. Primary to core/ground
    2. Secondary to core/ground
    3. Primary to secondary
    4. Between sections (if multi-winding)


    TYPICAL VALUES (at 500V or 1000V test voltage):

    Condition               Insulation Resistance
    ──────────────────────────────────────────────
    New, dry                >1000 MΩ
    Good, in service        100-1000 MΩ
    Acceptable (damp)       10-100 MΩ
    Marginal (wet/dirty)    1-10 MΩ
    Failed (shorted)        <1 MΩ


    TEMPERATURE CORRECTION:

    Insulation resistance decreases with temperature:
    ├── Double every 10°C rise (rough rule)
    ├── Correct to 20°C for comparison
    └── Always note temperature when testing


    INTERPRETATION:

    Low primary-secondary:
    ├── Winding insulation breakdown
    ├── Moisture ingress
    ├── Contamination (salt, dirt)
    └── Transformer unsafe for use

    Low winding-ground:
    ├── Core insulation failure
    ├── Mechanical damage
    ├── Creepage path contaminated
    └── Requires repair or replacement
```

## Common Transformer Problems

### Problem 1: Overheating

```
SYMPTOMS:
├── Transformer too hot to touch
├── Burning smell (insulation)
├── Reduced output voltage
├── Buzzing or humming louder than normal

CAUSES:
├── Overload (exceeding VA rating)
├── High ambient temperature
├── Poor ventilation (blocked cooling)
├── Core saturation (overvoltage or low frequency)
├── Short-circuited secondary
├── Winding insulation breakdown (internal short)

DIAGNOSIS:
├── Measure primary current (should be ≤ nameplate)
├── Measure secondary current (should be ≤ rating)
├── Check input voltage (should be ≤ nameplate)
├── Check frequency (50Hz vs 60Hz)
├── Measure no-load current (should be 5-10% of full load)
├── Thermal imaging (hot spots indicate internal shorts)

SOLUTIONS:
├── Reduce load (add second transformer in parallel)
├── Improve ventilation (fans, larger enclosure)
├── Correct voltage/frequency issue
├── Repair shorted windings (not economical for small units)
├── Replace transformer
```

### Problem 2: Humming and Vibration

```
SYMPTOMS:
├── Audible 50/60 Hz hum
├── Physical vibration
├── Sometimes buzzing at harmonics (100/120 Hz)

CAUSES:
├── Loose laminations (core not tight)
├── Loose windings (not secured)
├── Core saturation (overvoltage)
├── DC offset in primary (causes half-cycle saturation)
├── Loose mounting hardware
├── Mechanical resonance with enclosure

DIAGNOSIS:
├── Tighten all hardware
├── Check input voltage (should be ≤ rating)
├── Check for DC on primary (use coupling capacitor)
├── Apply load (hum often changes with load)
├── Compare to known good transformer

SOLUTIONS:
├── Re-tighten core bolts
├── Varnish impregnation (fills gaps, reduces noise)
├── Add rubber isolation mounts
├── Reduce voltage to eliminate saturation
├── Block DC with series capacitor
└── Replace if internal looseness (not repairable)
```

### Problem 3: Open Winding

```
SYMPTOMS:
├── No output voltage
├── Primary draws no current (or very little magnetizing current)
├── Secondary voltage = 0V

CAUSES:
├── Internal fuse (some transformers have thermal fuse)
├── Broken wire at terminal
├── Burned open winding (overload or short)
├── Corroded connection (inside transformer)
├── Thermal cycling fatigue (wire breaks)

DIAGNOSIS:
├── Measure primary continuity (should be low ohms, not open)
├── Measure secondary continuity
├── Check for thermal fuse (if present, may be accessible)
├── Visual inspection of terminals
├── If open, transformer is likely failed

SOLUTIONS:
├── Replace thermal fuse (if accessible and replaceable)
├── Re-solder terminal connections
├── If winding open internally → REPLACE (not repairable)
├── For expensive transformers, rewinding possible but costly
```

### Problem 4: Short Circuit

```
SYMPTOMS:
├── Excessive primary current (even with no load)
├── Transformer gets hot quickly (seconds to minutes)
├── Buzzing or growling sound
├── Voltage output low or zero
├── Circuit breaker trips or fuse blows

CAUSES:
├── Internal short between windings
├── Short between turns (same winding)
├── Core insulation failure (shorted to core)
├── Moisture ingress (conductive path)
├── Mechanical damage (pierced insulation)

DIAGNOSIS:
├── Measure no-load primary current (should be 5-10% of full load)
├── High no-load current (>30%) indicates short
├── Check insulation resistance with Megger
├── Compare primary and secondary resistance to specifications
├── Thermal imaging (hot spots indicate short location)

SOLUTIONS:
├── Short between turns: REPLACE (not repairable)
├── Short winding-core: Possibly repair if accessible
├── Moisture: Dry out (bake at low temperature 80-100°C for 24h)
├── Most internal shorts = REPLACE
```

## Summary

1. **Transformer** transfers electrical energy between circuits via electromagnetic induction

2. **Turns ratio** determines voltage and current: V_p/V_s = N_p/N_s = I_s/I_p

3. **Step-down:** N_p > N_s (lower V, higher I)
4. **Step-up:** N_p < N_s (higher V, lower I)
5. **Isolation:** N_p = N_s (same V/I, safety isolation)

6. **Core types:** Laminated steel (50/60Hz), ferrite (kHz-MHz), toroidal (low EMI)

7. **Power transformers** handle high power (VA to MVA) for grid distribution

8. **Isolation transformers** provide safety isolation, break ground loops

9. **Autotransformers** use single winding, are smaller but NOT isolated (dangerous!)

10. **Current transformers** (CT) measure high current; NEVER open secondary!

11. **Voltage transformers** (VT/PT) measure high voltage; NEVER short secondary!

12. **Losses:** Core loss (hysteresis + eddy current) + Copper loss (I²R)

13. **Ratings:** Voltage, current, VA (apparent power), frequency, temperature class

14. **Testing:** Polarity, turns ratio, insulation resistance (Megger), continuity

15. **Common failures:** Overheating, open winding, shorted turns, insulation breakdown

16. **Transformers work on AC only** (DC causes saturation and burnout!)
